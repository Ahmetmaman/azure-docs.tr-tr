# <a name="tools-for-using-ansible-with-azure"></a>Ansible Azure ile kullanmak için Araçlar

Aşağıdaki araçları, daha kolay ve verimli Ansible ve Azure ile çalışmayı kolaylaştırır.

## <a name="visual-studio-code-extension-for-ansible"></a>Ansible için Visual Studio Code uzantısı

[Ansible için Visual Studio Code uzantısı](https://marketplace.visualstudio.com/items?itemName=vscoss.vscode-ansible) Visual Studio kodundan Ansible kullanımı için çeşitli özellikler sunar:

- Otomatik Tamamlama Ansible yönergeleri, modüller ve eklentiler yazarken Ansible belgelerindeki yazın.
- Kod parçacıkları görüntü tıklatıldığında &lt;Ctrl >&lt;alanı > Ansible playbooks geliştirme sırasında.
- Söz dizimi vurgulamayı Ansible playbook kodunuzu farklı renk ve yazı tipleri YAML sözdizimi göre görüntüler.
- Visual Studio kod Terminal penceresinden Ansible playbooks çalıştırabilirsiniz.
    - (Yalnızca Windows) Ansible Docker kapsayıcıdan çalıştırabilirsiniz.
    - (Linux ve macOS) Ansible Docker kapsayıcısı veya yerel Ansible yüklemesi çalıştırabilirsiniz. 
- Azure bulut Kabuğu'ndan Ansible playbooks çalıştırabilirsiniz.
