# <a name="reset-the-password-for-a-user-in-azure-active-directory"></a>Azure Active Directory'de bir kullanıcı parolasını sıfırlama

## <a name="how-to-reset-the-password-for-a-user"></a>Bir kullanıcının parolasını sıfırlama
1. Oturum [Azure AD Yönetim Merkezi](https://aad.portal.azure.com) dizini için genel yönetici olan bir hesapla.
2. Seçin **kullanıcılar ve gruplar**.

   ![Açılış kullanıcı yönetimi](./media/active-directory-users-reset-password-azure-portal/create-users-user-management.png)
3. Seçin **tüm kullanıcılar**.

   ![Tüm kullanıcılar grubu açma](./media/active-directory-users-reset-password-azure-portal/create-users-open-users-blade.png)
1. Listeden bir kullanıcı seçin.
2. Seçili kullanıcı için **Genel Bakış**’ı seçin ve ardından komut çubuğunda **Parolayı sıfırla**’yı seçin.

    ![Sıfırlama parolası komutu seçme](./media/active-directory-users-reset-password-azure-portal/create-users-reset-password-command.png)
6. Üzerinde **parola sıfırlama**seçin **parola sıfırlama**.
