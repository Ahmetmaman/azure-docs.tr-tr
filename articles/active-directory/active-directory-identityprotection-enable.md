# <a name="enabling-azure-active-directory-identity-protection"></a>Azure Active Directory kimlik Koruması'nı etkinleştirme

Azure Active Directory kimlik koruması, Azure Active Directory (Azure AD) için kullanılan bir özelliktir. Azure AD Kimlik Koruması ile yapabilecekleriniz:

- Makine öğrenme algoritmalarını kullanarak bayrak eklenmiş kullanıcılar ve algılanan risk olayları hakkında birleştirilmiş bir görünüm elde edin
- Kullanıcılarınızı otomatik olarak korumak için risk tabanlı Koşullu Erişim ilkeleri belirleyin
- Güvenlik açıkları üzerinde eylemler gerçekleştirerek güvenliği artırın 

Daha fazla bilgi için [Azure Active Directory kimlik koruması](active-directory-identityprotection.md).

Bu makalede, Azure Active Directory kimlik Koruması'nı etkinleştirmek gösterilmektedir.

## <a name="steps-to-enable-azure-active-directory-identity-protection"></a>Azure Active Directory kimlik Koruması'nı etkinleştirme adımları

1. Oturum açma, [Azure portalında](https://ms.portal.azure.com/) genel yönetici olarak. 

2. Azure portalında **Market**.
   
    ![Oluşturma](./media/active-directory-identityprotection-enable/01.png "oluşturma")

3. Uygulamalar listesinde **kimlik**.
   
    ![Oluşturma](./media/active-directory-identityprotection-enable/05.png "oluşturma")

4. Tıklayın **Azure AD kimlik koruması**.
   
    ![Oluşturma](./media/active-directory-identityprotection-enable/06.png "oluşturma")

5. Üzerinde **Azure AD kimlik koruması** dikey penceresinde tıklayın **Oluştur**.
   
    ![Oluşturma](./media/active-directory-identityprotection-enable/04.png "oluşturma")
