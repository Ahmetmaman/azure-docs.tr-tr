# <a name="grant-permissions-to-users-from-partner-organizations-in-your-azure-active-directory-tenant"></a>Azure Active Directory kiracınızdaki iş ortağı kuruluşlarından kullanıcıların izinleri verin

Azure Active Directory (Azure AD) B2B işbirliği kullanıcıları Konuk kullanıcı olarak dizine eklenir ve dizinde Konuk izinleri varsayılan olarak sınırlıdır. İşletmenizi daha yüksek ayrıcalıklı rolleri kuruluşunuzdaki doldurmak için bazı Konuk kullanıcılar gerekebilir. Daha yüksek ayrıcalıklı rolleri tanımlama desteklemek için istediğiniz, kuruluşunuzun ihtiyaçlarına göre herhangi bir rolü Konuk kullanıcıları eklenebilir.
