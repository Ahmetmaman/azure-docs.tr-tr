# <a name="sync-service-manager-metaverse-search"></a>Eşitleme Hizmeti Yöneticisi'ni meta veri deposu arama

![Eşitleme Hizmeti Yöneticisi](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Meta veri deposu Arama sekmesi, verilerle ilgili sorunları gidermek için kullanışlıdır. Üst yarısında, öznitelik bir bileşimine dayalı bir sorgu oluşturabilirsiniz. Sorgunuzu ile memnun kaldığınızda, tıklayın **arama**. Sonuç alt kılavuz görülebilir. Hangi sütunların ile görünür olması seçebileceğiniz **sütun ayarlarını**.

Arama sonuçlarında, bir nesne seçin ve **özellikleri** görmek için [meta veri deposu nesne özellikleri](tshoot-connect-object-not-syncing.md#metaverse-object-properties).
