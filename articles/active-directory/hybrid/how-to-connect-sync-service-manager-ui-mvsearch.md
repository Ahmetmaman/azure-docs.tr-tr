# <a name="sync-service-manager-metaverse-search"></a>Eşitleme Servis Yöneticisi Metaverse Arama

![Eşitleme Servis Yöneticisi](./media/how-to-connect-sync-service-manager-ui-mvsearch/mvsearch.png)

Metaverse arama sekmesi, veriyle ilgili sorunları gidermek için yararlıdır. Üst yarısında, özniteliklerin bir leşimini temel alan bir sorgu oluşturabilirsiniz. Sorgunuzdan memnun **olduğunuzda, Arama'yı**tıklatın. Sonuç alt ızgarada görünür. **Sütun Ayarları**ile hangi sütunların görünür olması gerektiğini seçebilirsiniz.

Arama sonuçlarında, [metaverse nesne özelliklerini](tshoot-connect-object-not-syncing.md#metaverse-object-properties)görmek için bir nesne ve **Özellikler** seçin.
