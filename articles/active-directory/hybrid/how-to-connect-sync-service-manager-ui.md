# <a name="introduction-to-the-azure-ad-connect-synchronization-service-manager-ui"></a>Azure AD Connect Synchronization Service Manager Kullanıcı arabirimine giriş

![Eşitleme Service Manager](./media/how-to-connect-sync-service-manager-ui/ssmui.png)

**Synchronization Service Manager** Kullanıcı arabirimi, eşitleme altyapısının daha gelişmiş yönlerini yapılandırmak ve hizmetin işletimsel yönlerini görmek için kullanılır.

**Synchronization Service Manager** Kullanıcı arabirimini Başlat menüsünden başlatabilirsiniz. **Eşitleme hizmeti** olarak adlandırılır ve **Azure AD Connect** grubunda bulunabilir.
