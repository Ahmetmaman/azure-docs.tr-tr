# <a name="introduction-to-the-azure-ad-connect-synchronization-service-manager-ui"></a>Azure AD Connect Senkronizasyon Hizmet Yöneticisi Kullanıcı Ara Birimi'ne giriş

![Eşitleme Servis Yöneticisi](./media/how-to-connect-sync-service-manager-ui/ssmui.png)

**Eşitleme Servis Yöneticisi** Kullanıcı Aracı, eşitleme altyapısının daha gelişmiş yönlerini yapılandırmak ve hizmetin operasyonel yönlerini görmek için kullanılır.

**Eşitleme Hizmet Yöneticisi** Kullanıcı UI'sini başlat menüsünden başla. **Eşitleme Hizmeti** olarak adlandırılır ve Azure **AD Connect** grubunda bulunabilir.
