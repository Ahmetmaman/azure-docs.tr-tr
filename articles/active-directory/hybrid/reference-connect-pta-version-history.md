# <a name="azure-ad-pass-through-authentication-agent-version-release-history"></a>Azure AD Pass-through Authentication aracısı: Sürüm sürüm geçmişi 
 
Pass-through Kimlik Doğrulaması sağlayan şirket içinde yüklenen aracılar, yeni özellikler sağlamak için düzenli olarak güncelleştirilir. Bu makalede, yeni işlevler tanıtıldığında eklenen sürümleri ve özellikleri listeler. Geçiş kimlik doğrulama aracıları, yeni bir sürüm yayımlandığında otomatik olarak güncelleştirilir. 

İlgili konular şunlardır: 

- [Azure AD Pass-through Kimlik Doğrulaması ile kullanıcı oturum açma](how-to-connect-pta.md) 
- [Azure AD Pass-through Kimlik Doğrulama aracısı yükleme](how-to-connect-pta-quick-start.md) 



## <a name="1510070"></a>1.5.1007.0 
### <a name="release-status"></a>Yayın durumu 
22.01.2019: İndirmek için yayınlandı  
### <a name="new-features-and-improvements"></a>Yeni özellikler ve geliştirmeler 
- Giden bağlantılar için başka bir bağlantı esnekliği katmanı eklemek için Servis Veri Servisi güvenilir kanalları için destek eklendi 
- Acente kaydı sırasında TLS 1.2'yi uygulayın 

## <a name="156430"></a>1.5.643.0 
### <a name="release-status"></a>Yayın durumu 
04.10.2018: İndirmek için yayınlandı  
### <a name="new-features-and-improvements"></a>Yeni özellikler ve geliştirmeler 
- Web soketi bağlantı desteği 
- TLS 1.2'yi aracı için varsayılan protokol olarak ayarlayın 
 
## <a name="154050"></a>1.5.405.0 
### <a name="release-status"></a>Yayın durumu 
31.01.2018: İndirmek için yayınlandı  
### <a name="fixed-issues"></a>Düzeltilen sorunlar 

- Aracıda bazı bellek sızıntıları neden bir hata düzeltildi. 
- Bağlayıcı zaman aşım sorunları için hata düzeltmesi içeren Azure Hizmet Veri Servisi sürümünü güncelleştirin.
