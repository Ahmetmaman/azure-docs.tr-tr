# <a name="azure-ad-pass-through-authentication-agent-version-release-history"></a>Azure AD geçişli kimlik doğrulama Aracısı: sürüm yayınlama geçmişi 
 
Şirket içinde geçiş kimlik doğrulamasını etkinleştiren aracılar, yeni yetenekler sağlamak için düzenli olarak güncelleştirilir. Bu makalede, yeni işlevsellik sunulmasıyla eklenen sürümler ve özellikler listelenir. Yeni bir sürüm yayınlandığında doğrudan kimlik doğrulama aracıları otomatik olarak güncelleştirilir. 

İlgili konular aşağıda verilmiştir: 

- [Azure AD geçişli kimlik doğrulamasıyla Kullanıcı oturumu açma](how-to-connect-pta.md) 
- [Azure AD geçişli kimlik doğrulama Aracısı yüklemesi](how-to-connect-pta-quick-start.md) 

## <a name="1517420"></a>1.5.1742.0
### <a name="release-status"></a>Yayın durumu: 
04/09/2020: indirilmek üzere yayınlandı

### <a name="new-features-and-improvements"></a>Yeni özellikler ve geliştirmeler

- Yükleme sonrasında bulut ortamlarını hedefleme desteği eklendi. Paket, belirli bir bulut ortamına sabitlenebilir.



## <a name="1510070"></a>1.5.1007.0 
### <a name="release-status"></a>Yayın durumu

- Aracı kaydı sırasında TLS 1,2 'yi zorla
