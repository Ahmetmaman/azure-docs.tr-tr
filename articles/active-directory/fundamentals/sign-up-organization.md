# <a name="sign-up-your-organization-to-use-azure-active-directory"></a>Kuruluşunuzda kullanmak için kaydolun Azure Active Directory
Azure Active Directory (Azure AD) veya yeni bir Microsoft Azure aboneliğine kaydolma, aşağıdakilerden birini kullanarak:

- **Microsoft hesabı.** Azure 'a ve Outlook (hotmail), Messenger, OneDrive, MSN, Xbox LIVE veya Microsoft 365 gibi tüm tüketici odaklı Microsoft ürünlerine ve bulut hizmetlerine erişim sağlamak için kişisel Microsoft hesabı kullanın. Bir Outlook.com posta kutusuna kaydolma otomatik olarak bir Microsoft hesabı oluşturur. Daha fazla bilgi için bkz. [Microsoft hesabı genel bakış](https://account.microsoft.com/account).

- **İş veya okul hesabı.** Azure, Microsoft Intune veya Microsoft 365 gibi Microsoft 'un tüm küçük, orta ve Kurumsal bulut hizmetlerine erişim sağlamak için iş veya okulla ilgili hesabınızı kullanın. Bu hizmetlerden birine bir kuruluş olarak kaydolduktan sonra Azure AD, kuruluşunuzu temsil eden bulut tabanlı bir dizini otomatik olarak sağlar. Daha fazla bilgi için bkz. [Azure AD dizininizi yönetme](./active-directory-whatis.md).

  > [!Note]
  >   Azure AD 'ye zaten erişiminiz varsa iş veya okul hesabınızı kullanmanızı öneririz. Ancak, Azure aboneliğinizle ilişkili olan hesap türünü kullanmanız gerekir.
