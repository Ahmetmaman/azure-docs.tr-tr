# <a name="sign-up-your-organization-to-use-azure-active-directory"></a>Azure Active Directory'yi kullanmak için kuruluşunuzun kaydolun
Aşağıdakileri kullanarak Azure Active Directory (Azure AD) veya yeni bir Microsoft Azure aboneliği için kaydolun:

- **Microsoft hesabı.** Azure'a ve Outlook (Hotmail), Messenger, OneDrive, MSN, Xbox LIVE veya Office 365 gibi tüketici odaklı Microsoft ürünlerine ve bulut hizmetlerine erişmek için kişisel, Microsoft hesabınızı kullanın. bir Outlook.com posta kutusuna kaydolmak otomatik olarak bir Microsoft hesabı oluşturur. Daha fazla bilgi için [Microsoft hesabına genel bakış](https://account.microsoft.com/account)alameti ne bürünme.

- **İş veya okul hesabı.** Azure, Microsoft Intune veya Office 365 gibi Microsoft'un tüm küçük, orta ve kurumsal bulut hizmetlerine erişmek için iş veya okulla ilgili hesabınızı kullanın. Bu hizmetlerden birine kuruluş olarak kaydolduktan sonra, Azure AD kuruluşunuzu temsil eden bulut tabanlı bir dizini otomatik olarak kullanır. Daha fazla bilgi için azure [REKLAM dizinini yönet'e](active-directory-administer.md)bakın.

  > [!Note]
  >   Azure AD'ye zaten erişiminiz varsa iş veya okul hesabınızı kullanmanızı öneririz. Ancak, Azure aboneliğinizle ilişkili hangi hesap türünü kullanmanız gerekir.
