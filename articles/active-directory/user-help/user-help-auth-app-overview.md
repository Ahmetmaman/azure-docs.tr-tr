# <a name="microsoft-authenticator-app"></a>Microsoft Authenticator uygulaması

>[!Important]
>Bu içerik kullanıcılara yöneliktir. Yöneticiyseniz, Azure Active Directory (Azure AD) ortamınızı ayarlama ve yönetme hakkında daha fazla bilgi için [Azure Active Directory Belgelerine](https://docs.microsoft.com/en-us/azure/active-directory) bakabilirsiniz.

Microsoft Authenticator uygulaması, iki aşamalı doğrulama ile parolanızı girdikten sonra uygulamalarınızın güvenliğini sağlamanıza yardımcı olur. Ancak bunu kişisel Microsoft hesabınızın parolasının yerine kullanabileceğinizi biliyor muydunuz? Microsoft Authenticator uygulaması, kurulum ve kullanım adımları hakkında bilgi edinin.
