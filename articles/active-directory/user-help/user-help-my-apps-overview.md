# <a name="my-apps-portal-overview"></a>Uygulamalarım portalına genel bakış

İş veya okul hesabınız varsa ve yöneticiniz erişim izni verdiyse Uygulamalarım portalını kullanarak bulut tabanlı uygulamaları görüntüleyebilir ve başlatabilirsiniz.

>[!Important]
>Bu içerik kullanıcılara yöneliktir. Yöneticiyseniz, Azure Active Directory (Azure AD) ortamınızı ayarlama ve yönetme hakkında daha fazla bilgi için [Azure Active Directory Belgelerine](https://docs.microsoft.com/azure/active-directory) bakabilirsiniz.

|Makale |Açıklama |
|------|------------|
|[Uygulamalarım portalı nedir?](active-directory-saas-access-panel-introduction.md)|Uygulamalarım portalına erişme ve portalı kullanma adımlarını anlatır.|
|[Sorunları giderme](active-directory-saas-access-panel-user-help.md)|Uygulamalarım portalında oturum açma veya portalı kullanma sırasında karşılaşabileceğiniz sorunları giderme adımlarını anlatır.|
