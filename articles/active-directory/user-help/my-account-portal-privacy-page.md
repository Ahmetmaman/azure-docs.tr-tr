# <a name="view-how-your-organization-uses-your-privacy-related-data"></a>Kuruluşunuzun gizlilikle ilgili verilerinizi nasıl kullandığını görüntüleme

Kuruluşunuzun verilerinizi Nasıl kullandığını **Hesabım** portalının **Gizlilik** sayfasından görüntüleyebilirsiniz.

## <a name="view-your-privacy-related-info"></a>Gizlilikle ilgili bilgilerinizi görüntüleme

1. İş veya okul hesabınızda oturum açın https://myprofile.microsoft.com/ ve sayfanıza gidin.

2. Sol daki gezinti bölmesinden **Gizlilik'i** seçin veya **Gizlilik** bloğundan **Gizlilik** bağlantısını seçin.

    ![Vurgulanan Gizlilik bağlantılarını gösteren Hesabım sayfam](media/my-account-portal/my-account-portal-privacy.png)

3. Aşağıdakiler de dahil olmak üzere bilgileri gözden geçirin:

    - **Hizmetleri.** İş veya okul hesabınızı kullanarak bağlı olduğunuz çevrimiçi hizmetlerin listesi.

    - **Kullanım koşulları.** Kuruluşunuzun kullanım koşulları.

## <a name="next-steps"></a>Sonraki adımlar

Kuruluşunuzun verilerinizi nasıl kullandığını görüntüledikten sonra şunları yapabilirsiniz:

- [Güvenlik bilgilerinizi](user-help-security-info-overview.md)görüntüleyin veya yönetin.

- Bağlı [cihazlarınızı](my-account-portal-devices-page.md)görüntüleyin veya yönetin.

- [Kuruluşlarınızı](my-account-portal-organizations-page.md)görüntüleyin veya yönetin.

- Oturum [açma etkinliğinizi](my-account-portal-sign-ins-page.md)görüntüleyin.

- Microsoft Office ile ilgili kişi tercihlerinizi ve gizlilik bilgilerinizi [Office portalı, Güvenlik & gizlilik sayfasından](https://portal.office.com/account/#security)da görüntüleyebilirsiniz.
