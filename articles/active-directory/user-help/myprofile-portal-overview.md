# <a name="my-profile-preview-portal-overview"></a>My profili (Önizleme) portalına genel bakış

[!INCLUDE [preview-notice](../../../includes/active-directory-end-user-preview-notice-myprofile.md)]

**Profilim** (Önizleme) portalı, iş veya Okul hesabınızı ayarlama ve güvenlik bilgilerinizi yönetmek, bağlı kuruluşları ve cihazları yönetme ve görüntüleme, kuruluşunuz verilerinizi nasıl kullandığını yönetmek için yardımcı olur.

>[!Important]
>Bu içerik kullanıcılara yöneliktir. Yöneticiyseniz, Azure Active Directory (Azure AD) ortamınızı ayarlama ve yönetme hakkında daha fazla bilgi için [Azure Active Directory Belgelerine](https://docs.microsoft.com/azure/active-directory) bakabilirsiniz.

![Profilim portal, genel bakış sayfası](media/myprofile-portal/myprofile-portal-overview.png)

## <a name="go-to-the-my-profile-portal"></a>Profilim portalına gidin

Ulaşmak için **Profilim** portal (https://myprofile.microsoft.com), aşağıdaki tarayıcılardan birinin geçerli sürüm:

- Chrome
- Microsoft Edge
- Safari
- Firefox
- Internet Explorer 11

## <a name="my-profile-portal-articles"></a>My profili portal makaleleri

Ayrıntılı bilgi ve kullanılabilir sunulanlar hakkında yönergeler için **Profilim** portalı, aşağıdaki makalelere bakın:

|Makale |Açıklama |
|------|------------|
|[Güvenlik bilgilerinizi ekleyin](user-help-security-info-overview.md)| Birkaç makale Microsoft Authenticator uygulamasını, kısa mesaj veya telefon aramaları, doğrulama yöntemi olarak ayarlama hakkında yönergeler. Ayrıca, kendi Unutulan parolayı sıfırlayabilirsiniz için bir e-posta adresi veya önceden tanımlı güvenlik soruları gerekirse ekleme hakkında makaleler içerir.|
|[Görüntülemek veya bağlı kuruluşunuzun bırakın](myprofile-portal-organizations-page.md)| Görüntüleme ve kuruluşların bırakın hakkında yönergeler, iş veya Okul hesabınıza bağlı.|
|[Görüntülemek veya bağlı cihazlarınızın devre dışı bırakma](myprofile-portal-devices-page.md)| Görüntülemek veya herhangi bir aygıtı devre dışı bırakma hakkında yönergeler için iş veya Okul hesabınızı bağlandınız.|
|[Gizlilikle ilgili bilgilerinizi görüntüleme](myprofile-portal-privacy-page.md)| Çevrimiçi Hizmetleri çalışmanıza bağlı olduğundan veya Okul hesabınız, kuruluşunuzun koşullarını birlikte görüntülemek hakkında yönergeler deyimini kullanın.|
