# <a name="register-your-personal-device-on-your-organizations-network"></a>Kişisel cihazınızı kuruluşunuzun ağına kaydettirin
Kişisel cihazınızı (genellikle bir telefon veya tablet) kuruluşunuzun ağına kaydedin. Aygıtınız kaydolduktan sonra kuruluşunuzun kısıtlı kaynaklarına erişebilir.

>[!Note]
>Bu makalede, gösteri amacıyla bir Windows aygıtı kullanılır, ancak iOS, Android veya macOS çalıştıran aygıtları da kaydedebilirsiniz.

## <a name="what-happens-when-you-register-your-device"></a>Cihazınızı kaydettirdiğinizde ne olur?
Cihazınızı kuruluşunuzun ağına kaydederken aşağıdaki eylemler gerçekleşir:

- Windows cihazınızı kuruluşunuzun ağına kaydeder.

- İsteğe bağlı olarak, kuruluşunuzun seçimlerine bağlı olarak, [Çok Faktörlü Kimlik Doğrulama](multi-factor-authentication-end-user-first-time.md) veya güvenlik [bilgileri](user-help-security-info-overview.md)aracılığıyla iki aşamalı doğrulama ayarlamanız istenebilir.

- İsteğe bağlı olarak, kuruluşunuzun seçimlerine bağlı olarak, otomatik olarak Microsoft Intune gibi mobil aygıt yönetimine kaydolabilirsiniz. Microsoft Intune'a kaydolma hakkında daha fazla bilgi için [cihazınızı Intune'a kaydet'e](https://docs.microsoft.com/intune-user-help/enroll-your-device-in-intune-all)bakın.

- İş veya okul hesabınız için kullanıcı adı ve parolayı kullanarak oturum açma işleminden geçersiniz.

## <a name="to-register-your-windows-device"></a>Windows cihazınızı kaydetmek için

Kişisel cihazınızı abunuza kaydetmek için aşağıdaki adımları izleyin.

1. **Ayarları**açın ve ardından **Hesaplar'ı**seçin.

    ![Ayarlar ekranındaki hesaplar](./media/user-help-register-device-on-network/register-device-settings-accounts.png)

2. **İş veya okula Eriş'i**seçin ve ardından Access çalışmasından veya okul ekranından **Bağlan'ı** seçin. **Connect**

    ![Connect seçeneği vurgulanmış ken, iş veya okul ekranına erişin](./media/user-help-register-device-on-network/register-device-access-work-school-connect.png)

3. İş **veya okul hesabı ekle** ekranında, iş veya okul hesabınız için e-posta adresinizi yazın ve ardından **İleri'yi**seçin. Örneğin, alain@contoso.com.

4. İş veya okul hesabınızda oturum açın ve ardından **Oturum Aç'ı**seçin.

5. Kimlik doğrulama isteğinizi onaylamak (iki aşamalı doğrulama kullanıyorsanız) ve Windows Hello'yu (gerekirse) ayarlama dahil olmak üzere kayıt işleminin geri kalanını tamamlayın.

## <a name="to-verify-that-youre-registered"></a>Kayıtlı olduğunuzu doğrulamak için
Ayarlarınıza bakarak kayıtlı olduğunuzdan emin olabilirsiniz.

1. **Ayarları**açın ve ardından **Hesaplar'ı**seçin.

    ![Ayarlar ekranındaki hesaplar](./media/user-help-register-device-on-network/register-device-settings-accounts.png)

2. **İş veya okula Eriş'i**seçin ve çalışmanızı veya okul hesabınızı gördüğünüzden emin olun.

    ![Bağlı contoso hesabıyla iş veya okul ekranına erişin](./media/user-help-register-device-on-network/register-device-setup-verify.png)
