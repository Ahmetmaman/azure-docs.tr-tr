# <a name="saas-application-integration-with-azure-active-directory"></a>Azure Active Directory ile SaaS uygulama tümleştirmesi

Bulut-etkin tümleştirmenize yardımcı olmak için [(SaaS)](https://azure.microsoft.com/overview/what-is-saas/) Azure Active Directory ile uygulamaları, biz geliştirdiğinizde, yapılandırma aracılığıyla yol öğreticileri koleksiyonu.

Kullanıcı sağlama rehberi sahip olduğumuz uygulaması Öğreticisi yanındaki sağ sütundaki kullanıcı sağlama uygulaması Öğreticisi eklediniz.

Azure AD ile önceden tümleştirilmiş SaaS uygulamaları kapsamlı listesi için bkz: [Active Directory Marketi](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
