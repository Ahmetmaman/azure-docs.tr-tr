# <a name="saas-application-integration-with-azure-active-directory"></a>Azure Active Directory ile SaaS uygulama tümleştirmesi

Bulut özellikli tümleştirmenize yardımcı olmaya [(SaaS)](https://azure.microsoft.com/overview/what-is-saas/) uygulamaları Azure Active Directory ile biz geliştirdiğinizde koleksiyonu yapılandırma aracılığıyla size yol gösteren öğretici.

Kullanıcı sağlama Kılavuzu, sahip olduğumuz bir uygulaması Öğreticisi uygulaması Öğreticisi yanındaki sağ sütunda kullanıcı sağlama ekledik.

Azure AD ile önceden tümleştirilmiş olan esnek kapsamlı listesi için bkz [Active Directory Marketi](https://azuremarketplace.microsoft.com/marketplace/apps/category/azure-active-directory-apps).
