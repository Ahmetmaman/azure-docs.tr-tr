# <a name="smartsheet-and-azure-active-directory"></a>Smartsheet ve Azure Active Directory

Kuruluşunuz genelinde Smartsheet benimsemeyi hızlandırma ve önceden oluşturulmuş tümleştirmeyle Kullanıcı benimseme sürecini hızlandırın. Smartsheet 'i Azure Active Directory (Azure AD) ile bağlayarak kullanıcıların hızla işbirliği yapmasına izin verebilirsiniz.

![Smartsheet uygulama logosunu gösterir](./media/smartsheet.png)

> [!div class="nextstepaction"]
> [Azure AD > ile Smartsheet 'i deneyin](https://aka.ms/SmartsheetTryItNow)

## <a name="what-can-you-do-with-smartsheet-and-azure-ad"></a>Smartsheet ve Azure AD ile neler yapabilirsiniz?

- **Kolay çoklu oturum açma** -birden çok Kullanıcı adı ve parolayı ortadan kaldırın ve KULLANıCıLARıN Azure AD hesaplarını kullanarak Smartsheet 'te oturum açmasına izin vererek erişimi kolaylaştırın.
- **Kullanıcı sağlamayı otomatikleştirin** -kullanıcılar, kuruluştan katılırsanız, taşıntıklarında veya kuruluştan ayrıldığınızda Kullanıcı hesaplarının oluşturma, bakım ve kaldırma Işlemlerini Smartsheet 'e otomatik hale getirir.
- Koşullu erişim ilkeleri uygulayarak, erişim gözden geçirmeleri aracılığıyla erişimi yöneten ve oturum açma günlükleriyle erişimi izleyen Smartsheet 'e erişimi güvenli hale getirme, yönetme **ve izleme** .
