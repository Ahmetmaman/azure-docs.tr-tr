# <a name="servicenow-and-azure-active-directory"></a>ServiceNow ve Azure Active Directory

Kuruluşunuz genelinde ServiceNow benimseme sürecini hızlandırın ve önceden oluşturulmuş tümleştirmeyle Kullanıcı benimseme sürecini artırın. ServiceNow 'ı Azure Active Directory (Azure AD) bağlayarak BT hizmet yönetimi kolaylaştırabilirsiniz.

> [!div class="nextstepaction"]
> [ServiceNow 'ı Azure AD > deneyin](https://aka.ms/ServiceNowTryItNow)

## <a name="what-can-you-do-with-servicenow-and-azure-ad"></a>ServiceNow ve Azure AD ile neler yapabilirsiniz?

- **Kolay çoklu oturum açma** -birden çok Kullanıcı adı ve parolayı ortadan kaldırın ve KULLANıCıLARıN Azure AD hesaplarını kullanarak ServiceNow 'da oturum açmasına izin vererek erişimi kolaylaştırın.
- **Kullanıcı sağlamayı otomatikleştirin** -kullanıcılar, kuruluştan katılıma, taşıma veya kuruluştan ayrılma sırasında Kullanıcı hesaplarının oluşturulmasını, bakımını ve kaldırılmasını otomatik hale getirir.
- Koşullu erişim ilkeleri uygulayarak, erişim gözden geçirmeleri aracılığıyla erişimi yöneten ve oturum açma günlükleriyle erişimi izleyen ServiceNow 'a erişimi güvenli hale getirme, yönetme **ve izleme** .
