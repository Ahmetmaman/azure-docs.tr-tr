# <a name="workday-and-azure-active-directory"></a>İşgünü ve Azure Active Directory

Önceden oluşturulmuş tümleştirmenizi kullanarak Workday 'yi Azure Active Directory (Azure AD) ile bağlayın ve çalışanlar katılırsanız, taşındığında veya ayrıldığınızda Kullanıcı hesaplarını otomatik olarak sağlayın veya serbest bırakın.

![Workday uygulama logosunu gösterir](./media/workday.png)

> [!div class="nextstepaction"]
> [Azure AD > ile Workday 'i deneyin](https://aka.ms/WorkdayTryItNow)

## <a name="what-can-you-do-with-workday-and-azure-ad"></a>Workday ve Azure AD ile ne yapabilirsiniz?

- **Kullanıcı sağlamayı otomatik hale** getirme-iş Workday 'deki Grup, rol veya diğer özniteliklerine göre otomatik olarak çalışanlara uygulama atama.
- **Çalışan profillerini eşzamanlı hale getirmek** IÇIN Azure AD ile Workday arasında çalışan verilerinin güncel kalmasını sağlayın.
- **Ekleme ve çıkarma Işlemlerini basitleştirecek ve** Kullanıcı hesaplarını yeni bir süre içinde üretken hale getirerek, kullanıcılar oturum açtığında kullanıcı hesaplarını otomatik olarak devre dışı bırakarak güvenlik risklerini kısaltarak otomatik olarak sağlama.
- **Kolay çoklu oturum açma** Kullanıcı adını ve parolaları kaldırın ve kullanıcıların Azure AD 'de barındırılan kurumsal hesaplarını kullanarak oturum açmalarına izin verin.

## <a name="get-started"></a>başlarken
