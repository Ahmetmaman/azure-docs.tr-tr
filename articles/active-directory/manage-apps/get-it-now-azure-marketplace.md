# <a name="get-it-now---add-an-app-from-the-azure-marketplace"></a>Şimdi alın-Azure Marketi 'nden bir uygulama ekleyin

Neredeyse orada! 

Azure AD 'yi bir uygulama için kimlik sağlayıcısı olarak kullanmaya çalışıyorsanız, doğru yerde olursunuz. Bunu yalnızca Azure AD kiracınıza eklemeniz yeterlidir. Bunu nasıl yapacağınızı öğrenmek için buradaki hızlı başlangıç serisini izleyin: [Azure AD kiracınızdaki uygulamaları görüntüleyin](view-applications-portal.md).

Yerel bilgisayarınıza veya mobil cihazınıza uygulama yüklemeye çalışıyorsanız, yanlış yersiniz demektir. Azure Marketi, Azure kiracılar ile uygulama kullanan kuruluşlar için tasarlanmıştır. Kişisel bilgisayarlar ve cihazlar için [Microsoft Apps mağazası](https://www.microsoft.com/store/apps)' na göz atın.
