# <a name="add-or-change-profile-information-for-a-user-in-azure-active-directory"></a>Azure Active Directory'de bir kullanıcı için profil bilgileri ekleme veya değiştirme
Bu makalede bir profil resmi veya telefon gibi kullanıcı profili bilgilerini ve e-posta kimlik bilgileri, Azure Active Directory (Azure AD) nasıl ekleneceği açıklanmaktadır. Kuruluşunuzdaki yeni kullanıcı ekleme hakkında daha fazla bilgi için bkz: [Azure Active Directory'ye yeni kullanıcı ekleme](active-directory-users-create-azure-portal.md).

## <a name="to-change-profile-information"></a>Profil bilgilerini değiştirmek için
1. Oturum [Azure AD Yönetim Merkezi](https://aad.portal.azure.com) dizini için genel yönetici olan bir hesapla.
2. Seçin **kullanıcılar ve gruplar**.

   ![Açılış kullanıcılar ve gruplar](./media/active-directory-users-profile-azure-portal/create-users-user-management.png)
3. Seçin **tüm kullanıcılar**.

   ![Tüm kullanıcılar grubu açma](./media/active-directory-users-profile-azure-portal/create-users-open-users-blade.png)
1. Listeden bir kullanıcı seçin.
2. Seçilen kullanıcı için seçin **profil**.

    ![Açılış iş bilgileri](./media/active-directory-users-profile-azure-portal/active-directory-create-users-profile.png)
6. Ekleyebilir veya profil bilgilerini değiştirebilirsiniz. Komut çubuğunda, ardından **kaydetmek**.
