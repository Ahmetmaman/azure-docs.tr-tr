# <a name="introduction-to-the-azure-ad-connect-synchronization-service-manager-ui"></a>Giriş Azure ad Connect Eşitleme Hizmeti Yöneticisi kullanıcı Arabirimi

![Eşitleme Hizmeti Yöneticisi](./media/active-directory-aadconnectsync-service-manager-ui/ssmui.png)

**Eşitleme Hizmeti Yöneticisi'ni** UI eşitleme altyapısı daha gelişmiş yönlerini yapılandırmak ve işlemsel bakış açılarını hizmetinin görmek için kullanılır.

Başlattığınız **Eşitleme Hizmeti Yöneticisi'ni** Başlat menüsünden UI. Bu adlı **eşitleme hizmeti** ve bulunabilir **Azure AD Connect** grubu.
