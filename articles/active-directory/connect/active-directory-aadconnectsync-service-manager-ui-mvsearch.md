# <a name="sync-service-manager-metaverse-search"></a>Eşitleme Hizmeti Yöneticisi'ni meta veri deposu arama

![Eşitleme Hizmeti Yöneticisi](./media/active-directory-aadconnectsync-service-manager-ui/mvsearch.png)

Meta veri deposu Arama sekmesi, veri ilgili sorunları gidermek için kullanışlıdır. Üst yarısında özniteliklerin bir bileşimine dayalı bir sorgu oluşturabilirsiniz. Sorgunuz ile memnun kaldığınızda, tıklatın **arama**. Sonuç alt kılavuzunda görünür olur. İle hangi sütunların görünmesi gereken seçebilirsiniz **sütun ayarları**.

Arama sonuçlarında bir nesne seçin ve **özellikleri** görmek için [meta veri deposu nesne özellikleri](active-directory-aadconnectsync-troubleshoot-object-not-syncing.md#metaverse-object-properties).
