# <a name="create-resource-pools-for-your-private-cloud-with-azure-subscription-mapping"></a>Azure abonelik eşlemesiyle Özel Bulut'unuz için kaynak havuzları oluşturun
Azure abonelik eşleme, kullanılabilir vSphere kaynak havuzlarından Özel Bulut'unuz için kaynak havuzları oluşturmanıza olanak tanır. CloudSimple portalında, Özel Bulutlarınız için Azure aboneliğini görüntüleyebilir ve yönetebilirsiniz.

> [!NOTE]
> Kaynak havuzueşleme de herhangi bir alt kaynak havuzları eşler. Alt kaynak havuzları zaten eşlenmişse, üst kaynak havuzu eşlenemez.

1. [CloudSimple portalına erişin.](access-cloudsimple-portal.md)
2. **Kaynaklar** sayfasını açın ve **Azure abonelikleri eşlemi**seçin.  
3. **Azure abonelik eşlemi'ni edit'i**tıklatın.  
4. Kullanılabilir kaynak havuzlarını eşlemek için, soldakileri seçin ve sağa bakan oku tıklatın. 
5. Eşlemeleri kaldırmak için, sağdakileri seçin ve sola bakan oku tıklatın. 

    ![Azure abonelikleri](media/resources-azure-mapping.png)

6. **Tamam**'a tıklayın.
