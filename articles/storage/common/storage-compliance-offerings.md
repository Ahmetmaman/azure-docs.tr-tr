# <a name="azure-storage-compliance-offerings"></a>Azure Depolama uyumluluk teklifleri

Kuruluşların, bireylerin verilerinin toplanması ve kullanımıyla ilgili ulusal, bölgesel ve sektöre özel gereksinimlere uymasına yardımcı olmak için, Microsoft Azure & Azure Depolama, herhangi bir bulut hizmeti sağlayıcısının en kapsamlı sertifika kümesini ve attestation'larını sunar.

Hizmetinizin Azure Depolama hizmetini kullanırken düzenlendiğinden emin olmak için Azure Depolama'da uyumluluk tekliflerini aşağıda bulabilirsiniz. Bunlar aşağıdaki Azure Depolama teklifleri için geçerlidir: Blobs, Files, Queues, Tablolar, Diskler, Cool Depolama ve Premium Depolama.
