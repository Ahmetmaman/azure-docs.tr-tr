# <a name="guidance-for-azure-data-lake-storage-gen1-customers"></a>Azure Data Lake Storage Gen1 müşteriler için yönergeler

Veri katmanlama, yaşam döngüsü yönetimi ve Hadoop uyumlu dosya sistemi veri Gölü depolama Gen1 kullanılabilir yanı sıra gelişmiş ağ güvenliği de dahil olmak üzere azure Data Lake Storage Gen2 Önizleme teklifleri yerel Blob Depolama işlevselliği.

Varolan bir Data Lake Storage Gen1 müşteri değilseniz, Microsoft, tüm yeni ve geliştirilmiş özelliklerine Data Lake Storage Gen2 değerlendirmek ve tarafından geri bildirim sağlamak için önerir [önizlemede katılan](https://aka.ms/adlsgen2signup). Ayrıca, Data Lake Storage Gen1 desteklenmeye devam eder. Mevcut kaynakları ve Data Lake Storage Gen1 yararlanan uygulamalar güncelleştirmeler olmadan çalışmaya devam etmesi için data Lake Storage Gen1 ve Gen2 bulunabilir. Zamanınızda veri Gölü depolama Gen2 için geçiş planı geliştirebilirsiniz.

Data Lake Storage Gen1 değerlendiriyorsanız, Microsoft, Gen2 önizlemede katılmak ve Data Lake Storage Gen2 tarafından sunulan yeni ve geliştirilmiş yetenekler değerlendirmek için önerir. Benzer şekilde, varolan bir Blob Depolama müşteri değilseniz, Microsoft, Hadoop uyumlu dosya sistemi analitik ihtiyaçlarınız için uygun olup olmadığını değerlendirmek için önerir.
