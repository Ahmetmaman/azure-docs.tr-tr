# <a name="onboard-to-the-azure-certified-for-iot-portal"></a>IoT için Microsoft Azure Sertifikalı portalına ekleme

[IoT için Azure Sertifikalı portalını](https://aka.ms/ACFI) şu şekilde kullanabilirsiniz:

- IoT cihazınız için sertifika işlemini tamamlayın.
- IoT Tak ve Çalıştır Önizleme aygıtı özelliği modellerini bulun. Bu modelleri, IoT'ye hazır aygıtları hızlı bir şekilde oluşturmak ve çözümlerle entegre etmek için kullanabilirsiniz.
- Aygıt yetenek modelini genel depoya yayımlayın.

## <a name="sign-in"></a>Oturum aç

[IoT portalı için Azure Sertifikalı](https://aka.ms/ACFI) portalında oturum açabilmek için Microsoft iş _hesabınızı_ kullanın. Portal, oturum açarken Microsoft İş Ortağı Merkezi ile üyeliğinizi denetler:

- Kuruluşunuz zaten Microsoft İş [Ortağı Merkezi'nin](https://docs.microsoft.com/partner-center/)bir üyesiyse, portaldaki tüm işlevlere erişim hakkı sahibi siniz.
- Kuruluşunuz [Microsoft İş Ortağı Merkezi'nin](https://docs.microsoft.com/partner-center/)bir üyesi değilse, sertifika testleri çalıştırabilir ve aygıt yetenek modeli depolarını kullanabilirsiniz. Aygıtınızı IoT Sertifikalı Aygıt Kataloğu'nda yayınlayamazsınız.
- Microsoft İş Ortağı Ağı Kimliğinizin eksik olduğunu belirten bir bildirim iletisi görürseniz, [bir İş Ortağı Merkezi hesabı oluşturmanız](https://docs.microsoft.com/partner-center/mpn-create-a-partner-center-account)gerekir. Hesabınız onaylandığında, portala dönebilir ve cihazınızı yayınlayabilirsiniz. Daha fazla bilgi için [İş Ortağı Merkezi SSS'ye](https://support.microsoft.com/help/4340639/partner-center-account-faqs)bakın.

Cihazınızı IoT Sertifikalı Cihaz Kataloğu'nda yayımlamadan önce program sözleşmesini okuyun ve kabul edin. Anlaşmayı IoT için Azure Sertifikalı portalında ayarlar sayfasında bulabilirsiniz.

## <a name="next-steps"></a>Sonraki adımlar

Önerilen bir sonraki adım, sertifika için [bir IoT Tak ve Çalıştır cihazını](tutorial-certification-test.md)nasıl göndereceğini öğrenmektir.
