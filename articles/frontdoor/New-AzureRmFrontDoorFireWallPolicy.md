# Yeni AzureRmFrontDoorFireWallPolicy
WAF ilkesi oluşturma

## SÖZ DİZİMİ

```
New-AzureRmFrontDoorFireWallPolicy -ResourceGroupName <String> -Name <String> [-EnabledState <PSEnabledState>]
 [-Mode <PSMode>] -Customrule <PSCustomRule[]> [-ManagedRule <PSManagedRule[]>]
 [-DefaultProfile <IAzureContextContainer>] [-WhatIf] [-Confirm] [<CommonParameters>]
```

## AÇIKLAMA
**Yeni AzureRmFrontDoorFireWallPolicy** cmdlet'i, belirtilen kaynak grubunda geçerli bir abonelik altında yeni bir Azure WAF ilkesi oluşturur

## ÖRNEKLER
