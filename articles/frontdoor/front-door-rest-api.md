# <a name="rest-api-specification-for-azure-front-door-service"></a>Azure ön kapısı hizmeti REST API Belirtimi
Azure ön kapısı hizmeti, tanımlayın, yönetmek ve izlemek için en iyi performans ve yüksek kullanılabilirlik için genel anında yük devretme iyileştirerek yönlendirme, genel web trafiğiniz için olanak sağlar. Ön kapısı ile güçlü, yüksek performanslı kişiselleştirilmiş modern uygulamalar, API ve Azure ile küresel bir hedef kitlesine ulaşın içeriği genel (çok bölgeli) Tüketici ve kurumsal uygulamalara dönüştürebilirsiniz.


Aşağıdaki tabloda, Azure ön kapısı hizmeti için REST API için bir başvuru sağlar.

API sürümü: 2018-08-01
