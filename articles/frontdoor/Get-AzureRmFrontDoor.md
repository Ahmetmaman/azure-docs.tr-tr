# Get-AzureRmFrontDoor
Ön kapısı yapılandırmasını alma

## SÖZ DİZİMİ

```
Get-AzureRmFrontDoor [-ResourceGroupName <String>] [-Name <String>] [-DefaultProfile <IAzureContextContainer>]
 [<CommonParameters>]
```

## AÇIKLAMA
**Get-AzureRmFrontDoor** cmdletGet geçerli Abonelikteki tüm mevcut ön kapı eşleşme sağlanan bilgileri alır.
