# <a name="classic-deployment-model"></a>Klasik dağıtım modeli

Klasik dağıtım modeli, sanal makineler ve sanal makine ölçek kümeleri için global vCPU kotası Azure dağıtım modu land zorlar eski oluşturmadır. Klasik dağıtım modelinde artık önerilmez ve Resource Manager modeli tarafından artık almıştır. Bu iki dağıtımı hakkında daha fazla bilgi için modeller ve avantajı Resource Manager'ın Resource Manager dağıtım modeli sayfasına bakın. Yeni bir abonelik oluşturduğunuzda, varsayılan olarak Vcpu kotası kendisine atanır. Yeni bir sanal makine Klasik dağıtım modeli kullanılarak dağıtılmasını olduğu zaman, yeni ve mevcut Vcpu kullanım tüm bölgeler arasında toplamını Klasik dağıtım modeli için onaylanmış vCPU kotası aşmamalıdır. Azure aboneliği ve hizmet sınırları sayfasına Kotalar hakkında daha fazla bilgi edinin.

Klasik dağıtım modeli aracılığıyla Yardım + destek dikey veya kullanımları + kotası için Vcpu sınırında artış istemek portaldaki dikey pencere.
