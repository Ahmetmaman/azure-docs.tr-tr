# <a name="version-release-history"></a>Sürüm yayınlama geçmişi
Microsoft Genomics takım Microsoft Genomics Python istemcisini düzeltmeler yapmak ve yeni işlevler için düzenli olarak güncelleştirir. 

## <a name="latest-release"></a>En son sürüm
Geçerli Python istemcisini 0.8.1 sürümüdür. 3 Aralık 2018'e ve GATK4 GATK 3.5 ile iş akışlarını çalıştıran destekler yayınlanmıştır. Ayrıca gVCF çıkış destekler.


## <a name="release-history"></a>Sürüm Geçmişi 
Microsoft Genomics Python istemcisinin yeni sürümlerinin yılda iki kere hakkında yayınlanır. Düzeltmeler ve özellikler listesini, burada Microsoft Genomics Python istemcisini yeni sürümlerini serbest olarak güncelleştirilir. Yeni sürümleri yayımlandığında, önceki sürümler en az 90 gün boyunca desteklenmeye devam etmelidir. Önceki sürümleri artık desteklenmemektedir, bu sayfada gösterilir. 

### <a name="version-081"></a>Sürüm 0.8.1
Sürüm 0.8.1 küçük hata düzeltmeleri içerir.  

### <a name="version-080"></a>Sürüm 0.8.0
Sürüm 0.8.0 GATK4 ve gVCFs çıktısı için destek içerir.  

### <a name="version-074"></a>Sürüm 0.7.4
Sürüm 0.7.4 hesap anahtarları yerine SAS belirteçleri kabul etmek için destek içerir `config.txt` giriş. Daha fazla bilgi için [giriş SAS belirteçleri hızlı](quickstart-input-sas.md). 

### <a name="version-073"></a>Sürüm 0.7.3
Sürüm 0.7.3 küçük hata düzeltmeleri içerir.

### <a name="version-072"></a>Sürüm 0.7.2
Sürüm 0.7.2 ilk sürümdür. 1 Kasım 2017'de yayımlanmıştır.
