# <a name="version-release-history"></a>Sürüm yayınlama geçmişi
Microsoft Genomics ekibi, düzeltmeler ve yeni işlevler için Microsoft Genomics Python istemcisini düzenli olarak güncelleştirir. 

## <a name="latest-release"></a>Son sürüm
Geçerli Python istemcisi sürüm 0.9.0'dır. 6 Şubat 2019'da piyasaya sürüldü ve GATK 3.5 ve GATK4 ile iş akışlarının çalışmasını destekliyor. GVCF çıktısını destekler ve çıktı sıkıştırma için isteğe bağlı bir bağımsız değişken kabul edebilir.


## <a name="release-history"></a>Yayın geçmişi 
Microsoft Genomics Python istemcisinin yeni sürümleri yılda yaklaşık bir kez yayımlanır. Microsoft Genomics Python istemcisinin yeni sürümleri yayımlandıkça, düzeltmelerin ve özelliklerin listesi burada güncelleştirilir. Yeni sürümler yayımlandığında, önceki sürümler en az 90 gün boyunca desteklenmeye devam etmelidir. Önceki sürümler artık desteklenince, bu sayfada gösterilir. 

### <a name="version-090"></a>Sürüm 0.9.0
Sürüm 0.9.0 çıktı sıkıştırma desteği içerir. Bu, vcf `-bgzip` veya `-tabix` gvcf çıkışında takip edilen çalıştırmaya eşdeğerdir. Daha fazla bilgi için [bkz.](frequently-asked-questions-genomics.md) 

### <a name="version-081"></a>Sürüm 0.8.1
Sürüm 0.8.1 küçük hata düzeltmeleri içerir.  

### <a name="version-080"></a>Sürüm 0.8.0
Sürüm 0.8.0 GATK4 ve çıkış gVCFs desteği içerir.  

### <a name="version-074"></a>Sürüm 0.7.4
Sürüm 0.7.4, `config.txt` girişteki hesap anahtarları yerine SAS belirteçlerini kabul etme desteğini içerir. Daha fazla bilgi için [Bkz. Giriş SAS belirteçleri quickstart.](quickstart-input-sas.md) 

### <a name="version-073"></a>Sürüm 0.7.3
Sürüm 0.7.3 küçük hata düzeltmeleri içerir.

### <a name="version-072"></a>Sürüm 0.7.2
Sürüm 0.7.2 ilk sürümüdür. 1 Kasım 2017'de yayınlandı.
