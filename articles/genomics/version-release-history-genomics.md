# <a name="version-release-history"></a>Sürüm yayınlama geçmişi
Microsoft Genomics takım düzeltmeleri ve yeni işlevler Microsoft Genomics Python istemci düzenli olarak güncelleştirir. 

## <a name="latest-release"></a>En son sürüm
Geçerli Python istemci 0.7.4 sürümüdür. 5 Mart 2018 yayımlanmıştır ve GATK 3.3 kullanır. 


## <a name="release-history"></a>Sürüm Geçmişi 
Microsoft Genomics Python istemcisini yeni sürümlerini varsayılan olarak, iki kez yıllık hakkında yayınlanır. Microsoft Genomics Python istemcisini yeni sürümleri kullanıma gibi düzeltmeler ve özellikler listesini burada güncelleştirilir. Yeni sürümler yayımlandığında, önceki sürümler için en az 90 gün desteklenmeye devam etmelidir. Önceki sürümler artık desteklendiğinde, bu sayfada gösterilir. 

### <a name="version-074"></a>Sürüm 0.7.4
Sürüm 0.7.4 hesabı anahtarları yerine SAS belirteci kabul etmek için destek içerir `config.txt` giriş. Daha fazla bilgi için bkz: [giriş SAS belirteçler Hızlı Başlangıç](quickstart-input-sas.md). 

### <a name="version-073"></a>Sürüm 0.7.3
Sürüm 0.7.3 küçük hata düzeltmeleri içerir

### <a name="version-072"></a>Sürüm 0.7.2
Sürüm 0.7.2 ilk sürümdür. 1 Kasım 2017 yayımlanmıştır ve GATK 3.3 kullanır.
