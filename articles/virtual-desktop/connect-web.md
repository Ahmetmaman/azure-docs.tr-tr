# <a name="connect-with-the-web-client"></a>Web istemcisiyle bağlanma

Web istemcisi, uzun yükleme işlemi olmadan Windows Sanal Masaüstü kaynaklarınıza bir web tarayıcısından erişmenizi sağlar.

>[!NOTE]
>Web istemcisi şu anda mobil işletim sistemi desteğine sahip değil.

## <a name="supported-operating-systems-and-browsers"></a>Desteklenen işletim sistemleri ve tarayıcılar

Herhangi bir HTML5 özellikli tarayıcı çalışması gerekirken, resmi olarak aşağıdaki işletim sistemlerini ve tarayıcıları destekliyoruz.

| Tarayıcı           | Desteklenen İşletim Sistemi                     | Notlar               |
|-------------------|----------------------------------|---------------------|
| Microsoft Edge    | Windows                          |                     |
| Internet Explorer | Windows                          |                     |
| Apple Safari      | macOS                            |                     |
| Mozilla Firefox   | Windows, macOS, Linux            | Sürüm 55 veya sonraki |
| Google Chrome     | Windows, macOS, Linux, Chrome OS |                     |

## <a name="access-remote-resources-feed"></a>Uzak kaynaklar akışına erişin

Bir tarayıcıda, [Windows Sanal Masaüstü web istemcisine](https://rdweb.wvd.microsoft.com/webclient) gidin ve kullanıcı hesabınızla oturum açın.

>[!NOTE]
>Windows Sanal Masaüstü için kullanmak istediğinizden farklı bir Azure Active Directory hesabıyla zaten oturum açmışsanız, oturum açmanız veya özel bir tarayıcı penceresi kullanmanız gerekir.

Oturum açmadan sonra artık kaynakların listesini görmeniz gerekir. **Kaynakları, Tüm Kaynaklar** sekmesinde normal bir uygulama gibi seçerek kaynakları başlatabilirsiniz.

## <a name="next-steps"></a>Sonraki adımlar
