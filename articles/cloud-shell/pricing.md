# <a name="pricing"></a>Fiyatlandırma

## <a name="compute-cost"></a>Maliyet işlem
Azure bulut Kabuk ücretsiz Azure tarafından sağlanan bir makinede çalışır, ancak kullanmak için Azure dosya paylaşımının gerektirir. 

## <a name="storage-cost"></a>Depolama maliyeti
Bulut Kabuk yeni veya var olan Azure dosya paylaşımı oturumlarında dosyaları kalıcı hale getirmek için bağlanmasını gerektirir. Depolama normal maliyetler doğurur.

Denetleme [Azure dosyaları maliyetleri hakkında ayrıntılı bilgi için burada](https://azure.microsoft.com/pricing/details/storage/files/).
