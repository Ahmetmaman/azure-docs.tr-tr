# <a name="troubleshoot-azure-migrate"></a>Azure Geçişi sorunlarını giderme

[Azure Geçişi,](migrate-services-overview.md) değerlendirme ve geçiş için bir araç merkezinin yanı sıra üçüncü taraf bağımsız yazılım satıcısı (ISV) teklifleri sağlar. Bu makale, Azure Geçişi, Azure Geçir Sunucu Değerlendirmesi ve Azure Geçir Sunucu Geçişi ile ilgili sorunları gidermenize yardımcı olur.

## <a name="how-do-i-create-or-find-a-project"></a>Bir projeyi nasıl oluşturabilirim veya bulabilirim?

Azure [Geçiş proje sorun giderme kılavuzunu](troubleshoot-project.md)gözden geçirin.

## <a name="i-cant-get-the-appliance-working"></a>Cihazı çalıştıramıyorum.

Cihaz dağıtımıyla ilgili [sık karşılaşılan sorunlara yanıtları](troubleshoot-appliance-discovery.md) gözden geçirin.

## <a name="machines-arent-discovered"></a>Makineler keşfedilmemiş

Sık karşılaşılan bulma sorunlarını gözden geçirin.

## <a name="app-discovery-isnt-working"></a>Uygulama bulma çalışmıyor

Şirket içi makinelerde çalışan uygulamaların, rollerin ve özelliklerin keşfi şu anda yalnızca VMware VM'ler için desteklenir. Uygulama bulma için [sık karşılaşılan hataları gözden geçirin.](troubleshoot-appliance-discovery.md#common-app-discovery-errors)

## <a name="assessment-isnt-working"></a>Değerlendirme işe yaramıyor

Sık karşılaşılan değerlendirme sorunlarını ve hataları [gözden geçirin.](troubleshoot-assessment.md)
