# <a name="supported-devices"></a>Desteklenen cihazlar

[Azure portalı](https://portal.azure.com) web tabanlı bir konsoldur ve tüm modern masaüstü ve tablet aygıtlarının tarayıcısında çalışır. Portalı kullanmak için tarayıcınızda JavaScript etkinleştirilmiş olmalıdır.

Azure kaynaklarını bir mobil cihazdan yönetmeniz gerekiyorsa, [Azure mobil uygulamasını](https://azure.microsoft.com/features/azure-portal/mobile-app/)deneyin. iOS ve Android için kullanılabilir.

## <a name="recommended-browsers"></a>Önerilen tarayıcılar

İşletim sisteminizle uyumlu en güncel tarayıcıyı kullanmanızı öneririz. Aşağıdaki tarayıcılar desteklenir:

* Microsoft Edge (en son sürüm)
* Internet Explorer 11
* Safari (en so sürüm, yalnızca Mac)
* Chrome (en son sürüm)
* Firefox (en son sürüm)
