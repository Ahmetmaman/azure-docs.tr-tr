# <a name="setting-up-access-for-azure-dev-tools"></a>Azure dev araçları için erişimi ayarlama

Öğrencilerinize yazılım dağıtabilmeniz ve Laboratuvarlarınızı donabilmeniz için aboneliğinize erişmenin iki yolu vardır:
1. Yazılım ve anahtarları Visual Studio abonelik portalından indirerek.
1. Eğitim Merkezi deposuna öğrenciler için erişim izni veriliyor.

Daha fazla bilgi için [Azure for eğitim Web sitesini](https://azureforeducation.microsoft.com)ziyaret edin.

## <a name="for-administrators-only"></a>Yalnızca Yöneticiler için  
Yöneticiler, Azure dev araçlarına [Visual Studio abonelik portalı](https://my.visualstudio.com/)üzerinden erişir.

Visual Studio aboneliğinize erişim istemek için [Yönetim Portalı](https://azureforeducation.microsoft.com/account/Subscriptions)aracılığıyla bunu yapabilirsiniz. Onaylandığında, Visual Studio abonelikleri portalına erişebilirsiniz. Visual Studio abonelik portalına ilk kez erişiyorsanız, kaydolduğunuzda sağlanmış olan e-posta adresiyle bir Microsoft hesabı bağlamanız istenir. Abone KIMLIĞINIZ veya avantajları erişim numaranız, bir yöneticinin adı ve soyadı ile bu hesapları bağlamak için kayıt sırasında verilen e-posta gerekir. Hesaplarınızı bağlamadan sonra Visual Studio abone portalına erişebilirsiniz.

## <a name="for-students-faculty-and-administrators"></a>Öğrenciler, fakülte ve yöneticiler için
[Eğitim Merkezi](https://aka.ms/devtoolsforteaching)aracılığıyla Azure dev araçlarına erişin.

Öğrenciler ve fakülte, Eğitim Merkezi aracılığıyla tüm yazılım indirme avantajlarına erişim elde edebilir. Eğitim Merkezi, Azure portal içinde oluşturulmuştur ve öğrencilerinizin, tüm [Microsoft Learn](https://docs.microsoft.com/learn/) kataloğuna erişimi sağlamak için, tüm yazılım kataloğuna kolay erişim sağlar.
