# <a name="enrolling-students-in-azure-dev-tools-for-teaching"></a>Eğitim için Azure geliştirme araçları 'nda öğrenci kaydetme
Eğitim için Azure dev araçları satın aldıktan sonra, kurumunuzdaki etki alanında bulunan tüm öğrenciler otomatik olarak bu teklifin erişimine sunulacaktır. Satın alma işlemi sırasında, abonelik yöneticisinin e-posta adresini girmeniz istenir. Bu University-doğrulanan etki alanı, aynı e-posta etki alanına sahip öğrencilerle birlikte oturum açıp bunların yazılım indirme avantajlarından faydalanmalarını sağlar. Öğrenciler zaten bir Microsoft hesabına (MSA) sahip değilse veya bir üniversite Active Directory parçasıysa, kendi University e-postalarıyla eşleşen bir tane oluşturması istenir. Öğrenciler Eğitim Merkezi erişme sorunları yaşalıyorsa, aşağıdaki sorun giderme kılavuzumuzu gözden geçirin veya ' de bizimle iletişim kurun [https://aka.ms/studentsupport](https://aka.ms/studentsupport) .

## <a name="managing-access-for-students"></a>Öğrenciler için erişimi yönetme
Öğrenciler için Azure dev araçları 'nda oturum açmak için kullandıkları etki alanını temel alan öğrencilerin erişimini yönetebilirsiniz. Eğitim için Azure dev araçları için [Yönetim Portalı](https://azureforeducation.microsoft.com/en-us/account/Subscriptions) gidin. Burada tüm aboneliklerinizi görürsünüz. Erişimini yönetmek istediğiniz aboneliği seçin.
