# <a name="azure-for-students-starter"></a>Öğrenciler için Azure Başlangıç

Öğrenciler için Azure başlangıç, bulutta geliştirmeniz gereken Azure ürünlerini kullanmaya başlamanızı sağlar. Size herhangi bir maliyet yoktur. Bu avantaj, aşağıdaki hizmetlerin ücretsiz katmanına erişmenizi sağlar:

- Azure App Service
- Azure İşlevleri
- Azure Notification Hubs
- MySQL için Azure Veritabanı
- Application Insights
- Azure DevOps Server (eski adıyla Visual Studio Team Services)

Öğrenciler için Azure başlangıç, hiçbir ücret ödemeden ve taahhütte veya zaman sınırlaması olmadan doğrulanan öğrenciler tarafından kullanılabilir. Ayrıntılı kullanım koşulları için bkz. [öğrenciler Için Azure başlangıç teklifi](https://azure.microsoft.com/offers/ms-azr-0144p/) .

Bu teklif, büyük bir açık çevrimiçi kursla (MOOC) veya kar amacı gütmeyen kuruluşların diğer profesyonellerine yönelik olarak kullanılamaz.

Bu teklif, uygun müşteri başına öğrenciler için bir Azure başlangıç aboneliği ile sınırlıdır. Bu, Microsoft tarafından izin verilmediği takdirde aktarılamayan ve başka bir teklifle birleştirilemez.

Öğrenciler için Azure başlangıç teklifi için kredi kartı gerekli değildir. Bununla birlikte, [Azure hesap portalı](https://account.azure.com/)'nı kullanarak tüm Azure hizmetlerine erişim sağlamak için dilediğiniz zaman bir Kullandıkça Öde aboneliğine yükseltebilirsiniz.
