# <a name="azure-spatial-anchors-support-options"></a>Azure Uzamsal Çapalar destek seçenekleri

## <a name="open-a-tech-support-ticket"></a>Teknik destek bileti açın

Azure Uzamsal Çapalar için Azure Portalı'nda teknik destek bileti açmak için:

1. Azure [portalı](https://azure.microsoft.com/account/) açıkken, üst menü çubuğundan yardım simgesini seçin ve ardından **Yardım + destek** düğmesini seçin.

![Azure portalı yardım ve desteği](./media/spatial-anchor-support.png)

1. Yardım + destek sayfası açıkken + **Yeni destek isteğini**seçin.

![Azure portalı yeni destek isteği](./media/spatial-anchor-support2.png)

1. Destek bilet alanlarını tamamlarken:

- Sorun türü: Teknik
- Hizmet: Uzamsal Çapalar

![Azure portalı destek bilet alanları](./media/spatial-anchor-support3.png)

## <a name="community-support"></a>Topluluk desteği

Kullanıcı topluluğundan destek için, [yığın taşmada Azure Uzamsal Çapalar'a](https://stackoverflow.com/questions/tagged/azure-spatial-anchors)bakın.

## <a name="provide-content-feedback"></a>İçerik geri bildirimi sağlama

Her içerik makalesinin altında, bir GitHub sorunu açma ve Azure Uzamsal Bağlantı belgeleri içeriği hakkında geri bildirim sağlama fırsatı vardır.

## <a name="provide-product-feedback"></a>Ürün geri bildirimi sağlama

Geri bildirim sağlamak, Azure Uzamsal Çapalar hizmeti için bir fikir veya öneri paylaşmak veya başkalarının gönderdiği fikirleri oylamak için [Azure Uzamsal Çapalar Geri Bildirim Forumu'nu](https://feedback.azure.com/forums/919252-azure-spatial-anchors)ziyaret edin.

## <a name="next-steps"></a>Sonraki adımlar

Azure Uzamsal Bağlantılar hakkında sık sorulan sorular için [SSS](spatial-anchor-faq.md) sayfasına bakın.
