# <a name="deprecated-configure-baidu-cloud-push-settings-for-a-notification-hub-in-the-azure-portal"></a>Amortismana uğradı: Azure portalındaki bir bildirim merkezi için Baidu Cloud Push ayarlarını yapılandırın

Bu makalede, Azure portalını kullanarak Bir Azure bildirim merkezi için Baidu Bulut Push ayarlarını nasıl yapılandırabileceğiniz gösterilmektedir.

> [!IMPORTANT]
> Bu öğretici amortismana uğradı. 

## <a name="prerequisites"></a>Ön koşullar
Daha önce bir bildirim hub'ı oluşturmadıysanız, şimdi bir bildirim hub'ı oluşturun. Daha fazla bilgi için [bkz.](create-notification-hub-portal.md) 

## <a name="configure-baidu-cloud-push"></a>Baidu Cloud Push'ı yapılandırma
Aşağıdaki yordam, bir bildirim hub'ı için Baidu Bulut Push ayarlarını yapılandırmanız için adımlar verir:

1. Azure portalında, **Bildirim Merkezi** sayfasında sol menüde **Baidu (Android China)** seçeneğini belirleyin. 
2. Baidu bulut itme projesinde Baidu konsolundan aldığınız **Api Anahtarını** girin. 
3. Baidu bulut itme projesinde Baidu konsolundan aldığınız **Gizli Anahtarı** girin. 
4. **Kaydet'i**seçin. 

    ![Anında iletme bildirimleri için Baidu (Android China) yapılandırmasını gösteren Bildirim Hub'larının ekran görüntüsü](./media/notification-hubs-baidu-get-started/AzureNotificationServicesBaidu.png)
