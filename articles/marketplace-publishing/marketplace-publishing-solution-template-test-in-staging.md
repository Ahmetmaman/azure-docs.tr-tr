# <a name="test-your-solution-template-offer-in-staging"></a>Çözüm şablonu teklifinizi test, hazırlama
Hazırlama, özel bir "korumalı alan burada test edebilir ve üretime göndermeden önce işlevselliğini doğrulayın" aşamasında teklifinizi dağıtma anlamına gelir. Teklif hazırlama dağıtmış olan bir müşteri için olduğu gibi görüntülenir. Teklifinizi hazırlamaya için sertifika gerekir.

Teklif hazırlanan sonra görüntüleyebilir ve test teklifi [Azure portalı](https://portal.azure.com/).

Teklifinizi hazırlamaya Gönder ve bunu test etmek için aşağıdaki adımları [Azure portalı](https://portal.azure.com/):

1. Git [yayımlama portalı](https://publish.windowsazure.com) > **çözüm şablonları** sekmesi > teklifinizi > **Yayımla** > **hazırlamaiçinanındailetme**.
2. Önizleme ve teklifinizi sınamak için kullanacağınız Azure abonelikleri listesini sağlayın.
3. Önceki adımda kullanılan abonelik Kimliğini kullanarak Azure preview Portal'da oturum açın.
4. Azure Önizleme portalında aşağıda belirtilen noktalarında test en az bir gidiş yürütmek:
   * Pazarlama içeriği doğru Azure Marketi'nde göründüğünü doğrulayın.
   * Uçtan uca dağıtım topolojisinin.
   * Performans testi ve stres testi gerçekleştirin.
   * Topolojiniz için en iyi uyduğundan emin olun.

## <a name="next-steps"></a>Sonraki adımlar
