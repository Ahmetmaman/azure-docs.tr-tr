# <a name="get-pre-approved-via-the-microsoft-azure-certified-program"></a>Microsoft Azure sertifikası programı önceden onaylanan
Bu makalede, Microsoft Azure sertifikası Azure Marketi ile çözümünüzü yayımlayın ve pazarlama avantajlar için uygun hale hakkında yönergeler açıklanmaktadır. Microsoft Azure sertifikası programı ve faydalarını ayrıntılı bakış için ziyaret [ http://azure.com/certified ](http://azure.com/certified).

> [!IMPORTANT]
> Azure sertifikalarıyla, yalnızca Azure Market sanal makine ya da bir çözüm şablonu oluşturmak için kullanacaksanız geçerlidir.

## <a name="getting-started"></a>Başlarken
1. Sertifikalı çözümünüz ve eklenen Azure Marketi alma yolunda ilk adımı, üzerinde uygulama formu doldurun olmaktır [Microsoft Azure sertifikası](https://createopportunity.azurewebsites.net) Web sayfası.
2. Uygunluk ölçütleri karşılayan ve uygulamanızın onaylanan, yerleşik çözümünüzü Azure Marketi için sizinle çalışma başlayacağız.
3. Bir iş ortağı Hesap Yöneticisi veya DX ortak yönetici ile çalışıyorsanız, çözümünüzü Azure sertifikası için aday gösterin isteyin program veya Git [Microsoft Azure sertifikası](http://createopportunity.azurewebsites.net) Web sayfasını, uygulama formu doldurun ve girin e-posta iş ortağı Hesap Yöneticisi veya DX ortak Yöneticisi'nde Microsoft Sponsor kişinin alan.

> [!IMPORTANT]
> **"Satış yapan" ülkeleri** - onaylanan "satış yapan" ülkeleri biri kayıtlı varlık gereksinimlerinizi hizmetlerinizi Microsoft Azure Marketi'nde satmak için. Bu kısıtlama ödeme ve vergi amaçlıdır. Etkin bir şekilde arıyoruz yakın gelecekte bu ülkelerde listesini genişletin, bu nedenle Takipte kalın. Bölüm 1b, tam listesi için bkz. [Azure Marketi katılım ilkeleri](https://go.microsoft.com/fwlink/?LinkID=526833).
