# <a name="recommendations-api-discontinued"></a>Öneriler API (dönüştürülmesine)

Öneriler API Önizleme artık etkin geliştirilme aşamasındadır ve resmi olarak 15 Şubat devre dışı bırakılan. 

Yeni açık kaynaklı bir alternatif için göz önünde bulundurun [önerileri çözüm şablonu](http://aka.ms/recopcs), olanak sağlayan, öneriler altyapısı Azure aboneliğinize içinde ana bilgisayar. Ayrıca, herhangi bir Azure bölgesine dağıtmak için özgürlüğünü sırasında önerileri API çekirdek senaryolarını destekler ve modelleri eğitme ve öneriler sunmak için kullanılan hesaplama kaynaklarını daha yakından denetler. 

Okuyarak önerileri çözüm şablonu hakkında daha fazla bilgi [belgelerine](https://github.com/Microsoft/Product-Recommendations).
