# <a name="speech-sdk-api-reference"></a>Konuşma SDK API Başvurusu

Konuşma Yazılım Geliştirme Seti (SDK), konuşma özellikli yazılım geliştirmeyi kolaylaştıran konuşma hizmeti işleve, uygulamaları erişmenizi sağlar. Şu anda, SDK'ları erişmeyi **Konuşmayı metne dönüştürme**, **konuşma çevirisi**, ve **amaç tanıma**. Genel bir bakış özelliklere ve desteklenen platformlar hakkında belgeler bulunabilir [giriş sayfası](https://aka.ms/csspeech).
