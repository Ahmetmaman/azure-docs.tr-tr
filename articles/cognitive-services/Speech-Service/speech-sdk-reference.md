# <a name="speech-sdk-reference"></a>Konuşma SDK'sı başvurusu

Konuşma Yazılım Geliştirme Seti (SDK) konuşma etkin yazılım geliştirmek daha kolay uygulamaları erişiminizi konuşma hizmeti işlevleri sağlar. Şu anda, SDK'ları erişebilmesi için **metin konuşma** ve **konuşma çeviri**. Metin okuma kullanır [REST] (https://docs.microsoft.com/azure/cognitive-services/speech-service/rest-apis) HTTP üzerinden POST çağrıları.
