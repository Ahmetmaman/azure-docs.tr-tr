### <a name="nodejs-package-manager-npm"></a>Node.js Paket Yöneticisi (NPM)

JavaScript için konuşma SDK 'sını yüklemek için aşağıdaki `npm install` komutu çalıştırın.

```nodejs
npm install microsoft-cognitiveservices-speech-sdk
```

Daha fazla bilgi için <a href="https://github.com/Azure-Samples/cognitive-services-speech-sdk/tree/master/quickstart/javascript/node" target="_blank"> konuşma SDK 'sı hızlı başlangıçNode.js </a>bakın.
