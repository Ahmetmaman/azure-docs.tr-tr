#### <a name="c-nuget-package"></a>C# NuGet paketi

Aşağıdaki `dotnet add` komutla birlikte **.NET Core CLI** .net konuşma SDK 'sı de yüklenebilir.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET konuşma SDK 'Sı, aşağıdaki `Install-Package` komutla **paket yöneticisinden** yüklenebilir.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
