#### <a name="c-nuget-package"></a>C# NuGet Paketi

.NET Speech SDK aşağıdaki `dotnet add` komutla **.NET Core CLI'den** yüklenebilir.

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET Konuşma SDK aşağıdaki `Install-Package` komutu ile Paket **Yöneticisi'nden** yüklenebilir.

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
