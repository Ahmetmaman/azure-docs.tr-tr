#### <a name="c-nuget-package"></a>C# NuGet paketi

Aşağıdaki komutla birlikte **.NET Core CLI** .net konuşma SDK 'sı de yüklenebilir `dotnet add` .

```dotnetcli
dotnet add package Microsoft.CognitiveServices.Speech
```

.NET konuşma SDK 'Sı, aşağıdaki komutla **paket yöneticisinden** yüklenebilir `Install-Package` .

```powershell
Install-Package Microsoft.CognitiveServices.Speech
```
