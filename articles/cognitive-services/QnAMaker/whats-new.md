# <a name="whats-new-in-qna-maker"></a>QnA Maker'daki yenilikler

Hizmetteki yenilikleri öğrenin. Bu öğeler notlar, videolar, blog gönderileri ve diğer bilgi türlerini yayımlayabilir. Hizmetten haberdar olmak için bu sayfayı yer imi.

## <a name="release-notes"></a>Sürüm notları

QnA Maker ile ilgili yenilikleri öğrenin.

### <a name="march-2020"></a>Mart 2020

* TLS 1.2 artık bu hizmete tüm HTTP istekleri için uygulanır. Daha fazla bilgi için Azure [Bilişsel Hizmetler güvenliği'ne](../cognitive-services-security.md)bakın.

### <a name="february-2020"></a>Şubat 2020

* GenerateAnswer API ile [NPM paketi](https://www.npmjs.com/package/@azure/cognitiveservices-qnamaker)

### <a name="november-2019"></a>Kasım 2019

* [QnA](https://docs.microsoft.com/azure/azure-government/documentation-government-services-aiandcognitiveservices#qna-maker) Maker için ABD Hükümeti bulut desteği
* GA'da [çoklu dönüş](https://docs.microsoft.com/azure/cognitive-services/qnamaker/how-to/multiturn-conversation) özelliği
* Seviye-1 dillerinde [kullanılabilen Chit-chat desteği](https://docs.microsoft.com/azure/cognitive-services/qnamaker/how-to/chit-chat-knowledge-base#language-support)

### <a name="october-2019"></a>Ekim 2019

* QnA Maker hizmetindeki tüm bilgi tabanlarının [dilini açıkça ayarlım.](./how-to/language-knowledge-base.md#select-language-when-creating-first-knowledge-base)

### <a name="september-2019"></a>Eylül 2019

* [XLS dosya biçimiyle](concepts/content-types.md) alma ve dışa aktarma

### <a name="june-2019"></a>Haziran 2019

* Fransızca, İtalyanca, Almanca, İspanyolca, Portekizce için geliştirilmiş [sıralama modeli](concepts/query-knowledge-base.md#ranker-process)

### <a name="april-2019"></a>Nisan 2019

* Destek web sitesi içerik çıkarma
* Kimlik doğrulaması erişimden [SharePoint belge](how-to/add-sharepoint-datasources.md) desteği

### <a name="march-2019"></a>Mart 2019

* [Aktif öğrenme,](how-to/improve-knowledge-base.md) gerçek kullanıcı sorularına dayalı yeni soru alternatifleri için öneriler sunar
* İngilizce için geliştirilmiş doğal dil işleme (NLP) [ranker](concepts/query-knowledge-base.md#ranker-process) modeli
