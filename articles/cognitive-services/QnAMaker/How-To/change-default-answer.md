# <a name="set-default-answer-for-a-knowledge-base"></a>Bilgi tabanı için varsayılan yanıtı ayarlama

Varsayılan yanıt, soruyla eşleşme olmadığında döndürülür. Varsayılan yanıtı standart varsayılan yanıttan değiştirmek isteyebilirsiniz.

## <a name="change-default-answer"></a>Varsayılan Yanıtı Değiştir

1. [Azure portalına](https://portal.azure.com) gidin ve oluşturduğunuz QnA Maker hizmetini temsil eden kaynak grubuna gidin.

2. **Uygulama Hizmetini**açmak için tıklayınız.

    ![Azure portalında, QnA Maker için Uygulama hizmetine erişin](../media/qnamaker-concepts-confidencescore/set-default-response.png)

3. Uygulama **Ayarları'nı** tıklatın ve **Varsayılan Yanıt** alanını istenen varsayılan yanıta göre düzeltin. **Kaydet**'e tıklayın.

    ![Uygulama Ayarlarını seçin ve Ardından QnA Maker için Varsayılan Yanıt'ı](../media/qnamaker-concepts-confidencescore/change-response.png)

4. Uygulama hizmetinizi yeniden başlatın

    ![DefaultAnswer'ı değiştirdikten sonra QnA Maker uygulamasını yeniden başlatın](../media/qnamaker-faq/qnamaker-appservice-restart.png)
