# <a name="publish-a-knowledge-base"></a>Bilgi bankası yayımlama

Bilgi Bankası yayımlama, Bilgi Bankası soruyu yanıtlarken bir uç nokta olarak kullanılabilir hale getirme son adımdır. 

Bilgi Bankası yayımladığınızda, Bilgi Bankası QnA içeriğini taşır test dizinden Azure Search'te bir üretim dizini.

![Üretim test dizin yayımlama](../media/qnamaker-how-to-publish-kb/publish-prod-test.png)

## <a name="publish-a-knowledge-base"></a>Bilgi bankası yayımlama

1. KB değişiklikleri yaptıktan sonra seçin **Yayımla** üst gezinti çubuğunda. Azure Arama'ya ilişkin Bilgi Bankası ayrılan sayısı kadar yayımlayabilirsiniz. 

    ![Bilgi Bankası yayımlama](../media/qnamaker-how-to-publish-kb/publish.png)

2. Seçin **Yayımla** yeniden uygulama veya bot kodunuzda kullanılabilir uç nokta ayrıntılarını görmek için.

    ![Bilgi Bankası yayımlama](../media/qnamaker-how-to-publish-kb/publish-success.png)
