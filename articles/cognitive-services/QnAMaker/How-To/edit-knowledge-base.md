# <a name="edit-a-knowledge-base"></a>Bilgi Bankası Düzenle

QnA Maker, kullanımı kolay düzenleme deneyimi sağlayarak, Bilgi Bankası içeriğini yönetmenize olanak sağlar.

## <a name="edit-your-knowledge-base-content"></a>Bilgi Bankası içeriğinizi Düzenle

1.  Seçin **My Bilgi Bankası** üst gezinti çubuğunda. 

    Oluşturulan ya da azalan sırada sıralanır sizinle paylaşılan tüm hizmetleri görebilirsiniz **son değiştirilme tarihi** tarihi.

    ![My Bilgi Bankası](../media/qnamaker-how-to-edit-kb/my-kbs.png)

2. Düzenlemeleri yapmak için belirli bir Bilgi Bankası seçin.

3. Bilgi Bankası değişiklikleri tamamladıktan sonra tıklayın **kaydedin ve eğitmek** değişiklikleri kalıcı hale getirmek için sayfanın sağ üst köşesinde içinde.    

    ![Kaydet ve eğitim](../media/qnamaker-how-to-edit-kb/save-and-train.png)

    >[!NOTE]
    Sayfanın tıklatmadan önce kaydederken bırakarak ve tren değişiklikler kalıcı olmaz.

## <a name="add-a-qna-pair"></a>QnA çift Ekle

Seçin **ekleme QnA çifti** Bilgi Bankası tabloya yeni bir satır eklemek için.

![QnA çift Ekle](../media/qnamaker-how-to-edit-kb/add-qnapair.png)

## <a name="delete-a-qna-pair"></a>QnA çiftini Sil

Bir QnA silmek için tıklatın **silmek** simgesine sağ ucundaki QnA satır.

![QnA çiftini Sil](../media/qnamaker-how-to-edit-kb/delete-qnapair.png)

## <a name="add-alternate-questions"></a>Diğer sorular ekleyin

Diğer sorular kullanıcının sorguda bir eşleşme olasılığını artırmak için varolan QnA çiftini ekleyin.

![Diğer sorular ekleyin](../media/qnamaker-how-to-edit-kb/add-alternate-question.png)

## <a name="add-metadata"></a>Meta verileri ekleme


Meta veri çiftleri filtre simgesini seçerek ekleyin

![Meta verileri ekleme](../media/qnamaker-how-to-edit-kb/add-metadata.png)

> [!TIP]
> Düzenli aralıklarla kaydetmek ve değişiklikleri kaybetmek istemiyorsanız Düzenlemeleri yaptıktan sonra Bilgi Bankası eğitmek emin olun.

## <a name="manage-large-knowledge-bases"></a>Büyük Bilgi Bankası yönetme

1. QnAs olan **gruplandırılmış** bunlar ayıklanan veri kaynağı tarafından. Genişletin veya veri kaynağı daraltın.
2. Yapabilecekleriniz **arama** Bilgi Bankası tablonun üstündeki metin kutusuna yazarak Bilgi Bankası. Soru, yanıt veya meta veri içerik aramak için tıklatın girin. Arama filtresi kaldırmak için X simgesini tıklatın.
3. **Sayfa numaralandırma** büyük Bilgi Bankası yönetmenize olanak sağlar

    ![Arama, sayfalara bölme, Grup](../media/qnamaker-how-to-edit-kb/search-paginate-group.png)
