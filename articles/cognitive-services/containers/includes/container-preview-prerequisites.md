## <a name="prerequisites"></a>Ön koşullar

* Azure aboneliği kullanın. Azure aboneliğiniz yoksa, başlamadan önce [ücretsiz][free-account] bir hesap oluşturun.
* Azure [CLI'yi][azure-cli] (az) yükleyin.
* [Docker motoru][docker-engine] ve Docker CLI bir konsol penceresinde çalıştığını doğrulayın.

[free-account]: https://azure.microsoft.com/free
[azure-cli]: https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest
[docker-engine]: https://www.docker.com/products/docker-engine
