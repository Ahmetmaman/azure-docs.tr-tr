# <a name="quickstart-use-the-rest-apis-to-customize-your-solution"></a>Hızlı başlangıç: çözümünüzü özelleştirmek için REST API 'Lerini kullanma

Ölçüm Danışmanı REST API kullanmaya başlayın. Paketi yüklemek için bu adımları izleyin ve temel görevler için örnek kodu deneyin.

Şunları yapmak için ölçüm Danışmanı 'nı kullanın:

* Veri kaynağından veri akışı ekleme
* Alma durumunu denetle
* Algılama ve Uyarıları yapılandırma 
* Anomali algılama sonuçlarını sorgulama
* Bozukluklar tanılayın
