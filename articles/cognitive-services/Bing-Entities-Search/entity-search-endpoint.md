# <a name="entity-search-endpoints"></a>Varlık arama uç noktaları
**Varlık arama API** bir uç nokta içerir.

## <a name="endpoint"></a>Uç Nokta
Varlık arama sonuçları istemek için aşağıdaki uç noktası için bir istek gönderin. Üstbilgiler ve URL parametreleri daha ayrıntılı belirtimleri tanımlamak için kullanın.

Uç nokta `GET`: 
``` 
https://api.cognitive.microsoft.com/bing/v7.0/entities
```

Aşağıdaki URL parametreleri gereklidir:
- Mkt. Sonuçları alınacağı yeri Pazar. 
- q. Varlık arama sorgusu.
