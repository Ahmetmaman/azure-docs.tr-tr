# <a name="how-to-sign-up-for-the-microsoft-translator-speech-api"></a>Microsoft Çeviricisi konuşma API için kaydolma

Microsoft Çeviricisi konuşma API için kaydolmak için aşağıdaki adımları izleyin.

## <a name="1-sign-into-azure"></a>1: oturum Azure açın   
- Hesabınız yok mu? [Microsoft Azure hesabı için kaydolun](http://azure.com/)
- Bir hesabınız zaten var? [Oturum Aç](http://portal.azure.com/)

## <a name="2-subscribe-to-microsoft-translator"></a>2: Microsoft çevirmene abone olma     
- Azure'da oturum sonra Git [Bilişsel Hizmetler](https://portal.azure.com/#create/Microsoft.CognitiveServices) bölümü
- "API türü" altında Çeviricisi konuşma API seçin
- Fiyatlandırma katmanı bölümünde gereksinimlerine en uygun fiyatlandırma katmanı seçin
    - Her abonelik ücretsiz bir katman vardır. Ücretsiz katmanı aynı özellikler ve İşlevler Ücretli planlar olarak var ve sona erme tarihi yok.
    - Hesabınızda yalnızca bir ücretsiz aboneliğiniz olabilir
- Kalan formu doldurun ve Oluştur düğmesine seçin
- Abonelik hemen kullanıma hazır.

## <a name="3-authentication-key"></a>3: kimlik doğrulama anahtarı    
- Tüm kaynaklar menüsüne giderek, kimlik doğrulama anahtarı almak > aboneliğinizi tıklatın
- Anahtar değer kimlik doğrulaması için kullanılan, anahtar değer gizli kalmasını. Uygulamanızı geliştirirken anahtar değeri gerekiyor
