# <a name="what-is-the-linguistic-analysis-api-preview"></a>Dilbilimsel Analiz API'si nedir? (Önizleme)

> [!IMPORTANT]
> Dilbilimsel Analiz önizleme sürümü 9 Ağustos 2018 tarihinde kullanımdan kaldırılmıştır. Metin işleme ve analiz için [Azure Machine Learning metin analizi modüllerini](https://docs.microsoft.com/azure/machine-learning/studio-module-reference/text-analytics) kullanmanızı öneririz.

Dilbilimsel Analiz API'si, metnin yapısını tanımlayan doğal dil işleme (NLP) araçlarına erişim sağlar. Geçerli sürüm üç farklı analiz türü sunar:

- [Tümce ve belirteçlere ayırma](Sentences-and-Tokens.md)
- [Konuşma bölümü etiketleme](POS-tagging.md)
- [Hedef kitle ayrıştırma](Constituency-Parsing.md)

## <a name="getting-started"></a>Başlarken

Yukarıda listelenen temel analiz türleri hakkında bilgi edindikten sonra uç noktaya bakarak [kullanılabilir çözümleyicilerin listesini](AnalyzersMethod.md), [adlarını](Analyzer-Names.md) ve [yeni giriş çözümleme](AnalyzeMethod.md) uç noktasını alabilirsiniz.
