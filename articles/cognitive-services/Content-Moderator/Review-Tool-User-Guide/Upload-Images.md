# <a name="upload-images"></a>Resimleri Yükleyin

Kullanıcı arabirimini keşfetmek için en fazla beş görüntüleri birer birer, 2 MB dosya boyutu aşmayacak şekilde her bir görüntü karşıya yükleyin. Karşıya yüklemeden önce dosya boyutlarınızın denetlediğinizden emin olun.

![Resimleri Yükleyin](images/1-try-image-1.png)

## <a name="submit-for-moderation"></a>Denetimi için gönderme
Aynı anda beşten fazla görüntü karşıya yukarı gerekmesi halinde istenmeyen görüntülerini kaldırmak için Çöp Kutusu simgesini kullanın. Ardından, otomatik denetimden görüntülerde gönderin.

![Görüntüleri gönderme](images/1-try-image-2.png)
