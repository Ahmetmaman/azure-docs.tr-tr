# <a name="bing-visual-search-sdk"></a>Bing Visual arama SDK

Bing Visual arama aşağıdaki istemci Visual arama API uygulamanız için kullanabileceğiniz SDK'ları sağlar.

* [.NET örnekleri](https://github.com/Azure-Samples/cognitive-services-dotnet-sdk-samples/tree/master/BingSearchv7)
    * [NuGet paketi](https://www.nuget.org/packages/Microsoft.Azure.CognitiveServices.Search.VisualSearch/1.2.0)
    * [.NET kitaplıklarına](https://github.com/Azure/azure-sdk-for-net/tree/psSdkJson6/src/SDKs/CognitiveServices/dataPlane/Search/BingVisualSearch) tanımları ve bağımlılıklar için
* [Node.js örnekleri](https://github.com/Azure-Samples/cognitive-services-node-sdk-samples) 
    * [Node.js kitaplıkları](https://github.com/Azure/azure-sdk-for-node/tree/master/lib/services/imageSearch) tanımları ve bağımlılıklar için
* [Java örnekleri](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples) 
    * [Java kitaplıkları](https://github.com/Azure-Samples/cognitive-services-java-sdk-samples/tree/master/Search) tanımları ve bağımlılıklar için
* [Python örnekleri](https://github.com/Azure-Samples/cognitive-services-python-sdk-samples) 
    * [Python kitaplıkları](https://github.com/Azure/azure-sdk-for-python/tree/master/azure-cognitiveservices-search-imagesearch) tanımları ve bağımlılıklar için

Her dil için SDK örnek örnekleri çalıştırma ve yükleme için SDK adımları ile birlikte kullanma önkoşulları bulunduğu bir benioku dosyası içerir.
