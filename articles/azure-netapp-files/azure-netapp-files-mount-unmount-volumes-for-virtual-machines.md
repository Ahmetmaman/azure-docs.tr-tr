# <a name="mount-or-unmount-a-volume-for-windows-or-linux-virtual-machines"></a>Bağlamak veya bir birimi Windows veya Linux sanal makineleri için çıkarma 

Bağlamak veya bir birimi Windows veya Linux sanal makineleri için gerektiği şekilde çıkarın.  Linux sanal makineleri bağlama yönergelerini Azure NetApp dosya kullanılabilir.  

1. Tıklayın **birimleri** dikey penceresinde ve ardından bağlamak istediğiniz bir birimi seçin. 
2. Tıklayın **bağlama yönergeleri** seçili birimden ve ardından birimi bağlamak için yönergeleri izleyin. 

    ![NFS bağlama yönergeleri](../media/azure-netapp-files/azure-netapp-files-mount-instructions-nfs.png)

    ![SMB bağlama yönergeleri](../media/azure-netapp-files/azure-netapp-files-mount-instructions-smb.png)
