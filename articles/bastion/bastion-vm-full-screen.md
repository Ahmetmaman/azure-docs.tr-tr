# <a name="change-to-full-screen-view-for-a-vm-session-azure-bastion"></a>VM oturumu için tam ekran görünümüne değiştirme: Azure Bastion

Bu makale, sanal makine görünümünü tam ekrana ve tarayıcınızda değiştirmenize yardımcı olur. Bir VM ile çalışmadan önce, [bir Bastion ana bilgisayar oluşturmak](bastion-create-host-portal.md)için adımları izlediğinden emin olun. Ardından, [RDP](bastion-connect-vm-rdp.md) veya [SSH](bastion-connect-vm-ssh.md)kullanarak çalışmak istediğiniz VM'ye bağlanın.

## <a name="launch-the-clipboard-tool"></a>Pano aracını başlatın

Uzak oturum sırasında, oturumun sol merkezinde bulunan iki ok seçerek Bastion pano erişim aracı paletini başlatın.

![araçlar](./media/bastion-vm-manage/left.png)

## <a name="select-full-screen"></a>Tam ekran seçin

Oturumu tam ekran deneyimine geçmek için **Tam ekran** düğmesini seçin. Geçiş yaptığınızda, oturum tam ekrana yeniden başlayacaktır.

![tam ekran](./media/bastion-vm-manage/full-screen.png)
