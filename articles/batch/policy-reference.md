# <a name="azure-policy-built-in-definitions-for-azure-batch"></a>Azure Batch için Azure Ilkesi yerleşik tanımları

Bu sayfa, Azure Batch için [Azure ilkesi](../governance/policy/overview.md) yerleşik ilke tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../governance/policy/samples/built-in-policies.md)bölümüne bakın.
