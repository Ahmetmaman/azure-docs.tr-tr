# <a name="azure-policy-built-in-definitions-for-azure-api-for-fhir"></a>FHAR Azure API 'SI için Azure Ilkesi yerleşik tanımları

Bu sayfa, FHıR [Için Azure API 'sinin yerleşik ilke](../governance/policy/overview.md) tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../governance/policy/samples/built-in-policies.md)bölümüne bakın.

Her yerleşik ilke tanımının adı Azure portal ilke tanımına bağlanır. Kaynağı [Azure Policy GitHub](https://github.com/Azure/azure-policy)deposunda görüntülemek için **Sürüm** sütunundaki bağlantıyı kullanın.
