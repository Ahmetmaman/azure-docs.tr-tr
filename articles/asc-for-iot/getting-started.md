# <a name="get-started-with-azure-security-center-for-iot"></a>IoT için Azure Güvenlik Merkezi’ni kullanmaya başlama

Bu makalede, IoT için Azure Güvenlik Merkezi'nin farklı bileşenlerinin bir açıklaması ve iki olası dağıtım seçeneği kullanılarak hizmete nasıl başlatıyapılacağını açıklanmaktadır.

## <a name="deployment-options"></a>Dağıtım seçenekleri

IoT aygıtınızı ve ortam gereksinimlerinizi en iyi karşılayan hizmet senaryosunu seçin.

### <a name="built-in-deployment"></a>Yerleşik dağıtım

Sorunsuz, yerleşik dağıtım seçeneğini kullanarak, IoT için Azure Güvenlik Merkezi IoT Hub'ınıza hızla entegre edilebilir ve IoT hub yapılandırması, aygıt kimliği ve yönetimi ve hub-aygıt iletişim kalıplarının güvenlik analizini sağlayabilir.

IoT Hub izleme ve öneriler içeren [yerleşik bir dağıtım](iot-hub-integration.md) başlatın.
    <br>

### <a name="enhanced-deployment"></a>Geliştirilmiş dağıtım

Gelişmiş güvenlik yetenekleri için, IoT Hub güvenliğini etkinleştirmeye ek olarak IoT aracıları için Azure Güvenlik Merkezi'ni dağıtmak, IoT aygıtlarınızdaki önemli güvenlik verilerinin aracı tabanlı olarak toplanması, analizi ve tehdit algılamasının yanı sıra kapsamlı güvenlik duruş yönetimi yeteneklerini de sağlar.

Aracı tabanlı kapsamlı tehdit koruması ve güvenlik duruşu yönetimi çözümü içeren [Gelişmiş bir dağıtım](security-agents.md) başlatın.
