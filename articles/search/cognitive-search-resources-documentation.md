# <a name="documentation-resources-for-cognitive-search-workloads"></a>Bilişsel arama iş yükleri için belge kaynakları

Bilişsel arama, artık genel önizlemede olan metin olmayan kaynakları ve Azure Search'te tam metin aranabilir içeriğe dönüştürmek protokole metin görünmeyen bilgi bulduğu Azure arama dizini oluşturma, bir yeni zenginleştirme katmanıdır.

Bilişsel arama için kapsamlı belgeler makaleleridir.
