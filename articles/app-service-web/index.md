    <h1>Web Apps Belgeleri</h1>
    <p style="padding-bottom: 0px; max-width: 1050px;">Azure Web Apps, altyapı yönetimine gerek kalmadan kendi seçtiğiniz programlama dilinde web siteleri oluşturmanıza ve barındırmanıza olanak tanır. Otomatik ölçeklendirme ve yüksek kullanılabilirlik sunar, hem Windows hem de Linux’ı (önizleme) destekler ve GitHub, Team Services veya herhangi bir Git deposundan otomatik dağıtımlar sağlar. Hızlı başlangıç kılavuzlarımız, öğreticilerimiz ve örneklerimizle, Azure Web Apps’i kullanmayı öğrenin.</p>
