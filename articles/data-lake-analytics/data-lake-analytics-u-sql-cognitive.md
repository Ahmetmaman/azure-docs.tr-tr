# <a name="get-started-with-the-cognitive-capabilities-of-u-sql"></a>U-SQL Bilişsel özellikleriyle çalışmaya başlama

## <a name="overview"></a>Genel Bakış
U-SQL bilişsel özellikler, geliştiricilerin kendi büyük veri programlarında put zeka kullanarak belgelerinizden olanak sağlar. 

Aşağıdaki örnekler bilişsel yetenekleri kullanarak büyük/küçük harf kullanılabilir:
* Görüntüleme: [yüzleri algılayın](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Görüntüleme: [duygu algılama](https://github.com/Azure-Samples/usql-cognitive-imaging-emotion-detection-hello-world)
* Görüntüleme: [Algıla (etiketleme) nesneleri](https://github.com/Azure-Samples/usql-cognitive-imaging-object-tagging-hello-world)
* Görüntüleme: [OCR (optik karakter tanıma)](https://github.com/Azure-Samples/usql-cognitive-imaging-ocr-hello-world)
* Metin: [anahtar ifade ayıklama ve yaklaşım analizi](https://github.com/Azure-Samples/usql-cognitive-text-hello-world)
