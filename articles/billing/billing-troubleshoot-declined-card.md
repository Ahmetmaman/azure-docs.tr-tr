# <a name="troubleshoot-a-declined-card-at-azure-sign-up"></a>Azure kaydı 'nda reddedilen bir kartın sorunlarını giderme

Bu makale, bir kredi kartının Azure portal veya Azure Hesap Merkezi ' nde Azure 'da oturum açma sırasında reddedildiğini sorunları çözmenize yardımcı olur. Sorunu gidermeye başlamadan önce aşağıdaki noktaları kontrol edin:

- İletişim e-postası, adresi ve telefon numarası gibi Azure Hesap profiliniz için verdiğiniz bilgilerin doğru olduğundan emin olun.
- Kredi kartı bilgilerinin doğru olduğundan emin olun.
- Aynı bilgilerle oluşturulmuş başka bir Microsoft hesabınızın olmadığından emin olun.
- Borç kartları kabul edilmez

## <a name="issues"></a>Sorunlar

Aşağıda, Azure kayıt sırasında kredi kartının reddedilmesine neden olabilecek yaygın sorunlar listelenmektedir.

### <a name="the-credit-card-provider-is-not-accepted-for-your-country"></a>Ülkeniz için kredi kartı sağlayıcısı kabul edilmedi

Bir kart seçtiğinizde, Azure seçtiğiniz ülkede geçerli olan kart seçeneklerini görüntüler. Kredi kartınızın uluslararası işlemler için etkinleştirildiğini doğrulamak için bankanızla veya kart vereninizle iletişim kurun. Desteklenen ülkeler ve para birimleri hakkında daha fazla bilgi için bkz. [Azure satın alma SSS](https://azure.microsoft.com/pricing/faq/).

>[!Note]
>American Express kredi kartları şu anda Hindistan 'da ödeme gereci olarak desteklenmemektedir. Kabul edilebilir bir ödeme formu olabileceği zaman çerçeveceğiz.

### <a name="youre-using-a-virtual-or-prepaid-card"></a>Sanal veya ön ödemeli kart kullanıyorsunuz 

Sanal veya ön ödemeli kredi veya banka kartları, Azure abonelikleri için ödeme olarak kabul edilmez.

### <a name="your-credit-information-is-inaccurate-or-incomplete"></a>Kredi bilgileriniz yanlış veya eksik 

Girdiğiniz ad, adres ve CVV kodu, kartın üzerinde yazılı olarak tam olarak eşleşmelidir.

### <a name="the-card-is-inactive-or-blocked"></a>Kart etkin değil veya engelleniyor 

Kartınızın etkin olduğundan emin olmak için bankanızla iletişim kurun.

Başka Kaydolma sorunları yaşıyor olabilirsiniz 

Azure kaydolma sorunlarını giderme hakkında daha fazla bilgi için aşağıdaki Bilgi Bankası makalesine bakın: 

[Azure portal veya Azure Hesap Merkezi 'nde Azure 'a kaydolayamıyorum](billing-troubleshoot-azure-sign-up.md)

### <a name="you-represent-a-business-that-doesnt-want-to-pay-by-card"></a>Kart ile ödeme yapmak istemediğiniz bir işi temsil edersiniz 

Bir işletmeyi temsil ediyorsanız, Azure aboneliğiniz için ödeme yapmak üzere denetimler, fazla gece denetimleri veya tel aktarımları gibi fatura ödeme yöntemlerini kullanabilirsiniz. Hesabı faturaya göre ödeme yapacak şekilde ayarladıktan sonra, Microsoft Müşteri anlaşmanız yoksa Azure Web sitesi üzerinden Azure 'a kaydolmadığınız müddetçe başka bir ödeme seçeneğine geçiş yapamazsınız.

Faturaya göre ödeme yapma hakkında daha fazla bilgi için bkz. [Azure aboneliğini faturaya göre ödeme için bir Istek gönderme](billing-how-to-pay-by-invoice.md).

### <a name="your-credit-card-information-is-outdated"></a>Kredi kartı bilgileriniz güncel değil 

Kartı değiştirme veya kaldırma dahil olmak üzere kart bilgilerinizi yönetme hakkında bilgi için bkz. [Azure için kredi ekleme, güncelleştirme veya kaldırma](billing-how-to-change-credit-card.md).
