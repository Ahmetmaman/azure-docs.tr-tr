# <a name="avoid-getting-charged-for-your-azure-free-account"></a>Azure ücretsiz hesabınız için sizden ücret kaçının

Ücretsiz Azure hesabına, 200 ilk 30 gün ve ücretsiz hizmetlerin 12 ay içinde harcamanız Azure kredisi sağlar. Daha fazla bilgi için bkz: [ücretsiz Azure hesabına](https://azure.microsoft.com/free/) ve [Ücretsiz Hizmetler ile ücretsiz Azure hesabına oluşturma](billing-create-free-services-included-free-account.md). Kredi durumu, bağlı olarak, kredi kullanabilir veya ücretsiz Azure hesabına avantajları kapsamında olmayan kullanım için sizden ücret.

## <a name="you-still-have-active-azure-free-account-credit"></a>Hesap kredi serbest etkin Azure çözümlenmedi 
Dolmadığından Azure kredi hala varsa, Azure ücretsiz hesabınız sonucunda oluşan herhangi bir ücret ödemeyi kredi kullanın.

## <a name="your-credit-ran-out-or-has-expired"></a>Kredi bitmiştir veya süresi dolmuş
Kredi bitmiştir veya 30 gün sonunda süresi dolduysa, Azure aboneliğiniz devre dışı bırakır. Aboneliğinizi Kullandıkça Öde aboneliğine yükselterek etkinleştirmediğiniz sürece, sizden ücret istenmese. Daha fazla bilgi için bkz: [ücretsiz deneme Azure aboneliğinizi Kullandıkça Öde aboneliğine yükseltme](billing-upgrade-azure-subscription.md). Aboneliğinizi yükseltme yaptıktan sonra Azure ücretsiz hesabınız sonucunda oluşan herhangi bir ücret için Kullandıkça Öde ücretleri temel alınarak. 


Azure ücretsiz hesabınız ücrete tabi nedenlerinden bazıları bakalım.


### <a name="your-usage-exceeds-the-limits-of-free-services-included-with-azure-free-account"></a>Kullanımınızı Ücretsiz Hizmetler ile ücretsiz Azure hesabına dahil sınırlarını aşıyor

Azure ücretsiz hesabınız ile her ay Ücretsiz Hizmetler sınırlı bir miktar alın. Ücretsiz miktar ay sonunda zaman aşımına uğrar ve sonraki aya değil, UTC'ye. Örneğin, her ay dosya depolama alanının 5 GB alın. Bir ay içinde yalnızca 2 GB kullanırsanız, kalan 3 GB sonraki aya UTC'ye değil. Ücret önlemek için kullanım sınırları içinde tutun. Ücretsiz hizmetlere sınırları hakkında bilgi edinmek için [ücretsiz Azure hesabına nedir](https://azure.microsoft.com/free/free-account-faq/). Ücretsiz hizmet kullanımınız denetlemek için bkz: [ile ücretsiz Azure hesabına dahil Ücretsiz Hizmetler kullanımını denetleme](billing-check-free-service-usage.md). 

### <a name="you-are-using-services-not-included-with-azure-free-account"></a>Ücretsiz Azure hesabı bulunmayan Hizmetleri kullanma

Ücretsiz Azure ücretsiz hesabınızla dahil edilmez Hizmetleri kullanma Kullandıkça Öde ücretleri temel alınarak. Ücretsiz hesapla dahil hizmetleri hakkında bilgi edinmek için [ücretsiz Azure hesabına nedir](https://azure.microsoft.com/free/free-account-faq/). Azure portalında veya Azure kullanım dosyası aracılığıyla kullanımınızı kontrol edebilirsiniz. Daha fazla bilgi için bkz: [düzenli olarak Maliyet Dökümü portal denetleyin ve yazma oranı](billing-getting-started.md#costs) ve [Merkezi'nden kullanım hesap](billing-download-azure-invoice-daily-usage-date.md). 

### <a name="you-have-reached-the-end-of-your-azure-free-account-benefits"></a>Ücretsiz Azure hesabına Avantajlarınızı sonuna ulaşıldı

Ücretsiz Azure hesabına Avantajlarınızı 12 ay sonunda süresi dolacak. Bundan sonra Azure, kullanmakta olduğunuz hizmetleri için Kullandıkça Öde hızlarını giderler. Kullanmadığınız hizmetler için kaynakları silmek için Azure portalını kullanabilirsiniz. Herhangi bir Azure hizmetinde kullanmak istemiyorsanız, şunları yapabilirsiniz [aboneliğinizi iptal](billing-how-to-cancel-azure-subscription.md).
