# <a name="azure-policy-built-in-policy-definitions-for-azure-sql-database"></a>Azure SQL veritabanı için Azure Ilkesi yerleşik ilke tanımları

Bu sayfa, Azure SQL veritabanı için [Azure ilkesi](../governance/policy/overview.md) yerleşik ilke tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../governance/policy/samples/built-in-policies.md)bölümüne bakın.
