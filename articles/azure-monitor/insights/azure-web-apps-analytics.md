# <a name="view-analytic-data-for-metrics-across-all-your-azure-web-app-resources"></a>Tüm Azure Web App kaynaklarınızda ölçümler için analitik verileri görüntüleme

![Web Apps simgesi](./media/azure-web-apps-analytics/azure-web-apps-analytics-symbol.png)  

> [!NOTE]
> Azure Web App Analytics çözümü topluluk desteğine geçti.  
>- Çözüm artık Azure Marketi'nden kullanılamıyor, ancak topluluk tarafından desteklenen [Azure Quickstart şablonlarından](https://azure.microsoft.com/resources/templates/101-webappazure-oms-monitoring/) yüklenebilir.
>- Çözümü zaten yüklemiş olan müşteriler, çözümdeğişikliğini kullanmaya devam edebilir.  
>- Microsoft, web uygulamalarınızı izlemek için [Application Insights'ı](../../azure-monitor/app/app-insights-overview.md) kullanmanızı önerir.
