# <a name="azure-status-overview"></a>Azure durumuna genel bakış

[Azure durumu,](https://status.azure.com/status/) Azure hizmetlerinin ve bölgelerinin sistem durumu hakkında genel bir görünüm sağlar. Azure durumu yla, hizmet kullanılabilirliği hakkında bilgi alabilirsiniz. Azure durumu, hizmet sistem durumunu bildiren tüm hizmetlerin yanı sıra geniş kapsamlı etkiye sahip olayları herkesin görüntülemesi için kullanılabilir. Ancak, geçerli bir Azure kullanıcısıysanız, [Azure Hizmet Durumu'ndaki](https://aka.ms/azureservicehealth)kişiselleştirilmiş deneyimi kullanmanızı şiddetle öneririz. Azure Hizmet Durumu tüm kesintileri, yaklaşan planlanan bakım etkinliklerini ve hizmet danışma belgelerini içerir.

![Azure durum sayfası](./media/azure-status-overview/azure-status.PNG)

## <a name="azure-status-updates"></a>Azure durum güncellemeleri

Azure hizmetlerinin durumu değiştikçe Azure durum sayfası gerçek zamanlı olarak güncellenir. Azure durum sayfasını açık bırakırsanız, sayfanın yeni verilerle yenilenme hızını denetleyebilirsiniz. En üstte, sayfanın en son ne zaman güncelleştiredildiğini görebilirsiniz.

![Azure durum yenileme](./media/azure-status-overview/update.PNG)

## <a name="azure-status-history"></a>Azure durum geçmişi

Azure durum sayfası her zaman en son sistem durumu bilgilerini gösterirken, azure [durum geçmişi sayfasını](https://status.azure.com/status/history/)kullanarak eski olayları görüntüleyebilirsiniz.

Kasım 2019'da yayınlanan etkinliklerle başlayan durum geçmişi sayfasında, azure için ön neden, azaltma ve sonraki adımlar da dahil olmak üzere geçmişte 365 güne kadar olan olaylar gösterilmektedir.

## <a name="rss-feed"></a>RSS Akışı

Azure durumu, abone olabileceğiniz Azure hizmetlerinin sağlık durumundaki değişikliklerin [RSS akışı](https://status.azure.com/status/feed/) da sağlar.
