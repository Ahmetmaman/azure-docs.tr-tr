# <a name="azure-policy-built-in-policy-definitions-for-azure-database-for-mariadb"></a>MariaDB için Azure veritabanı için Azure Ilkesi yerleşik ilke tanımları

Bu sayfa, MariaDB için Azure veritabanı 'nda [Azure ilkesi](../governance/policy/overview.md) yerleşik ilke tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../governance/policy/samples/built-in-policies.md)bölümüne bakın.

Azure portal içindeki ilke tanımına yapılan her bir yerleşik bağlantı adı. Kaynağı [Azure Policy GitHub](https://github.com/Azure/azure-policy)deposunda görüntülemek için **kaynak** sütunundaki bağlantıyı kullanın.
