# <a name="azure-security-mvp-program"></a>Azure güvenlik MVP programı
Microsoft'un en değerli uzmanları (MVP) yardımcı başkalarının deneyimlerini Microsoft teknolojileri ile en iyi almak için örnek bir taahhüt gösterilen topluluk Kılavuzları ' dir. Bunlar kendi olağanüstü tutku, gerçek hayattaki bilgi ve teknik uzmanlığı topluluk ve Microsoft ile paylaşın.

Microsoft Azure Azure güvenlik şimdi özel uzmanlık topluluk uzmanlarıyla tanıdığı duyurmaktan Mutluluk duyuyoruz. Microsoft MVPs Microsoft Azure Azure güvenlik katkı alanındaki MVP tahakkuk eden.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Teknoloji ve kendi ömrü tarafından değiştiğinden bir MVP kısmen olma için hiçbir Kıyaslama olsa da, biz değerlendirme ölçütleri Microsoft Answers, TechNet ve MSDN gibi çevrimiçi forumları nominee'nın Katkıları etkisini bazıları; Wiki'ler ve çevrimiçi içeriği; konferans ve kullanıcı grupları; pod yayınları, Web sitelerini, blogları ve sosyal medya; ve makaleleri ve books.

Azure güvenlik konusunda uzman misiniz? Birisi kim olduğunu biliyor musunuz? Ardından [kendinize veya başkalarına belirler](https://mvp.microsoft.com/Nomination/nominate-an-mvp) bir Azure güvenlik MVP bugün olacak!
