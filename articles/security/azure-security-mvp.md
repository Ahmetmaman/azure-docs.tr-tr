# <a name="azure-security-mvp-program"></a>Azure güvenlik MVP programı

Microsoft'un en değerli uzmanları (MVP) kullanan diğer yardımcı olmak için örnek bir taahhüt sergilemiştir topluluk liderleriyle ' dir. MVP diğer en iyi deneyimlerini Microsoft teknolojileriyle almak etkinleştirin. Bunlar kendi tutkularını, gerçek hayattaki bilgi ve teknik uzmanlığı Microsoft ve topluluk ile paylaşın.

Microsoft Azure Topluluk Uzmanları ile özel uzmanlık Azure güvenliği artık tanır. Microsoft azure'da Azure güvenlik katkı alanında MVP Microsoft MVPs puanlarımın.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Bir MVP olma için hiçbir Kıyaslama yoktur. Teknoloji ve yaşam döngüsü tarafından değiştiğinden kısmen budur. Bazı ölçütler içerir:

- Microsoft Answers, TechNet ve MSDN gibi çevrimiçi Forum Katkıları adayınız'ın etkisi
- Wikiler ve çevrimiçi içerik
- Konferans ve kullanıcı grupları
- Pod yayını, Web sitelerini, blogları ve sosyal medya
- Makaleler ve kitapları.

Azure güvenlik konusunda uzman misiniz? Öğrenciyseniz biliyor musunuz? Ardından [kendinize veya başka birisine aday gösterin](https://mvp.microsoft.com/Nomination/nominate-an-mvp) bir Azure güvenlik MVP bugün olacak!
