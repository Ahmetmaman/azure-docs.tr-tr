# <a name="azure-security-mvp-program"></a>Azure güvenlik MVP programı

Microsoft'un en değerli uzmanları (MVP) yardımcı başkalarının deneyimlerini Microsoft teknolojileri ile en iyi almak için örnek bir taahhüt göstermiştir topluluk Kılavuzları ' dir. Bunlar kendi tutku, gerçek hayattaki bilgi ve teknik uzmanlığı topluluk ve Microsoft ile paylaşın.

Microsoft Azure, Azure güvenlik özel uzmanlık topluluk uzmanlarıyla şimdi tanır. Microsoft MVPs Microsoft Azure Azure güvenlik katkı alanındaki MVP tahakkuk eden.

![Microsoft MVP](./media/azure-security-mvp/azure-security-mvp-fig1.png)

Teknolojisiyle değiştiğinden bir MVP kısmen olma için hiçbir Kıyaslama ve yaşam döngüsü olur, bazı ölçütler içerirken Microsoft Answers, TechNet ve MSDN gibi çevrimiçi forumları nominee'nın Katkıları etkisini; Wiki'ler ve çevrimiçi içeriği; konferans ve kullanıcı grupları; pod yayınları, Web sitelerini, blogları ve sosyal medya; ve makaleleri ve books.

Azure güvenlik konusunda uzman misiniz? Birisi kim olduğunu biliyor musunuz? Ardından [kendinize veya başkalarına belirler](https://mvp.microsoft.com/Nomination/nominate-an-mvp) bir Azure güvenlik MVP bugün olacak!
