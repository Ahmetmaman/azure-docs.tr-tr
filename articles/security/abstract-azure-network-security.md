# <a name="azure-network-security"></a>Azure ağ güvenliği
## <a name="abstract"></a>Özet
Azure Ağ Hizmetleri esneklik, kullanılabilirlik, dayanıklılık, güvenlik ve bütünlüğünü tasarım gereği en üst düzeye çıkarın. Bu teknik incelemede, Azure üzerinde ağ işlevlerini ayrıntılar sağlar. Müşteriler, bilgi varlıklarının korunmasına yardımcı olmak için Azure'da yerel güvenlik özelliklerini nasıl kullanabileceğinizi açıklar.
Bu teknik incelemeyi için hedeflenen izleyiciler dahil et:
- Teknik yöneticileri, ağ yöneticileri ve kullanılabilir ve azure'da desteklenen güvenlik çözümleri arayan geliştiriciler.
- SMEs veya Azure teknolojileri ve Hizmetleri Azure genel bulutunda ağ güvenliği ile ilgili üst düzey bir genel bakış isteyen iş işlem Yöneticiler.
