# <a name="mapping-of-azure-security-benchmark-v2-and-cis-microsoft-azure-foundations-benchmark"></a>Azure Güvenlik kıyaslaması v2 ve CIS Microsoft Azure, temelleri kıyaslama

CIS Microsoft Azure temel bilgiler kıyaslaması, Azure için güvenli bir temel yapılandırma oluşturmaya yönelik Internet Güvenliği Merkezi tarafından sunulan güvenlik yönergesidir. Kıyaslama kapsamı, Azure bulutunu benimseirken temel güvenlik düzeyini oluşturmak için kullanılır. Bu kıyaslama, [Azure Security kıyaslama v2](overview.md)ile hizalanır.

[Cıs Microsoft Azure Yönetim kıyaslama en son sürümü v 1.3.0](https://www.cisecurity.org/benchmark/azure/) ve [Azure Security kıyaslama v2](overview.md)arasında bir eşleme sağlıyoruz. Bu, her iki kıyaslamalar için önerilerin nasıl hizalandığını anlamanıza yardımcı olur. Tüm eşlemeyi [buradan](https://github.com/MicrosoftDocs/SecurityBenchmarks/blob/master/Azure%20Security%20Benchmark/2.0/asb_v2_to_cis_microsoft_azure_foundations_benchmark_v1.3.0.xlsx)indirebilirsiniz.
