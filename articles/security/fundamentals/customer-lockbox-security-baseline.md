# <a name="azure-security-baseline-for-customer-lockbox-for-microsoft-azure"></a>Microsoft Azure için Müşteri Kasası Azure Güvenlik temeli

Microsoft Azure için Müşteri Kasası Azure Güvenlik temeli, dağıtımınızın güvenlik duruşunu artırmanıza yardımcı olacak öneriler içerir.

Bu hizmetin taban çizgisi, Azure [güvenlik kıyaslama sürümü 1,0](https://docs.microsoft.com/azure/security/benchmarks/overview)' dan çizilir ve bu, en iyi yöntemler kılavuzumuzdan Azure 'da bulut çözümlerinizi nasıl güvence altına almak için öneriler sağlar.

Daha fazla bilgi için bkz. [Azure güvenlik temelleri 'ne genel bakış](https://docs.microsoft.com/azure/security/benchmarks/security-baselines-overview).

### <a name="311-monitor-attempts-to-access-deactivated-accounts"></a>3,11: devre dışı bırakılmış hesaplara erişme girişimlerini izleme

**Rehberlik**: uygun yerlerde merkezi kimlik doğrulama ve yetkilendirme sistemi olarak Azure Active Directory kullanın. Azure Active Directory, bekleyen ve aktarım sırasında veriler için güçlü şifrelemeyi kullanarak verileri korur. Ayrıca, Azure Active Directory sallar, karma ve Kullanıcı kimlik bilgilerini güvenli bir şekilde depolar.
