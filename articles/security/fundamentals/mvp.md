# <a name="azure-security-mvp-program"></a>Azure Güvenlik MVP programı

En değerli Microsoft uzmanları (MVP), başkalarına yardımcı olma taahhüdüne yönelik bir taahhüt isteyen topluluk liderlerdir. MVP 'ler, diğer kullanıcıların Microsoft teknolojileriyle deneyimlerinden en iyi şekilde yararlanmalarına olanak tanır. BT, gerçek dünyada bilgi ve teknik uzmanlığını topluluk ile ve Microsoft ile paylaşır.

Microsoft Azure artık Azure Güvenlik 'teki özel uzmanlığa sahip topluluk uzmanlarını tanımıştır. Microsoft MVP 'leri Azure Güvenlik katkısı alanındaki Microsoft Azure MVP 'ye alabilir.

![Microsoft MVP](./media/mvp/azure-security-mvp-fig1.png)

MVP olmak için bir kıyaslama yoktur. Bu, teknolojiye ve onun yaşam döngüsüne göre farklılık gösterdiği için bir parçasıdır. Bazı ölçütler şunları içerir:

- Bir aday Ee 'ın Microsoft yanıtları, TechNet ve MSDN gibi çevrimiçi forumlara katkılarının etkisi
- Wiksıs ve çevrimiçi içerik
- Konferanslar ve Kullanıcı grupları
- Pod yayınları, Web siteleri, Bloglar ve sosyal medya
- Makaleler ve Kitaplar.
