# <a name="azure-operational-security"></a>Azure operasyonel güvenlik
## <a name="abstract"></a>Özet
Microsoft Azure işlemsel güvenlik, Azure 'daki veri, uygulama ve diğer varlıkları korumak için kullanıcılara sunulan hizmetleri, denetimleri ve özellikleri ifade eder. Azure operasyonel güvenlik, Microsoft güvenlik geliştirme yaşam döngüsü (SDL), Microsoft Güvenlik Yanıt Merkezi programı gibi Microsoft 'a özgü çeşitli yetenekler aracılığıyla elde edilen bilgileri içeren bir çerçeve üzerine kurulmuştur. ve siber güvenlik tehdidi yatay olarak ayrıntılı bir şekilde tanıma. Bu Teknik İnceleme, Azure kullanarak işletimsel güvenliğe nasıl yaklaşılanabileceğini özetler. Aşağıdakiler dahil olmak üzere çeşitli Azure hizmetlerini içerir:
* Azure İzleyici günlükleri
* Azure Backup
* Azure Güvenlik Merkezi
* Azure İzleyici
* Azure ağ Izleyicisi
* Azure Depolama Analizi
* Azure Active Directory
