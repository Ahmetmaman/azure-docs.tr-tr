# <a name="threat-modeling-tool-releases"></a>Tehdit modelleme Aracı sürümleri

Microsoft tehdit modelleme aracı şu anda ücretsiz sunulan [tıklatın indirme](https://aka.ms/threatmodelingtool) Windows için uygulama. Bu teslim mekanizması müşterilere, en son geliştirmeleri ve hata düzeltmeleri, araç her açtığınızda anında iletme olanak sağlıyor.

## <a name="system-requirements"></a>Sistem Gereksinimleri

- Desteklenen İşletim Sistemleri
  - Microsoft Windows 10
- Ek Gereksinimler
  - Şablonlar ve araç için güncelleştirmeleri almak için Internet bağlantısı gerekir.
