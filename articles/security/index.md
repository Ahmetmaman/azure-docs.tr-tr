# <a name="azure-security-documentation"></a>Azure Güvenlik Belgeleri

Güvenlik, Azure’un her yönüne tümleştirilmiştir. Azure, genel güvenlik zekasından türetilmiş benzersiz güvenlik avantajları, müşteriye dönük gelişmiş denetimler ve güvenli bir sağlamlaştırılmış altyapı sağlar.
