# <a name="threat-modeling-tool-releases"></a>Tehdit Modelleme Aracı Bültenleri

Microsoft Tehdit Modelleme Aracı şu anda Windows için ücretsiz bir [tıklama-to-download](https://aka.ms/threatmodelingtool) uygulaması olarak yayımlanır. Bu teslim mekanizması, aracı her açtıklarında müşterilere en son geliştirmeleri ve hata düzeltmelerini zorlamamızı sağlar.

## <a name="system-requirements"></a>Sistem Gereksinimleri

- Desteklenen İşletim Sistemleri
  - Microsoft Windows 10 Yıldönümü Güncelleştirmesi veya sonrası
- .NET Sürümü Gerekli
  - .NET 4.7.1 veya sonrası
- Ek Gereksinimler
  - Araç ve şablonların güncelleştirmelerini almak için Internet bağlantısı gerekir.
