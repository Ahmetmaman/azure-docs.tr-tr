# <a name="yaml-configuration-options-to-customize-the-build-tasks"></a>Yapı görevlerini özelleştirmek için YAML yapılandırma seçenekleri

Bu makalede, yapı görevlerinin her birinde bulunan tüm YAML yapılandırma seçenekleri listelenir. Makale, güvenlik kodu çözümleme araçlarının görevleriyle başlar. İşlem sonrası görevlerle sona erer.
