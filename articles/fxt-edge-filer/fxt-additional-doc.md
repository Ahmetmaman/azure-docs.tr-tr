# <a name="additional-documentation"></a>Diğer belgeler 

Bazı kaynaklar bu docs.microsoft.com Web sitesinde dışında anlamanıza ve Microsoft Azure FXT Edge dosyalayıcı karma önbelleğinizle çalışmak yardımcı olabilir.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Azure önbellek belgeleri için Avere vFXT

[Azure için Avere vFXT](https://docs.microsoft.com/azure/avere-vfxt/) Microsoft Azure'da sanal düğümü kullanan benzer bir küme önbellek ürün. Azure portalı ile oluşturuldu, ancak aynı Denetim Masası yapılandırma yazılımı Azure FXT Edge dosyalayıcı donanım önbellek olarak kullanır. 

## <a name="legacy-hardware-documentation"></a>Eski donanım belgeleri 

Bu teknoloji, ürünlerinde Avere sistemlerden daha önce kullanıldı. Bu daha eski ürünler için kullanıcı belgeleri, üzerinde kullanılabilir [Avere eski belgeleri Web sitesinde](https://azure.github.io/Avere/). 

Özellikle, bu belgeleri yararlı ayrıntıları olabilir:

* [Yapılandırma Kılavuzu küme](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) - ortak küme kurulum ve yönetim görevleri hakkında genel bilgiler içerir ve küme ayarları sayfaları açıklamalarını ayrıntılı 

* [Pano Kılavuzu](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) -Denetim Masası özelliklerini açıklayan **Pano** sekmesi

* [Küme oluşturma kılavuzu FXT](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) -önceki ürünlerden küme oluşturma Kılavuzu
