# <a name="additional-documentation"></a>Diğer belgeler 

Bu docs.microsoft.com web sitesinin dışındaki bazı kaynaklar, Microsoft Azure FXT Edge Filer karma önbelleğini anlamanıza ve bunlarla çalışmanıza yardımcı olabilir.

## <a name="avere-vfxt-for-azure-cache-documentation"></a>Azure önbellek belgeleri için Avere vFXT

[Azure için Avere vFXT,](https://docs.microsoft.com/azure/avere-vfxt/) Microsoft Azure'da sanal düğümler kullanan benzer bir küme önbellek ürünüdür. Azure portalı ile oluşturulur, ancak Azure FXT Edge Filer donanım önbelleğiyle aynı denetim masası yapılandırma yazılımını kullanır. 

## <a name="legacy-hardware-documentation"></a>Eski donanım belgeleri 

Bu teknoloji daha önce Avere Systems ürünlerinde kullanılmıştır. Bu eski ürünler için kullanıcı belgeleri [Avere Legacy Documentation web sitesinde](https://azure.github.io/Avere/)mevcuttur. 

Özellikle, bu belgelerin yararlı ayrıntıları olabilir:

* [Küme Yapılandırma Kılavuzu](https://azure.github.io/Avere/legacy/ops_guide/4_7/html/ops_conf_index.html) - Ortak küme kurulumu ve yönetim görevleri hakkında genel bilgiler içerir ve küme ayarları sayfalarının ayrıntılı açıklamalarına sahiptir 

* [Pano Kılavuzu](https://azure.github.io/Avere/legacy/dashboard/4_7/html/ops_dashboard_index.html) - Kontrol paneli **Pano** sekmesinin özelliklerini açıklar

* [FXT Küme Oluşturma Kılavuzu](https://azure.github.io/Avere/legacy/create_cluster/4_8/html/create_index.html) - Önceki ürünlerden küme oluşturma kılavuzu
