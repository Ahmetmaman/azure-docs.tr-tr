# <a name="azure-ad-domain-services---contact-us"></a>Azure AD Etki Alanı Hizmetleri - Bize Ulaşın

## <a name="feedback"></a>Geri Bildirim
Hizmetimiz veya teknik olmayan sorularınız için geri bildiriminiz varsa, talebinizi Azure AD Etki Alanı Hizmeti ürün grubuyla bizimle paylaşabilirsiniz: [Azure AD Etki Alanı Hizmetleri Geri Bildirimi.](mailto:aaddsfb@microsoft.com) Bu adrese gönderilen e-postalar incelenir. Ürün grubunun üyeleri daha fazla bilgi isteğine yanıt **verebilir.** Bu e-posta adresine teknik destek talepleri yanıtlanmaz.

## <a name="technical-assistance"></a>Teknik yardım
Yönetilen etki alanınızla ilgili sorunlarınız varsa, [Sorun Giderme Kılavuzu'nda](troubleshoot.md) özetlenen adımların sorunu çözüp çözmeyeceğini denetleyin. Sorun olmaya devam ediyorsanız, Azure portalında oturum açın. Azure **AD Etki Alanı Hizmetleri** ana sayfasını açın ve **Destek + sorun giderme**altındaki eylem bölmesinden **Yeni destek isteği** seçin.

![yeni destek isteği](./media/contact-us/supportRequest.png) 

Destek isteğinin her bölümünü mümkün olduğunca doğru bir şekilde doldurun. Bir hizmet seçerken, isteğinizi uygun destek uzmanına yönlendirmek için **Azure Active Directory Domain Services 'ı (VM - Etki Alanı Denetleyicileri)** seçtiğinizden emin olun.

> [!IMPORTANT]
> Azure Etkin Dizin ideniyle ilgili yardım için hizmet listelerinden **Azure Etkin Dizin Dizinleri, Etki Alanları ve Nesneler'i** seçin.
>
>

Azure Active Directory Etki Alanı Hizmetlerini nasıl geliştirebileceğimiz hakkında bir fikriniz var mı? Bunu duymak istiyoruz. Azure Geri Bildirim Forumlarına gidin, [Etki Alanı Hizmetleri](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) kategorisine gidin ve fikrinizi paylaşın. Ürün grubunun üyeleri bu istekleri inceler.
