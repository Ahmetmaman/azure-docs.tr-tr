# <a name="azure-ad-domain-services---contact-us"></a>Azure AD etki alanı Hizmetleri - bizimle iletişime geçin

## <a name="feedback"></a>Geri Bildirim
Sunduğumuz hizmeti ya da teknik olmayan sorular için Geribildiriminiz varsa, Azure AD etki alanı hizmeti ürün grubu ile isteğiniz adresinden bize e-posta tarafından paylaşabilirsiniz: [Azure AD etki alanı Hizmetleri geri bildirim](mailto:aaddsfb@microsoft.com). E-postalar bu adrese gözden geçirilir. Ürün grubu üyelerinin **olabilir** daha fazla bilgi isteğine yanıt vermedi. Bu e-posta adresi için teknik destek istekleri yanıtlanacak değil.

## <a name="technical-assistance"></a>Teknik Yardım
Yönetilen etki alanınızla sorun yaşarsanız, adımları'bölümünde açıklanan olmadığını görmek için kontrol [sorun giderme kılavuzu](troubleshoot.md) sorunu çözün. Hala sorun yaşıyorsanız, Azure portalında oturum açın. Açık **Azure AD Domain Services** giriş sayfası ve select **yeni destek isteği** altında Eylem bölmesinden **destek + sorun giderme**.

![Yeni destek isteği](./media/contact-us/supportRequest.png) 

Destek isteğinin her bölümü olarak doğru bir şekilde doldurun. Bir hizmet seçerken seçtiğinizden emin olun **Azure Active Directory etki alanı Hizmetleri (VM - etki alanı denetleyicileri)** isteğiniz için uygun destek uzmanı yönlendirmek için.

> [!IMPORTANT]
> Seçin **Azure Active Directory dizin etki alanları ve nesneleri** Azure Active Directory ile etki alanı doğrulama ile ilgili Yardım için hizmetler listesinden.
>
> 

## <a name="feature-requests"></a>Özellik istekleri
Azure Active Directory etki alanı hizmetleri nasıl geliştirebileceğimizi üzerinde bir fikriniz mi var? Bu konuda fikrinizi öğrenmek isteriz. Go için Azure geri bildirim forumları, gitmek [etki alanı Hizmetleri](https://feedback.azure.com/forums/169401-azure-active-directory?category_id=160593) kategorisi ve fikrinizi paylaşın. Ürün grubunun üyeleri bu isteği gözden geçirin.
