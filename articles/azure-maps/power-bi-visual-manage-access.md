# <a name="manage-the-azure-maps-visual-within-your-organization"></a>Kuruluşunuzun içindeki Azure haritalar görselini yönetme

Power BI, tasarımcılar ve kiracı yöneticilerinin Azure haritalar görsellerinin kullanımını yönetmesine olanak sağlar. Aşağıda, her bir rolün uygulayabileceği adımları bulacaksınız.

### <a name="designer-options"></a>Tasarımcı seçenekleri

Power BI Desktop, tasarımcılar güvenlik sekmesinden Azure haritalar görselini devre dışı bırakabilir. **Dosya** &gt; **seçenekleri ve ayarlar** ' ı seçin ve ardından **Seçenekler** &gt; **Önizleme özellikleri**' ni seçin. Devre dışı bırakıldığında, Azure Maps varsayılan olarak yüklenmez.  
  
![Azure haritalar görselinizle ilgili Önizleme özellikleri seçeneklerini gösteren seçenekler paneli Power BI](media/power-bi-visual/preview-options-panel.png)

## <a name="tenant-admin-options"></a>Kiracı yöneticisi seçenekleri

PowerBI.com ' de, kiracı yöneticileri tüm kullanıcılar için Azure haritalar görselini kapatabilir. **Ayarlar** &gt; **yönetici** **portalı** &gt; **kiracı ayarları**' nı seçin. Devre dışı bırakıldığında, Power BI artık görsel öğeler bölmesinde Azure haritalar görselini görüntülemez.

> [!div class="mx-imgBorder"]
> ![Azure haritalar görseli için kiracı ayarlarını gösteren Yönetici portalı Power BI](media/power-bi-visual/tenant-admin-settings.png)
