# <a name="what-is-azure-database-migration-service"></a>Azure Veritabanı Geçiş Hizmeti nedir?

Azure Veritabanı Geçiş Hizmeti, birden çok veritabanı kaynağından Azure veri platformlarına en az kapalı kalma süresi (çevrimiçi geçiş) ile sorunsuz geçişler sağlamak üzere tasarlanmış tam olarak yönetilen bir hizmettir.

## <a name="migrate-databases-to-azure-with-familiar-tools"></a>Tanıdık araçlarla veritabanlarını Azure'a geçirme

Azure Veritabanı Geçiş Hizmeti, varolan araç ve hizmetlerimizin bazı işlevlerini tümleştirir. Müşterilere kapsamlı ve yüksek oranda kullanılabilir bir çözüm sunar. Hizmet, bir geçiş gerçekleştirmeden önce gerekli değişiklikler de size rehberlik etmek için öneriler sağlayan değerlendirme raporları oluşturmak için [Veri Geçiş Yardımcısı](https://aka.ms/dma) kullanır. Gerekli düzeltmeyi gerçekleştirmek size kalmış. Geçiş işlemine başlamaya hazır olduğunuzda, Azure Veritabanı Geçiş Hizmeti gerekli tüm adımları gerçekleştirir. İşlemin, Microsoft tarafından belirlenen en iyi deneyimlerden yararlandığını bilerek geçiş projelerinizi sonrasını düşünmeden gönül rahatlığıyla gerçekleştirebilirsiniz.

> [!NOTE]
> Çevrimiçi geçiş gerçekleştirmek için Azure Veritabanı Geçiş Hizmeti'ni kullanmak, Premium fiyatlandırma katmanını temel alan bir örnek oluşturmayı gerektirir.

## <a name="regional-availability"></a>Bölgesel kullanılabilirlik

Azure Veritabanı Geçiş Hizmetinin bölgesel kullanılabilirliği hakkında güncel bilgiler [için, bölgeye göre kullanılabilen Ürünler'e](https://azure.microsoft.com/global-infrastructure/services/?products=database-migration)bakın.

## <a name="pricing"></a>Fiyatlandırma

Azure Veritabanı Geçiş Hizmeti fiyatlandırması hakkında güncel bilgiler için Azure [Veritabanı Geçiş Hizmeti fiyatlandırması'na](https://azure.microsoft.com/pricing/details/database-migration/)bakın.
