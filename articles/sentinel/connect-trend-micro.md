# <a name="connect-trend-micro-deep-security"></a>Trend Micro Deep Security'yi bağlayın

Trend Micro Deep Security konektörü, panoları görüntülemek, özel uyarılar oluşturmak ve araştırmayı iyileştirmek için Derin Güvenlik günlüklerinizi Azure Sentinel ile kolayca bağlamanızı sağlar. Bu, kuruluşunuzun ağları ve sistemleri hakkında daha fazla bilgi verir ve güvenlik işlem yeteneklerinizi geliştirir.


## <a name="prerequisites"></a>Ön koşullar

- Azure Sentinel çalışma alanı için izinleri okuyun ve yazın.

## <a name="connect-azure-sentinel-to-trend-micro-deep-security"></a>Azure Sentinel'i Trend Micro Deep Security'ye bağlayın

1. Azure portalında Azure **Sentinel** > **Veri bağlayıcılarına** gidin ve **ardından Trend Micro Deep Security** bağlayıcısını seçin.

2. **Bağlayıcıyı Aç'ı**seçin.

3. Talimatları ve sonraki adımları izleyin.

Bu belgede, Trend Micro Deep Security'yi Azure Sentinel'e nasıl bağlayabileceğinizi öğrendiniz. Azure Sentinel hakkında daha fazla bilgi edinmek için aşağıdaki makalelere bakın:
- [Verilerinize ve olası tehditlere](quickstart-get-visibility.md)nasıl görünürlük elde edebilirsiniz öğrenin.
- Azure [Sentinel ile tehditleri algılamaya](tutorial-detect-threats-built-in.md)başlayın.
- Verilerinizi izlemek için [çalışma kitaplarını kullanın.](tutorial-monitor-your-data.md)
