# <a name="connect-one-identity-safeguard"></a>Bir Kimlik Koruması Bağlayın
One Identity Safeguard veri bağlayıcısı, Ayrıcalıklı Oturumlara Özel Panolar için Koruma ile standart Ortak Etkinlik Biçimi (CEF) bağlayıcısını geliştirir. Görselleştirme, uyarılar, araştırmalar ve daha fazlası için cihazınız tarafından oluşturulan olayları kolayca kullanmaya başlamak için bu bağlayıcıyı kullanın.


## <a name="prerequisites"></a>Ön koşullar

- Azure Sentinel çalışma alanı için izinleri okuyun ve yazın.

## <a name="connect-azure-sentinel-to-one-identity-safeguard"></a>Azure Sentinel'i Tek Kimlik Korumasına Bağlayın

1. Azure portalında Azure **Sentinel** > **Veri bağlayıcılarına** gidin ve ardından **Tek Kimlik Koruması** bağlayıcısını seçin.

2. **Bağlayıcıyı Aç'ı**seçin.

3. **Tek Kimlik Koruması** sayfasındaki yönergeleri izleyin.
