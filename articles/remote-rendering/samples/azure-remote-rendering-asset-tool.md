# <a name="azure-remote-rendering-asset-tool-arrt"></a>Azure uzaktan Işleme varlık Aracı (ARRT)

Azure uzaktan Işleme varlık Aracı (ARRT), C++/Qt' de geliştirilen açık kaynaklı bir masaüstü uygulamasıdır.

![ARRT](./media/azure-remote-rendering-asset-tool.png "ARRT ekran görüntüsü")

## <a name="features"></a>Özellikler

Uygulama şu şekilde kullanılabilir:

* 3B modeli karşıya yükleme
* Model dönüştürmeyi denetleme
* Uzaktan işleme oturumu oluşturma ve yönetme
* 3B model yükleme
* 3B modelin önizlemesini görüntüleyin
* Materyalleri değiştirme

Ayrıca, 3B model dönüştürmeyi yönetmek için Azure Storage Istemci kitaplığı 'nı kullanarak Azure uzaktan Işleme C++ SDK 'sının ön ucunun nasıl uygulanacağını öğrenmek için onu açık kaynaklı bir örnek olarak da kullanabilirsiniz.
