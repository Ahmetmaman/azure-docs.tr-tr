Bu makalede, Azure Veri Bilimi Sanal Makinesi sürümleri hakkında bilgi edinin. Dahil edilen araçların tam listesi için, sürüm numaralarıyla birlikte [Bu sayfaya](./tools-included.md)göz atın.
