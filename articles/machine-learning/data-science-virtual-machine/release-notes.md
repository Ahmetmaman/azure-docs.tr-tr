# <a name="azure-data-science-virtual-machine-release-notes"></a>Azure Veri Bilimi Sanal Makine sürüm notları

Bu makalede, Azure Veri Bilimi Sanal Makine sürümleri hakkında bilgi edinin. Sürüm numaralarıyla birlikte dahil edilen araçların tam listesi için [bu sayfaya](./tools-included.md)göz atın.

Bilinen hatalar ve geçici geçici işler hakkında bilgi edinmek için [bilinen sorunların listesine](reference-known-issues.md) bakın.

## <a name="2020-02-24"></a>2020-02-24

[Ubuntu 18.04](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.ubuntu-1804?tab=Overview) ve [Windows 2019](https://azuremarketplace.microsoft.com/marketplace/apps/microsoft-dsvm.dsvm-win-2019?tab=Overview) görüntüleri için Data Science Sanal Makine görüntüleri artık mevcuttur.

- Güç BI Masaüstü 2.73.55xx
- Firefox 69.0.2
