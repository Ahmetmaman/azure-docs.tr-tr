# <a name="what-tools-are-included-on-the-azure-data-science-virtual-machine"></a>Azure Veri Bilimi Sanal Makinesi'nde hangi araçlar yer alıyor?

Veri Bilimi Sanal Makine, verileri keşfetmenin ve bulutta makine öğrenimi yapmanın en kolay yoludur. Veri Bilimi Sanal Makineler, tam işletim sistemi, güvenlik yamaları, sürücüler ve popüler veri bilimi ve geliştirme yazılımı ile önceden yapılandırılmıştır. Düşük maliyetli CPU merkezli makinelerden birden fazla GPU'ya, NVMe depolama alanına ve büyük miktarda belleğe sahip çok güçlü makinelere kadar değişen donanım ortamını seçebilirsiniz. GPU'lu makineler için tüm sürücüler yüklenir, tüm makine öğrenimi çerçeveleri GPU uyumluluğu için sürümle eşleşir ve GPU'ları destekleyen tüm uygulama yazılımlarında hızlanma sağlanır.

Veri Bilimi Sanal Makine en yararlı veri-bilim araçları önceden yüklenmiş ile birlikte geliyor:

## <a name="how-to-choosecondaenvironment"></a>Conda ortamı nasıl seçilir  

> [!NOTE]
> Eski sürümden Data Science Virtual Machine'in yeni sürümüne geçerken kırılma değişiklikleri vardır. 

**Windows Server 2016'dan Windows Server 2019'a geçiş** </br> 

* Conda ortamı python37_default ayarlanır. Python2'yi destekliyoruz. Python conda ortamınızda tensorflow veya pytorch kullanıyorsanız sırasıyla py37_tensorflow veya py37_pytorch kullanın. Yalnızca AzureML kullanıyorsanız azureml_py36_automl </br>

* AzureML conda ortamınızda tensorflow veya pytorch kullanıyorsanız sırasıyla azureml_py36_tensorflow veya azureml_py36_pytorch kullanın. </br>

**Ubuntu 16.04'ten Ubuntu 18.04'e geçiş** </br>

* Azureml_py36 konda ortamınızda tensorflow veya pytorch kullanıyorsanız sırasıyla azureml_py36_tensorflow veya azureml_py36_pytorch kullanın. </br>

* Py37 conda ortamınızda tensorflow veya pytorch kullanıyorsanız sırasıyla py37_tensorflow veya py37_pytorch kullanın. </br>

* Aksi takdirde, python37_default kullanın veya sadece azureml_py36 kullanıyorsanız azureml_py36_automl </br>
