Machine Learning Studio kullanmak için bir Microsoft Azure Machine Learning çalışma alanına sahip olmanız gerekir. Bu çalışma alanı, denemeleri oluşturmak, yönetmek ve yayımlamak için ihtiyacınız olan araçları içerir.  

Azure aboneliğiniz için Yönetici çalışma alanı oluşturun ve ardından sahibi veya katkıda bulunanı eklemeniz gerekir. Ayrıntılar için bkz [oluşturma ve paylaşma bir Azure Machine Learning çalışma alanı](create-workspace.md).

Machine Learning Studio çalışma alanınızı oluşturduktan sonra açın ([https://studio.azureml.net/Home](https://studio.azureml.net/Home)). Birden fazla çalışma alanı varsa, araç penceresinin sağ üst köşesindeki çalışma alanını seçebilirsiniz.

![Studio çalışma alanı seçin][2]

> [!TIP]
> Çalışma alanının sahibi yapıldıysa, çalışma alanına davet ederek diğerleri üzerinde çalıştığınız denemeleri paylaşabilirsiniz. Machine Learning Studio'da üzerinde bunu yapabilirsiniz **ayarları** sayfası. Microsoft hesabı veya kuruluş hesabı her bir kullanıcı için yeterlidir.
> 
> Üzerinde **ayarları** sayfasında **kullanıcılar**, ardından **daha fazla kullanıcı davet** pencerenin alt kısmındaki.
>
