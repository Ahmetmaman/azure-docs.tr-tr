# <a name="azure-machine-learning-curated-environments"></a>Azure Machine Learning seçkin ortamlar

Bu makalede Azure Machine Learning içindeki seçkin ortamlar listelenmektedir. Seçkin ortamlar Azure Machine Learning tarafından sağlanır ve varsayılan olarak çalışma alanınızda kullanılabilir. Bunlar, Azure Machine Learning SDK 'sının en son sürümünü kullanan önbelleğe alınmış Docker görüntüleri tarafından desteklenir, böylece çalışma hazırlığı maliyeti azalır ve daha hızlı dağıtım süresine izin verilir. Çeşitli makine öğrenimi çerçevelerini hızlıca kullanmaya başlamak için bu ortamları kullanın.

> [!NOTE]
> Bu liste 2020 Aralık itibariyle güncelleştirilir. En güncel ortamlar listesini ve bunların bağımlılıklarını almak için Python SDK 'sını kullanın. Daha fazla bilgi için [ortamlar makalesine](./how-to-use-environments.md#use-a-curated-environment)bakın.
