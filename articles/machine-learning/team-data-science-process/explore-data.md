# <a name="explore-data-in-the-team-data-science-process"></a>Team Data Science Process verilerini keşfedin

Verileri [keşfetmek, Ekip Veri Bilimi Sürecinde](overview.md)bir adımdır.

Aşağıdaki makaleler, genellikle Veri Bilimi Süreci'nde kullanılan üç farklı depolama ortamındaki verilerin nasıl keşfedilecek lerini açıklar:

* [Pandas](https://pandas.pydata.org/) Python paketini kullanarak [Azure blob kapsayıcı](explore-data-blob.md) verilerini keşfedin.
* SQL kullanarak ve Python gibi bir programlama dili kullanarak [SQL Server](explore-data-sql-server.md) verilerini keşfedin.
* Kovan sorgularını kullanarak [Kovan tablosu](explore-data-hive-tables.md) verilerini keşfedin.

Buna ek olarak, [Azure Machine Learning Studio'daki Ön İşlem Verileri](https://azure.microsoft.com/documentation/videos/preprocessing-data-in-azure-ml-studio/)adlı videoda, verileri temizlemek ve dönüştürmek için yaygın olarak kullanılan modüller açıklanmaktadır.
