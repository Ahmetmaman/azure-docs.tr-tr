# <a name="sql-server-data-science-walkthroughs-using-r-python-and-t-sql"></a>SQL Server veri bilimi R, Python ve T-SQL kullanarak izlenecek yollar

Bu izlenecek yollar Tahmine dayalı analiz yapmak için SQL Server, SQL Server R Services ve SQL Server Python Services'ı kullanın. R ve Python kodu saklı yordamlarda dağıtılır. Bunlar takım veri bilimi işleminde açıklanan adımları izleyin. Takım veri bilimi işlemine genel bakış için bkz: [veri bilimi işlemi](overview.md). 

Takım veri bilimi işlemi yürütmek ek veri bilimi talimatlara göre gruplandırılır **platform** kullandıkları. Bkz: [takım veri bilimi işlemi yürütülürken izlenecek yollar](walkthroughs.md) bir döküm Bu örnekler için.


## <a name="predict-taxi-tips-using-python-and-sql-queries-with-sql-server"></a>SQL Server ile Python ve SQL sorgularını kullanarak ücreti ipuçları tahmin etme 

[Kullanım SQL Server](sql-walkthrough.md) izlenecek gösterir nasıl derleme ve makine öğrenme sınıflandırma dağıtma ve SQL Server ve genel kullanıma açık NYC kullanarak regresyon modeli ücreti seyahat ve ücreti veri kümesi.


## <a name="predict-taxi-tips-using-microsoft-r-with-sql-server"></a>SQL Server ile Microsoft R kullanma ücreti ipuçları tahmin etme 

[Kullanım SQL Server R Hizmetleri](https://msdn.microsoft.com/library/mt612857.aspx) oluşturmak ve bir R dağıtmak için özel SQL işlevleri için SQL Server modeli ve izlenecek yol veri bilimcilerine ile birlikte R kodu, SQL Server veri sağlar. İzlenecek yol R geliştiriciler R Hizmetleri (veritabanı-) tanıtmak için tasarlanmıştır.
