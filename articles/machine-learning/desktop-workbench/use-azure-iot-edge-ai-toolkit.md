# <a name="use-the-ai-toolkit-for-azure-iot-edge"></a>Azure IOT köşesi AI araç kullanma

Yapay zeka, machine learning ve gelişmiş analizler gücünü kenarına getirin. AI için araç seti ile Azure IOT kenar, AI ve machine learning, verilerinizin nerede yaşıyor dağıtabilirsiniz. REST API ile Docker kapsayıcısı olarak modelinizi faaliyete ve modeli, bir sınır cihazının Azure IOT Hub ile iletin. Azure IOT kenar için AI Araç Seti yönetim bilgileri, esnek hesaplama ve bulut güç sınırlı, aralıklı, yerler veya hiç bağlantı getirir.

Azure IOT kenar için AI araç seti, komut dosyaları, kod ve dağıtılabilir kapsayıcıları koleksiyonudur. Tahmine dayalı bakım, görüntü sınıflandırma ve konuşma işleme yanı sıra Azure Machine Learning ve Azure IOT hub'ı aracılığıyla özel model Dağıtım örnekleri içerir. Araç Seti modelleri kullanılabilir olarak-olduğu, ancak tüm kaynak kodu ve veri kullanılabilir geliştiricilerin özelleştirmek için kendi ihtiyaçlarına göre.

Azure IOT genel GitHub depo bulunabilir kenar AI Toolkit [AI/aka.ms-toolkit](http://aka.ms/AI-toolkit).
