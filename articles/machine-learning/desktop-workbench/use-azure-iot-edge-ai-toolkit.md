# <a name="use-the-ai-toolkit-for-azure-iot-edge"></a>Azure IOT Edge için AI Toolkit kullanın

Yapay zeka, makine öğrenimi ve Gelişmiş analiz gücünü uç cihazlara taşıyın. Azure IOT Edge için AI Toolkit ile yapay ZEKA ve makine öğrenimi verileriniz nerede dağıtabilirsiniz. REST API ile bir Docker kapsayıcısı olarak modelinizi kullanıma hazır hale getirin ve daha sonra modeli bir edge cihazını Azure IOT Hub ile gönderin. Azure IOT Edge için AI Toolkit, zeka, elastik bilgi işlem ve bulutun tüm gücü yerler sınırlı aralıklı, ya da bağlantı getirir.

Azure IOT Edge için AI Toolkit, betikleri, kod ve dağıtılabilir kapsayıcılar oluşan bir koleksiyondur. Tahmine dayalı bakım, görüntü sınıflandırma ve konuşma işleme yanı sıra Azure Machine Learning ve Azure IOT Hub aracılığıyla özel model dağıtımı örneklerindendir. Araç Seti modelleri olarak kullanılabilir-olduğunu, ancak tüm kaynak kodu ve veri kullanılabilir özelleştirmek, geliştiriciler için ihtiyaçlarını temel alarak.

Genel GitHub deposunda bulunabilir Azure IOT Edge için AI Toolkit [aka.ms/AI-toolkit](http://aka.ms/AI-toolkit).
