## <a name="get-support-for-azure-machine-learning-services"></a>Azure Machine Learning Hizmetleri için destek alma

Aşağıdaki yollarla Azure Machine Learning Hizmetleri için destek alabilirsiniz:

+ **Teknik Destek**: kişi Azure teknik destek adresi ziyaret ederek [Azure Destek](https://azure.microsoft.com/support/options/) ve Machine Learning seçme. 

+ **MSDN kullanıcı forumu**: diğer kullanıcılarla bağlanmak soru sorun ve soruları [Destek Forumu MSDN'deki Azure Machine Learning Hizmetleri](https://social.msdn.microsoft.com/Forums/windowsdesktop/home?forum=MachineLearning).

+ **Yığın Taşması**: Azure Machine Learning topluluğu ziyaret [StackOverflow](https://stackoverflow.com/questions/tagged/azure-machine-learning) "Azure-Machine-Learning ile" etiketli.

+ **Paylaşım ürün önerileri** ve özellik istekleri bizim [Azure Machine Learning geri bildirim kanalı](https://feedback.azure.com/forums/257792-machine-learning), olduğu ayrıca her makalenin alt kısmında bağlantıyı kullanarak erişilebilir.
