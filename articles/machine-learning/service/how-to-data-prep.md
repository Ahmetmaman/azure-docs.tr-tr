# <a name="prepare-data-with-the-azure-machine-learning-data-prep-sdk-for-python"></a>Python için Azure Machine Learning veri hazırlığı SDK'sı ile verileri hazırlama
 
Veri hazırlama, machine learning iş akışı önemli bir parçasıdır. Modellerinizi daha doğru ve verimli kullanmak daha kolay bir biçimde verileri temizlemek için erişime sahip oldukları olacaktır. Kullanabileceğiniz [Azure Machine Learning veri hazırlığı SDK'sı](https://docs.microsoft.com/python/api/overview/azure/dataprep?view=azure-dataprep-py) çeşitli biçimlerden veri yükleme, daha kullanışlı olması için dönüştürmek ve Modellerinizi erişmek için bir konum verileri yazmak için. 


## <a name="steps-for-data-preparation-with-azure-machine-learning"></a>Azure Machine Learning ile veri hazırlama adımları 
1. [Veri yükleme](how-to-load-data.md )
2. [Verileri dönüştürme](how-to-transform-data.md )
3. [Veri yazma](how-to-write-data.md )
