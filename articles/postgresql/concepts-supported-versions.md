# <a name="supported-postgresql-database-versions"></a>Desteklenen PostgreSQL veritabanı sürümleri
Microsoft Azure şu anda yayımlanmış ana sürüm (n) ve iki önceki ana sürüm (-2) anlamına veritabanında PostgreSQL hizmeti PostgreSQL altyapısı n-2 sürümleri desteklemek için amaçlar.

Azure veritabanı PostgreSQL için şu anda aşağıdaki sürümlerini destekler:

## <a name="postgresql-version-103"></a>PostgreSQL sürüm 10.3
Başvurmak [PostgreSQL belgelerine](https://www.postgresql.org/docs/10/static/release-10-3.html) geliştirmeleri ve PostgreSQL 10.3 düzeltmeler hakkında daha fazla bilgi edinmek için.

## <a name="postgresql-version-967"></a>PostgreSQL sürüm 9.6.7
Başvurmak [PostgreSQL belgelerine](https://www.postgresql.org/docs/9.6/static/release-9-6-7.html) geliştirmeleri ve PostgreSQL 9.6.7 düzeltmeler hakkında daha fazla bilgi edinmek için.

## <a name="postgresql-version-9511"></a>PostgreSQL sürüm 9.5.11
Başvurmak [PostgreSQL belgelerine](https://www.postgresql.org/docs/9.5/static/release-9-5-11.html) geliştirmeleri ve düzeltmeler hakkında bilgi edinmek için PostgreSQL 9.5.11 içinde.

## <a name="managing-updates-and-upgrades"></a>Yönetme güncelleştirmeleri ve yükseltmeleri
Azure veritabanı PostgreSQL için ikincil sürüm güncelleştirmeleri düzeltme eki uygulama otomatik olarak yönetir. Şu anda, ana sürüm yükseltme desteklenmiyor. Örneğin, PostgreSQL 9.6 PostgreSQL 9.5 ' yükseltme desteklenmiyor.
