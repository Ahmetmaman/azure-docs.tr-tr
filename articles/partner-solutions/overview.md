# <a name="extend-azure-with-solutions-from-partners"></a>Azure 'ı iş ortaklarının çözümleriyle genişletme

İş ortağı kuruluşları, Azure 'da bulut altyapınızı geliştirmek için kullanabileceğiniz çözümler sunar. Bu çözümler Azure ile tamamen tümleşiktir. Bu çözümlerle, Microsoft 'un çözümleriyle aynı şekilde çalışırsınız. Çözümü yönetmek için bir kaynak sağlayıcısı, kaynak türleri ve bir API kullanırsınız.

İş ortağı çözümleri Market üzerinden kullanılabilir.

| İş ortağı çözümü | Description |
| :--- | :--- |
| [Konluent bulutu için Apache Kafka](./apache-kafka-confluent-cloud/overview.md) | Apache Kafka tarafından desteklenen tam olarak yönetilen olay akışı platformu |
