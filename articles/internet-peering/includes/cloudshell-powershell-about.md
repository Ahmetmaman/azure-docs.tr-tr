Cmdlets'i çalıştırmak için ücretsiz bir etkileşimli kabuk olan Azure Cloud Shell'i kullanabilirsiniz. Yaygın Azure araçları, kabuğa önceden yüklenmiştir ve kabuk, hesabınızla birlikte kullanılacak şekilde yapılandırılmıştır. Kodu **Copy** kopyalamak için Kopyala'yı seçin ve Bulut Kabuğu'na yapıştırın. Ardından çalıştırmak için **Enter'u** seçin. Cloud Shell'i başlatmanın birkaç yolu vardır:


|  |   |
|-----------------------------------------------|---|
| Cloud Shell’i tarayıcınızda açın. | [![https://shell.azure.com/powershell](../media/launchcloudshell.png)](https://shell.azure.com/powershell) |
| Azure portalının sağ üst kısmındaki araç çubuğundaki **Bulut Kabuğu** düğmesini seçin. | [![Portalda Bulut Kabuğu](../media/cloud-shell-menu.png)](https://portal.azure.com) |
|  |  |


Azure Bulut Shell'i kullanmak istemiyorsanız, PowerShell'i yerel olarak yükleyebilirsiniz. PowerShell'i yerel olarak yüklemeyi ve kullanmayı seçerseniz, Azure Kaynak Yöneticisi PowerShell cmdlets'in en son sürümünü yüklediğinizden emin olun. PowerShell cmdlets sık sık güncellenir. En son özellik işlevselliğini elde etmek için genellikle PowerShell cmdletlerinizi güncellemeniz gerekir. Bunu yapmazsanız, sorunlarla karşılaşabilirsiniz.

PowerShell'in yerel olarak çalıştırdığınız sürümünü bulmak için **'Get-Module -ListAvailable Az'** cmdlet'ini kullanın. Güncellemek için [Azure PowerShell modüllerini yükleyin'](https://docs.microsoft.com/powershell/azure/azurerm/install-azurerm-ps)e bakın. Daha fazla bilgi için [Azure PowerShell'in nasıl yüklenir ve yapılandırılın.](https://docs.microsoft.com/powershell/azure/azurerm/overview)

PowerShell'i macOS'ta kullanıyorsanız, [powershell'i macOS'a yükleme](https://docs.microsoft.com/powershell/scripting/install/installing-powershell-core-on-macos?view=powershell-6) adımlarını izleyin.
