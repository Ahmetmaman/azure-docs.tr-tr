# <a name="seller-insights-release-notes"></a>Satıcı Insights sürüm notları 

(Yayın tarihi: 1 Mart 2019)

Bu makalede, [Bulut İş Ortağı Portalı'ndaki](https://cloudpartner.azure.com/#insights)Satıcı Öngörüleri özelliğinde yapılan değişiklikler hakkında bilgi verilmektedir.

## <a name="release-highlights-for-march-1-2019"></a>1 Mart 2019 için öne çıkan lar

* *Müşteri Trendi* Özete eklendi
* Özet'teki *en iyi beş Müşteri,* bir müşterinin sahip olduğu tüm Azure aboneliklerini yansıtır
* Normalleştirilmiş Kullanım Eğilimi & Bir *Bakışta Aylık Siparişler* Altında Hareket Edilen *Özetteki Aktif Siparişler Trendi*
* *Ödeme Mutabakat Raporu* güncellendi
* Ödeme'deki *en iyi beş Müşteri,* bir müşterinin sahip olduğu tüm Azure aboneliklerini yansıtır
* Müşteri Kimliği ile güncelleştirilen *Kullanım Raporu*
* Siparişlerde *Müşteri Görev Süresi* & Kullanım, müşterinin sahip olduğu tüm Azure aboneliklerini yansıtır


(Yayın tarihi: 28 Temmuz 2018)

## <a name="release-highlights-for-july-28-2018"></a>28 Temmuz 2018 için öne çıkan lar


-   *Tahmini fiyatlar,* para birimi dönüştürme etkileriyle birlikte müşteri ücretlerinin görünümünü sağlar.
-   *Öngörülen ödemeler,* olası ödemelere daha erken bir görünüm sağlar.
-  *Kullanım referans tanımlayıcıları,* müşteri kullanımı ile siparişler arasında ödemelerle veri doğruluğu sağlar
-   *Günlük tahıl kullanımı* daha ayrıntılı lık ve müşteri kullanımına ilişkin daha iyi bilgiler sağlar.


### <a name="changes-to-data-structure-and-taxonomy"></a>Veri yapısı ve taksonomi değişiklikleri

Aşağıdaki tablo, bu sürümle eklenen veya önemli ölçüde değiştirilen ölçümleri listeler. 

| **Yeni Dönem**                   |    **Tanım**                                                             |
|--------------------------------|  ---------------------------------------------------------------------------- |
| Fiyat (CC)                     | Belirli bir SKU için bir kullanım birimi için fiyat (müşterinin para birimi cinsinden).       |
| Tahmini Genişletilmiş Yük (CC) | Belirli bir SKU için kullanım birimlerinin miktarı için tahmini genişletilmiş ücret (müşterinin para biriminde). Yuvarlama veya kesilme hataları nedeniyle bu değer kesin olmayabilir.   |
| Yayıncı Para Birimi (PC)        | Ödeme için yayımcı tarafından tercih edilen para birimi.                               |
| Tahmini Fiyat (PC)           | Kullanım tarihindeki döviz dönüştürmesi temel alınabağlı olarak belirli bir SKU kullanım birimi için tahmini fiyat hesaplanır (yayıncının para birimi cinsinden). Yuvarlama veya kesilme hataları nedeniyle bu değer kesin olmayabilir.   |
| Tahmini Genişletilmiş Şarj (PC) | Kullanım tarihindeki döviz dönüştürmesi esas alınabağlı olarak belirli bir SKU kullanım birimi nin kullanım birimi miktarı için tahmini genişletilmiş ücret hesaplanır (yayıncının para birimi cinsinden). Yuvarlama veya kesilme hataları nedeniyle bu değer kesin olmayabilir. |
| Tahmini Ödeme (PC)          | Belirli bir SKU kullanım biriminin kullanım biriminin miktarı için, kullanım hesaplanır tarihteki döviz dönüştürmesi (yayıncının para birimi cinsinden) dayalı tahmini ödeme. Yuvarlama veya kesilme hataları nedeniyle bu değer kesin olmayabilir.   |
| Kullanım Başvurusu                | Ödeme raporundaki bir girişle ilişkili belirli bir SKU için bir veya daha fazla müşteri kullanımı için tanımlayıcı. |
|  |  |
