# <a name="cloud-partner-portal-migration-to-partner-center"></a>İş Ortağı Merkezi'ne Bulut İş Ortağı Portalı geçişi

Microsoft, hesabınızı kolaylaştırmak ve yönetim deneyimini sunmak için Bulut İş Ortağı Portalı tekliflerini Iş ortağı merkezine geçirmektedir. Tek bir ticari Iş ortağı go-to-market (OCP GTM) kataloğunda ortak satış yardımcı malzemeleri yönetiyorsanız, Iş Ortağı Merkezi 'nde ortak satış yardımcı malzemeleri yakında güncelleştirebilirsiniz.

| **Teklif türü** | **Teklif yönetimi Bulut İş Ortağı Portalı 'ten Iş ortağı merkezine geçirildi** | **Iş Ortağı Merkezi 'nde ortak satış yardımcı malzemeleri güncelleştirme yeteneği** |
|    -----------------------    |    -----------------------------  |  -----------------------------  |
| Hizmet olarak yazılım (SaaS) |     Kullanıma sunuldu       |   Şubat 'ta genel kullanıma hazır. 2020      |
| PowerApps & müşteri katılımı için Dynamics 365      |     Kullanıma sunuldu        |   Kullanıma sunuldu      |
| Dynamics 365 for Operations     |     Kullanıma sunuldu       |   Kullanıma sunuldu     |
| Dynamics 365 Business Central      |     Kullanıma sunuldu        |   Şubat 'ta genel kullanıma hazır. 2020     |
| Azure Uygulaması     |     Genel kullanılabilirlik ara. 2019       |   Şubat 'ta genel kullanıma hazır. 2020     |
| Azure Sanal Makinesi      |     Duyurulacak       |   Duyurulacak    |
| Azure kapsayıcısı     |     Duyurulacak       |   Duyurulacak     |
| Danışmanlık Hizmetleri      |     Duyurulacak       |   Duyurulacak     |
| IoT Edge modülü      |     Duyurulacak       |   Duyurulacak     |
| Power BI hizmet uygulaması     |     Duyurulacak       |   Duyurulacak     |
