# <a name="saas-application-channel-info-tab"></a>SaaS uygulama kanal bilgisi sekmesi

Git pazara açılma (GTM) malzemeleri dahil olmak üzere, uygulamanız için kanal bilgileri sağlayın ve iletişim bilgileri için kanal bilgileri sekmesini kullanın.

![Kanal bilgi formu](./media/saas-channel-info.png)

## <a name="to-configure-channel-information"></a>Kanal bilgileri yapılandırmak için

1. Altında **kanal bilgileri > GTM malzemeleri**, GTM materyallerinizi barındıran Web sitesinin URL'sini girin.
2. Altında **kanal kurmanız**, Microsoft ulaşmak için destek ve iş sorunlarını manager kişi sağlayın. Bu alanlar için bilgileri girin: Ad, e-posta adresi ve telefon numarası.
3. Seçin **Kaydet** kanal bilgileri ayarlama işlemini sonlandırmak için.
