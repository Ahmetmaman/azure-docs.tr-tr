# <a name="azure-and-dynamics-365-consulting-service-offer"></a>Azure ve Dynamics 365 danışmanlık hizmeti teklifi

Bu makalede, Microsoft <a href="https://azuremarketplace.microsoft.com">Azure Marketi</a> veya <a href="https://appsource.microsoft.com">AppSource'a</a>bir danışmanlık hizmeti teklifinin nasıl yayımlanabileceği açıklanmaktadır. Microsoft Dynamics <a href="https://dynamics.microsoft.com">365,</a> <a href="https://powerbi.microsoft.com">Power BI</a>ve <a href="https://powerapps.microsoft.com">Power Apps'a</a> dayalı çözümler AppSource'da listelenebilir. Diğer Microsoft Azure <a href="https://azure.microsoft.com/services">hizmetlerine</a> dayalı diğer teklifler Azure Marketi'nde listelenebilir.

## <a name="publishing-benefits"></a>Yararlanma avantajları

Microsoft pazarlarından herhangi birini yayımlamanın aşağıdaki avantajları vardır:

- Microsoft markasını yararlanarak şirketinizi tanıtın.
- AppSource'ta 100 milyondan fazla Office 365 ve Dynamics 365 kullanıcısına ve Azure Marketi aracılığıyla 200.000'den fazla kuruluşa ulaşma potansiyeline sahip.
- Bu pazarlardan yüksek kaliteli müşteri adayları alın.
- Hizmetlerinizi Microsoft alanı ve tele-satış ekipleri tarafından tanıtın.

## <a name="publish-a-consulting-service-offer"></a>Danışmanlık hizmeti teklifi yayınlama

Bu bölümde, bir danışmanlık hizmeti teklifi yayınlama nın öğeleri özetler. Yayımlama aşağıdaki ana bölümlere ayrılmıştır:

- [Danışmanlık hizmeti önkoşulları,](./cpp-consulting-service-prerequisites.md) aşağıdakiler için bir danışmanlık hizmeti teklifi yayınlamak için gerekli gereksinimleri listeler:
 
    - Microsoft Azure
    - Dynamics 365 Müşteri Katılımı 
    - Dynamics 365 Finans & İşlemleri 
    - Dynamics 365 Customer Insights
    - Dynamics 365 Business Central 
    - Power BI 
    - Power Apps
    - Power Automate
- [Yeni bir teklif oluşturun,](./cpp-consulting-service-create-offer.md) Bulut İş Ortağı Portalı'nı kullanarak bir danışmanlık hizmeti teklifi girişi oluşturmak için gereken adımları listeler:
    - [Teklif ayarlarını tanımlayın.](./cpp-consulting-service-define-offer-settings.md)
    - [Azure Marketi'nde veya AppSource'da yayımlayıp yayımlanmayacağı ve mağaza nın ayrıntılarını girin.](./cpp-consulting-service-storefront-details.md)
- [Teklifinizi yayımlamak,](./cpp-consulting-service-publish-offer.md) teklifin yayımlama için nasıl gönderilebildiğini açıklar.
