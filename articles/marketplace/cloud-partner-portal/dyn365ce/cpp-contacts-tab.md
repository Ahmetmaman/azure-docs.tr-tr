# <a name="azure-application-contacts-tab"></a>Azure uygulama kişileri sekmesi

Kullanım **kişiler** mühendislik ve müşteri destek bilgileri için Dynamics 365 müşteri katılımı teklif için sağlamak için sekmesinde.

![Teklif için sekmesinde kişiler](./media/dynce-contacts-tab.png)


## <a name="specify-customer-and-engineering-contacts"></a>Müşteri ve mühendislik kişileri belirtin

Birincil tanımlamak için aşağıdaki değerleri sağlayın **mühendislik birimi ilgili kişisi** ve teklifiniz için. Gerekli alanlar yıldız (*) indicted.

|    Alan        |  Açıklama                    |
|    ---------    |  ---------------                |
|  **Ad\***       |  Uygulamanız için mühendislik kişinin adı. Bu kişi Microsoft Teknik iletişimleri alırsınız.   |
|  **E-posta\***      |  Mühendislik birimi ilgili kişisi e-posta adresi  |
|  **Telefon\***      |  Mühendislik birimi ilgili kişisi için telefon numarası  |
|  |  |

Birincil tanımlamak için aşağıdaki değerleri sağlayın **destek ilgili kişisi** ve teklifiniz için.

|    Alan        |   Açıklama                   |
|    ---------    |  ---------------                |
|  **Ad\***       |  Uygulamanız için müşteri destek ilgili kişisi adı. Bu kişi Microsoft desteğe ilişkin iletişimleri alırsınız.   |
|  **E-posta\***      |  E-posta adresi Müşteri destek ilgili kişisi  |
|  **Telefon\***      |  Telefon numarası, müşteri desteğine başvurun  |
|  |  |
