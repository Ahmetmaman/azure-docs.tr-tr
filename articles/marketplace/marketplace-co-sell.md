# <a name="co-sell-partner-engagement"></a>Ortak satış iş ortağı etkileşimi

İş ortağıysanız, Microsoft ortak satış programını kullanarak doğrudan Microsoft 'un ortak satış fırsatları üzerinde çalışmasını sağlayabilirsiniz. Program, iş ortaklarımızın büyük bir müşteri topluluğuna ulaşmasını ve iş artışını hızlandırmaya yardımcı olabilecek yeni satış fırsatları oluşturmalarına yardımcı olur.

## <a name="partner-selling-motions"></a>İş ortağı satış modülleri

Microsoft üç iş ortağı satışı sunmaktadır:

* *İş ortağı aracılığıyla satın*alabilirsiniz: Microsoft teknolojisini kendi satıcılarınız aracılığıyla yeniden satabilirsiniz.
* *Hizmetler ortak satış*: satıcılar ve iş ortakları, Microsoft Technology Embedded ile projenizi veya yönetilen Hizmetleri satmak için işbirliği yapıyor.
* *IP ortak satış*: satıcılar ve iş ortakları, tekrarlanabilir, paketlenmiş IP çözümünüzü satmaya işbirliği uygular. *

\*Bu IP ortak satış çözümlerinin bazıları, Microsoft alan satış ekipleriyle başarıyla satıldığında ve dağıtıldığında ek teşvikleri alabilir.
