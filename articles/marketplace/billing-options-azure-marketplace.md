# <a name="billing-options"></a>Faturalandırma seçenekleri  

## <a name="commercial-considerations-on-the-marketplace"></a>Market'te ticari konuları  
Aşağıdaki liste türleri için gelir Market paylaşmaz.  
*   Liste  
*   Deneme  
*   Kendi lisansını getir (KLG) faturalandırma modeliyle transact

Market'te vitrinler katılmak için faturalandırılan ek ücretler değildir.  
*   Daha fazla bilgi için ziyaret edin sayfasında bulunan Azure Marketi katılım ilkeleri [azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-bring-your-own-license-billing-options"></a>Kullandıkça Öde ve kendi lisans faturalandırma seçenekleri  
Bir yayımlama seçeneği olarak Kullandıkça Öde faturalandırma modeli kullandığınız zaman, kullanım tabanlı yazılım lisans gelir: 80/20 ve Microsoft arasında paylaşılan. Kullandıkça Öde hem KLG faturalandırma modelleri kullanarak tek bir teklif fiyatlandırılır. İki faturalandırma modelimiz teklif düzeyinde ayrı SKU'ları bir arada. Bulut iş ortağı portalı teklifinizi faturalandırma modelleri yapılandırın. 

Aşağıdaki örneklere bakın.  
*   Kullandıkça Öde seçeneği etkinleştirirseniz, aşağıdaki sonucu vardır.  
    | Lisansınızı maliyeti | saat başına 1,00 $ |  
    |:--- |:--- |  
    | Azure kullanım maliyeti (D1/1-çekirdek) | saat başına 0.14 |  
    | *Müşteri, Microsoft tarafından faturalandırılır* | *saat başına 1,14* |  

    Microsoft, bu senaryoda, yayımlanmış VM görüntülerinden birini kullanmak için saat başına 1,14 düzenler.  
    | Microsoft faturalar | saat başına 1,14 |  
    |:--- |:--- |  
    | Microsoft, lisans maliyeti 80 oranında öder | saat başına 0,80 $ |  
    | Microsoft, lisans maliyeti %20 tutar. | saat başına $0,20 |  
    | Microsoft Azure kullanım maliyeti tutar. | saat başına 0.14 |  

*   KLG seçeneği etkinleştirirseniz, aşağıdaki sonucu vardır.  
    | Lisansınızı maliyeti | Lisans ücreti anlaşma ve sizin tarafınızdan faturalandırılır |  
    |:--- |:--- |  
    | Azure kullanım maliyeti (D1/1-çekirdek) | saat başına 0.14 |  
    | *Müşteri, Microsoft tarafından faturalandırılır* | *saat başına 0.14* |  

    Microsoft, bu senaryoda, yayımlanmış VM görüntülerinden birini kullanmak için saat başına 0.14 düzenler.  
    | Microsoft faturalar | saat başına 0.14 |  
    |:--- |:--- |  
    | Microsoft Azure kullanım maliyeti tutar. | saat başına 0.14 |  
    | Microsoft, lisans maliyeti %0 tutar. | saat başına 0,00 ABD Doları |  

## <a name="single-billing-and-payment-methods"></a>Tek faturalandırma ve ödeme yöntemleri  
Seçenek listesi Transact kullanmanın önemli bir avantajı lisanslama maliyetleri ve Azure kullanım müşterinize doğrudan tek faturalandırılır olmasıdır.  

Microsoft, senaryoda düzenler ve sizin adınıza toplar. Microsoft faturalama, müşteri kendi tedarik ilişki oluşturmak için gereksinimini ortadan kaldırır. Tek-fatura zamandan ve kaynaklardan tasarruf. Tek-fatura satışı giriş, yerine fatura toplama odaklanmanıza yardımcı olabilir. 

## <a name="enterprise-agreement"></a>Kurumsal Anlaşma  
Microsoft ile bir Kurumsal Anlaşma (EA) müşterisiyseniz, Microsoft ürünleri için ödeme yapmayı EA'nıza kullanabilir. Azure kullanımınızı dahil olmak üzere ürünler için fatura. Ödeme EA'nıza kullanarak, yazılım lisans ve bulut Hizmetleri için en az üç yıl kuruluşlar için tasarlanmıştır. Ödeme bir ön ödeme yapmak yerine kullanıma yayılır. Kullandıkça Öde kullanıyorsanız liste türü transact, sonra üç aylık EA fazla kullanım fatura döngüsü yazılım lisansı maliyetlerini faturalandırması izler.  

### <a name="monetary-commitment"></a>Parasal Taahhüt  
EA müşterisiyseniz, Azure sözleşmenize ekleyebilir. Azure için önden parasal taahhütte bulunarak Azure sözleşmenize ekleyin. Parasal taahhüt yıl boyunca kullanılır. Taahhüdünüz, Azure'da çeşitli Hizmetleri kullanımını herhangi bir birleşimini içerir.
