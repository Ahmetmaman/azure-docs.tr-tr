# <a name="billing-options"></a>Faturalama seçenekleri  

## <a name="commercial-considerations-on-the-marketplace"></a>Market'te ticari konuları  
Market aşağıdaki liste türleri için geliri paylaşmaz.  
*   Liste  
*   Deneme  
*   Getir bilgisayarınızı kendi lisans (KLG) fatura modelini kullanarak transact

Market'te veriş katılan için faturalanan ek ücretler değildir.  
*   Daha fazla bilgi için Azure Market katılım sayfasında bulunan ilkeleri ziyaret [azure.microsoft.com/support/legal/marketplace/participation-policies](https://azure.microsoft.com/support/legal/marketplace/participation-policies).  

## <a name="pay-as-you-go-and-bring-your-own-license-billing-options"></a>Kullandıkça Öde ve kendi lisans faturalama seçeneklerini Getir  
Kullandıkça Öde faturalama modelini yayımlama seçeneği olarak kullandığınızda, kullanım tabanlı yazılım lisans gelir: 80/20 ve Microsoft arasında paylaşılan. Kullandıkça Öde ve KLG faturalama modelleri kullanarak tek bir teklif fiyatlandırılır. İki faturalama modelleri teklif düzeyinde ayrı SKU'ları arada. Faturalama modelleri teklifiniz bulut iş ortağı portalında yapılandırın. 

Aşağıdaki örneklere bakın.  
*   Kullandıkça Öde seçeneği etkinleştirirseniz, aşağıdaki sonucu sahip.  
    | Maliyet lisansı | saat başına 1.00 $ |  
    |:--- |:--- |  
    | Azure kullanım maliyeti (1/D1-çekirdek) | saat başına 0.14 |  
    | *Müşteri Microsoft tarafından faturalandırılır* | *saat başına 1,14* |  

    Bu senaryoda, Microsoft, yayımlanmış bir VM görüntüsü kullanımı için saat başına 1,14 ödemenizi işler.  
    | Microsoft faturalar | saat başına 1,14 |  
    |:--- |:--- |  
    | Microsoft lisans maliyetinizi % 80'i ödeyen | saat başına 0,80 |  
    | Microsoft lisans maliyetinizi % 20 tutar | saat başına 0,20 |  
    | Microsoft Azure kullanım maliyeti tutar | saat başına 0.14 |  

*   KLG seçeneği etkinleştirirseniz, aşağıdaki sonucu sahip.  
    | Maliyet lisansı | Üzerinde anlaşılan ve sizin tarafınızdan faturalandırılır lisans ücreti |  
    |:--- |:--- |  
    | Azure kullanım maliyeti (1/D1-çekirdek) | saat başına 0.14 |  
    | *Müşteri Microsoft tarafından faturalandırılır* | *saat başına 0.14* |  

    Bu senaryoda, Microsoft, yayımlanmış bir VM görüntüsü kullanımı için saat başına 0.14 ödemenizi işler.  
    | Microsoft faturalar | saat başına 0.14 |  
    |:--- |:--- |  
    | Microsoft Azure kullanım maliyeti tutar | saat başına 0.14 |  
    | Microsoft lisans maliyetinizi %0 tutar | saat başına 0,00 |  

## <a name="single-billing-and-payment-methods"></a>Tek faturalandırma ve ödeme yöntemleri  
Seçenek listeleme Transact kullanmanın önemli bir avantajı Azure kullanım ve lisanslama maliyetleri doğrudan müşterinize tek faturalandırılır olmasıdır.  

Senaryoda Microsoft bills ve sizin adınıza toplar. Microsoft faturalama Müşterinizle kendi satın alma ilişkisi oluşturmak için gerekliliğini ortadan kaldırır. Tek faturalama zaman ve kaynak kaydedebilirsiniz. Tek faturalama satışı giriş yerine fatura toplama odaklanmanıza yardımcı olabilir. 

## <a name="enterprise-agreement"></a>Kurumsal Anlaşma  
Microsoft ile bir Kurumsal Anlaşma (Kurumsal Sözleşme) müşteri varsa, Microsoft ürünleri için ödeme yapmak için EA kullanabilir. Azure kullanımınızı gibi ürün reçetesi. Ödeme yapmak için EA kullanarak, yazılım lisans ve bulut Hizmetleri üç veya daha fazla yıllık istediğiniz kuruluşlar için tasarlanmıştır. Bir açık ödeme yapmak yerine ödemeler çıkışı yayılır. Kullandıkça Öde kullanıyorsanız liste türü transact, ardından üç aylık EA aşma faturalama döngüsü maliyetleri lisans için fatura izler.  

### <a name="monetary-commitment"></a>Parasal Taahhüt  
Ardından bir EA müşterisiyseniz, Azure sözleşmenize ekleyebilir. Azure için önden parasal taahhütte yaparak Azure sözleşmenize ekleyin. Parasal taahhüt yıl kullanılır. Taahhüt Azure'da çok çeşitli hizmetlerini kullanımını herhangi bir birleşimini içerir.
