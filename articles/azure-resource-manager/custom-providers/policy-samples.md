# <a name="azure-policy-built-in-definitions-for-azure-custom-resource-providers"></a>Azure özel kaynak sağlayıcıları için Azure Ilkesi yerleşik tanımları

Bu sayfa, Azure özel kaynak sağlayıcıları için [Azure ilkesi](../../governance/policy/overview.md) yerleşik ilke tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../../governance/policy/samples/built-in-policies.md)bölümüne bakın.

Her yerleşik ilke tanımının adı Azure portal ilke tanımına bağlanır. Kaynağı [Azure Policy GitHub](https://github.com/Azure/azure-policy)deposunda görüntülemek için **Sürüm** sütunundaki bağlantıyı kullanın.
