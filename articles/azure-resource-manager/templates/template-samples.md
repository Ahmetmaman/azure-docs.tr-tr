# <a name="azure-resource-manager-templates-for-management-features"></a>Yönetim özellikleri için Azure Kaynak Yöneticisi şablonları

Aşağıdaki tablo, Kaynak Yöneticisi tarafından sağlanan özellikler için Azure Kaynak Yöneticisi şablonlarına bağlantılar içerir.

| | |
|-|-|
|**Rol atamaları**||
| [Kaynak grubu için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-resourcegroup)| Varolan bir kaynak grubu için kullanıcıya yerleşik bir rol atar. |
| [Varolan sanal makine için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/101-rbac-builtinrole-virtualmachine)| Varolan bir VM için kullanıcıya yerleşik bir rol atar. |
| [Birkaç sanal makine için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/201-rbac-builtinrole-multipleVMs)| Birden fazla sanal makine için kullanıcılara yerleşik roller atar. |
| [Azure aboneliği için rol atama](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/subscription-role-assigment)| Azure aboneliği için bir kullanıcıya rol atar. |
|**Rol tanımı**||
| [Özel rol tanımı oluşturma](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-role-def)| Azure aboneliğinde yeni bir rol tanımı oluşturur. |
|**Kaynak kilidi**||
| [Kaynak grubunu kilitleme](https://github.com/Azure/azure-quickstart-templates/tree/master/subscription-level-deployments/create-rg-lock-role-assignment)| Bir kaynak grubu oluşturur ve kaynak grubuna **DoNotDelete** kilidi uygular. Katkıda bulunan rolü bir kullanıcıya atar. |
