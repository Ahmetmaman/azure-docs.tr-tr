# <a name="service-bus-messaging-samples"></a>Service Bus Mesajlaşma örnekleri
Service Bus Mesajlaşma örnekleri anahtar özelliklerini gösteren [Service Bus mesajlaşması](https://azure.microsoft.com/services/service-bus/). Şu anda aşağıdaki konumlarda örnekleri bulabilirsiniz:

| Programlama dili | SDK'sı veya örnekleri konumu |
| -------------------- | ----------------------- |
| .NET, Java ve Yönetimi | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Service Bus Explorer

Ayrıca, [hizmet veri yolu Gezgini](https://github.com/paolosalvatori/ServiceBusExplorer) örnek, bir Service Bus hizmeti ad alanınıza bağlanın ve mesajlaşma varlıkları kolayca yönetmenize olanak tanır ve GitHub üzerinde barındırılır. Araç, içeri/dışarı aktarma işlevi gibi gelişmiş özellikler ve mesajlaşma varlıkları ve geçiş hizmetleri test etme olanağı sağlar. Üzerinde tam hizmet veri yolu Gezgini kaynak ve belgeler bulabilirsiniz [GitHub](https://github.com/paolosalvatori/ServiceBusExplorer).
