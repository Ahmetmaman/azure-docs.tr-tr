# <a name="service-bus-messaging-samples"></a>Servis Veri Günü mesajlaşma örnekleri
Servis Veri Servisi mesajlaşma [örnekleri, Servis Veri Servisi mesajlaşmasındaki](https://azure.microsoft.com/services/service-bus/)temel özellikleri gösterir. Şu anda, örnekleri aşağıdaki yerlerde bulabilirsiniz:

| Programlama dili | SDK veya örnek konumu |
| -------------------- | ----------------------- |
| .NET, Java ve Yönetim | https://github.com/Azure/azure-service-bus/ |
| Node.js | https://github.com/Azure/azure-sdk-for-js/tree/master/sdk/servicebus/service-bus/samples |
| Python | https://github.com/Azure/azure-sdk-for-python/tree/master/sdk/servicebus/azure-servicebus |

## <a name="service-bus-explorer"></a>Service Bus Explorer

Buna ek olarak, [Service Bus Explorer,](https://github.com/paolosalvatori/ServiceBusExplorer) GitHub'da barındırılan ve bir Service Bus hizmeti ad alanına bağlanmanızı ve ileti varlıklarını kolayca yönetmenize olanak tanıyan bir örnektir. Araç, alma/dışa aktarma işlevselliği ve ileti varlıklarını ve geçiş hizmetlerini sınama olanağı gibi gelişmiş özellikler sağlar. Tam Servis Veri Günü Explorer kaynağını ve belgelerini [GitHub'da](https://github.com/paolosalvatori/ServiceBusExplorer)bulabilirsiniz.
