# <a name="azure-cosmos-db-query-cheat-sheets"></a>Azure Cosmos DB sorgu hile sayfaları

**Azure Cosmos DB sorgu hile sayfaları,** sık sık yazdırılan PDF başvuru sayfalarında sık karşılaşılan veritabanı sorgularını, işlemleri, işlevleri ve işleçleri görüntüleyerek verileriniz için sorguları hızla yazmanıza yardımcı olur. Hile sayfaları SQL, MongoDB, Tablo ve Gremlin API'leri için referans bilgilerini içerir. 

Harf boyutunda veya A3 boyutlu indirme ler arasından seçim yapın. 

## <a name="letter-sized-cheat-sheets"></a>Harf boyutunda hile sayfaları

Harf boyutunda kağıda (8,5" x 11") yazdıracaksanız [Azure Cosmos DB harf boyutunda sorgu hile kağıtlarını](https://go.microsoft.com/fwlink/?LinkId=623215) indirin.

![Sql API, JavaScript, MongoDB, Gremlin ve Table API sorguları ve işlevleri ile Azure Cosmos DB sorgu hile sayfaları - harf boyutunda](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-letter.png)

## <a name="oversized-cheat-sheets"></a>Büyük boy hile sayfaları
A3 boyutlu kağıda (11,7" x 16,5") bir çizici veya büyük ölçekli yazıcı kullanarak yazdıracaksanız [Azure Cosmos DB A3 boyutlu sorgu hile sayfalarını](https://go.microsoft.com/fwlink/?linkid=870413) indirin.

![Azure Cosmos DB sorgu hile sayfaları - A3 boyutlu, SQL API, JavaScript, MongoDB, Gremlin ve Tablo API sorguları ve işlevleri ile](./media/query-cheat-sheet/azure-cosmos-db-cheat-sheet-a3.png)
