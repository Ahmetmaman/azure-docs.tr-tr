# <a name="azure-policy-built-in-policy-definitions-for-azure-cosmos-db"></a>Azure Cosmos DB için Azure Ilkesi yerleşik ilke tanımları

Bu sayfa, Azure Cosmos DB için [Azure ilkesi](../governance/policy/overview.md) yerleşik ilke tanımlarının bir dizinidir. Diğer hizmetlere yönelik diğer Azure Ilkesi için yerleşik olarak [bulunan Azure ilkesi yerleşik tanımları](../governance/policy/samples/built-in-policies.md)bölümüne bakın.
