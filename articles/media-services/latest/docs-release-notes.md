# <a name="azure-media-services-v3-documentation-updates"></a>Azure Media Services v3 dokümantasyon güncellemeleri

>RsS özet akışı okuyucunuza bu URL'yi `https://docs.microsoft.com/api/search/rss?search=%22Azure+Media+Services+v3+documentation+updates%22&locale=en-us` kopyalayıp yapıştırarak güncellemeler için bu sayfayı ne zaman tekrar ziyaret edeceğiz konusunda bilgilendirilin.

Bu makalede, en son Medya Hizmetleri v3 dokümantasyon güncelleştirmeleri hakkında görüşmeler.

## <a name="april-2020"></a>Nisan 2020

* Azure Media Player dokümanları [Azure belgelerine](../azure-media-player/azure-media-player-overview.md)geçirildi.
* [Open Broadcasting Studio (OBS)](live-events-obs-quickstart.md) quickstart ile Canlı akış eklendi. Azure portalını ve OBS'yi kullanarak Medya Hizmetleri canlı akışının nasıl oluşturulacak larını gösterir.

## <a name="march-2020"></a>Mart 2020

[Telestream Wirecast](live-events-wirecast-quickstart.md) quickstart ile Canlı akış eklendi. Azure portalını ve Telestream Wirecast'ı kullanarak Azure Medya Hizmetleri canlı akışını nasıl oluşturabileceğinizi gösterir.
