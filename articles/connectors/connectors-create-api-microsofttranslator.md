# <a name="get-started-with-the-microsoft-translator-connector"></a>Microsoft Translator Bağlayıcısı ile çalışmaya başlama
Metinleri çevirin, bir dil ve daha fazlasını algılamak için Microsoft Translator'a bağlanın. Microsoft Translator ile şunları yapabilirsiniz: 

* Microsoft Translator alma verileri temel alan, iş akışınızı oluşturun. 
* Eylemler metinleri çevirin, bir dil ve daha fazlasını algılamak için kullanın. Bu Eylemler, yanıt alın ve çıkış diğer eylemler için kullanılabilir yapın. Örneğin, Dropbox'ta yeni dosya oluşturulduğunda, Microsoft Translator'ı kullanarak başka bir dile dosyayı metin çevirebilirsiniz.

Şimdi mantıksal uygulama oluşturmaya başlama, bkz: [mantıksal uygulama oluşturma](../logic-apps/quickstart-create-first-logic-app-workflow.md).
