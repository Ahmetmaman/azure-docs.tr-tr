# <a name="get-started-with-the-excel-connector"></a>Excel Bağlayıcısı ile çalışmaya başlama
Logic Apps içinde hiçbir Excel bağlayıcı yoktur. 

## <a name="to-use-excel-data"></a>Excel verilerini kullanmak için
Excel verileri virgülle ayrılmış değer (CSV) dosyası olarak bir depolama klasörü gibi depolayabilirsiniz [OneDrive](connectors-create-api-onedrive.md). Bu CSV dosyası ile de kullanabileceğiniz [düz dosya bağlayıcı](../logic-apps/logic-apps-enterprise-integration-flatfile.md).

## <a name="connector-specific-details"></a>Bağlayıcı özgü ayrıntıları

Tüm tetikleyiciler ve Eylemler swagger tanımlanan görüntüleyebilir ve ayrıca herhangi bir sınır bkz [Bağlayıcısı ayrıntıları](/connectors/excel/).
