# <a name="develop-for-azure-stack"></a>Azure Stack için geliştirme
Bir Azure yığın ortamına erişimi yoksa bile uygulamaları Bugün, geliştirme başlayabiliriz. Azure yığını, veri merkezinizde çalışan bir Microsoft Azure hizmetleri sunan olduğundan, Azure ile olduğu gibi Azure yığın karşı geliştirmek için benzer araçlarını ve işlemlerini kullanabilirsiniz.  Hazırlama ve kılavuzunun aşağıdaki konulardan bit ile bir Azure yığın ortamını benzetmek için Azure kullanabilirsiniz:

* Azure'da, ayrıca Azure yığınına dağıtılabilir Azure Resource Manager şablonları oluşturabilirsiniz.  Bkz: [şablonu konuları](azure-stack-develop-templates.md) Şablonlarınızı taşınabilirlik emin olmak için geliştirme konusunda yönergeler için.
* Hizmet kullanılabilirliği ve hizmet sürümü oluşturma Azure yığını ile Azure arasındaki fark yoktur. Kullanabileceğiniz [Azure yığın ilke modülü](azure-stack-policy-module.md) Azure yığınında kullanılabilir olanlarla Azure hizmet kullanılabilirliği ve kaynak türlerini kısıtlamak üzere. Kullanılabilir hizmetler sınırlama Azure yığını tarafından kullanılabilen hizmetler kullanır, uygulamanızın yardımcı olur.
* [Azure yığın hızlı başlangıç şablonlarını](https://github.com/Azure/AzureStack-QuickStart-Templates) hem Azure hem de Azure yığın dağıtılabilmesi amacıyla, şablonlarını nasıl geliştireceğinizi yaygın senaryo örnekleri.
