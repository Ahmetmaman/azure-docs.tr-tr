# <a name="support-for-32-bit-operating-systems-in-azure-virtual-machines"></a>Azure sanal makinelerinde 32 bit işletim sistemleri desteği

Microsoft Azure artık kullanıcıların 32 bit Windows İşletim sistemlerini Azure'a getirmelerine olanak sağlıyor. Bu işletim sistemlerinden bazıları yaşam desteklenebilirlik anlaşmasının sonuna çoktan ulaştığından, Microsoft onlar için ek destek sunmayabilir. Linux tabanlı veya Microsoft Azure sanal makinede (VM) çalışan Berkeley Software Distribution (BSD) tabanlı işletim sistemleri için de destek sunulmaz.

## <a name="more-information"></a>Daha fazla bilgi

Azure sanal makinelerinde desteklenen işletim sistemleri hakkında daha fazla bilgi için aşağıdaki Microsoft Bilgi Bankası makalelerine gidin:

* [Microsoft Azure sanal makineleri için Microsoft sunucu yazılımı desteği](https://support.microsoft.com/help/2721672/microsoft-server-software-support-for-microsoft-azure-virtual-machines)
* [Azure'da Linux ve açık kaynak teknolojisi desteği](https://support.microsoft.com/help/2941892/support-for-linux-and-open-source-technology-in-azure)
