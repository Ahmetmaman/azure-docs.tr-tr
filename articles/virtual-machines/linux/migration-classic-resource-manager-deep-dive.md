# <a name="technical-deep-dive-on-platform-supported-migration-from-classic-to-azure-resource-manager"></a>Klasik modelden Azure Resource Manager’a platform destekli geçişe ayrıntılı teknik bakış

Azure Klasik dağıtım modelinden Azure Resource Manager dağıtım modeline geçirilmesi hakkında derinlemesine ele alalım. Azure platformu kaynaklar iki dağıtım modelleri arasında nasıl geçirdiğini anlamanıza yardımcı olması için kaynak ve özellik düzeyinde kaynakları arayın. Daha fazla bilgi için lütfen hizmeti duyurusunu makaleyi okuyun: [Iaas kaynaklarının Klasik modelden Azure Resource Manager'a Platform destekli geçişe](migration-classic-resource-manager-overview.md?toc=%2fazure%2fvirtual-machines%2flinux%2ftoc.json).
