# <a name="scenario-apache-tez-application-hangs-in-azure-hdinsight"></a>Senaryo: Apache Tez uygulaması Azure HDInsight'ta asılı

Bu makalede, Azure HDInsight kümeleriyle etkileşimde olurken sorun giderme adımları ve sorunlarla ilgili olası çözümler açıklanmaktadır.

## <a name="issue"></a>Sorun

Tez görünümünden Apache Hive iş gönderdikten sonra, iş durumu "Çalışan", ancak herhangi bir ilerleme görünmüyor

## <a name="cause"></a>Nedeni

Çok fazla iş gönderildi; uzun İplik kuyruğu.

## <a name="resolution"></a>Çözüm

Kümeyi ölçeklendirin veya İplik sırasının drene geçmesini bekleyin.

Varsayılan `yarn.scheduler.capacity.maximum-applications` olarak çalışan veya bekleyen en fazla uygulama sayısını denetler `10000`ve varsayılan olarak .
