Hub ağ geçidini içerir. Hub oluşturulduktan sonra site eklemeseniz bile hub için ücretlendirilirsiniz. Hub ve ağ geçidinin oluşturulması 30 dakika sürer.

1. Oluşturduğunuz sanal WAN 'ı bulun. Sanal WAN sayfasında, **sanal WAN mimarisi** bölümünde, **Hublar**' a tıklayın.
2. Hub 'Lar sayfasında **+ yeni hub** ' a tıklayarak **sanal hub oluştur** sayfasını açın.
3. **Sanal hub oluştur** sayfasında, aşağıdaki alanları doldurun:

   * Konum
   * Name
   * Hub özel adres alanı

Hub’ı oluşturmak için **Onayla**'ya tıklayın. Hub 'ı **hub 'lar** sayfasında görüntülemek için **Yenile** ' ye tıklayın.
