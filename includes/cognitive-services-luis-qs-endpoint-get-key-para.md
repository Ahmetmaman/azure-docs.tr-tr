1. LUIS hesabınızı kullanarak oturum açın. 

    [![Ekran görüntüsü, arama Language Understanding (LUIS) uygulama listesini](media/cognitive-services-luis/app-list.png "ekran görüntüsü, arama Language Understanding (LUIS) uygulama listesi")](media/cognitive-services-luis/app-list.png)

2. Sağ üst köşedeki menüden adınızı seçin ve ardından **ayarları**.

    ![LUIS kullanıcı ayarları menüsüne erişime](media/cognitive-services-luis/get-user-settings-in-luis.png)

3. Değerini kopyalayın **yazma anahtar**. Bu hızlı başlangıçta kullanacaksınız. 

    [![Ekran görüntüsü, arama Language Understanding (LUIS) kullanıcı ayarları](media/cognitive-services-luis/get-user-authoring-key.png "ekran görüntüsü, arama Language Understanding (LUIS) kullanıcı ayarları")](media/cognitive-services-luis/get-user-authoring-key.png)
