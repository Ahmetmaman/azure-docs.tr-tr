#### <a name="to-install-the-storsimple-adapter-for-sharepoint"></a>SharePoint için StorSimple Adaptörü yüklemek için
1. Yükleyiciyi, SharePoint Merkezi Yönetim web uygulamasını çalıştırmak üzere yapılandırılan web ön uç (WFE) sunucusuna kopyalayın. 
2. WFE sunucusunda oturum açabilmek için yönetici ayrıcalıklarına sahip bir hesap kullanın.
3. Yükleyiciyi çift tıklatın. SharePoint Kurulum Sihirbazı için StorSimple Adaptörü başlar. Yüklemeye başlamak için **İleri'yi** tıklatın.
   
    ![StorSimple adaptör kurulum başlangıç sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup1-include.png)
4. SharePoint kurulum yapılandırma sayfasında StorSimple Adaptörü'nde, bir yükleme konumu seçin, StorSimple cihazınızdaki DATA 0 ağ arabirimi için IP adresini yazın ve **Sonra İleri'yi**tıklatın. 
   
    ![StorSimple adaptör kurulum yapılandırma sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup2-include.png) 
5. Kurulum onay **sayfasında, Yükle'yi**tıklatın.
   
    ![StorSimple adaptörü kurulum onay sayfası](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Confirm_Setup-include.png) 
6. Kurulum Sihirbazı'nı kapatmak için **Bitir'i** tıklatın.
   
    ![StorSimple adaptör kurulumu sayfa yı bitirdi](./media/storsimple-install-sharepoint-adapter/HCS_SSASP_Setup_finish-include.png) 
7. SharePoint Merkezi Yönetim sayfasını açın. SharePoint bağlantıları için StorSimple Adaptörü içeren bir StorSimple Configuration grubunu görmeniz gerekir.
8. Bir sonraki adıma geçin: [RbS'yi yapılandırın.](#configure-rbs)
