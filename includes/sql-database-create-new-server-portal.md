## <a name="create-an-azure-sql-database-logical-server"></a>Azure SQL Database mantıksal sunucusu oluşturma
Azure SQL Database mantıksal sunucu oluşturmak için Azure Portal'da aşağıdaki adımları kullanın.

1. O anda bağlı değilseniz [Azure portal](http://portal.azure.com)’a bağlanın.
2. **Yeni**'ye ve **Veri + Depolama**'ya tıklayın; ardından, **SQL Veritabanı**'nı bulun.
3. SQL Veritabanı dikey penceresini açmak için **SQL Veritabanı**'na tıklayın.
4. Şu sunucu özellikleri için değerleri girin:
   
   * Sunucu adı
   * Sunucu yöneticisi oturum açma
   * Parola
   * Abonelik (yalnızca birden fazla aboneliğiniz varsa)
   * Kaynak grubu (yeni veya varolan)
   * Konum
5. **Oluştur**’a tıklayın. Bildirim alanında dağıtımın başlatıldığını görebilirsiniz.
6. Sonraki adıma geçmeden önce dağıtımın tamamlanmasını bekleyin.
