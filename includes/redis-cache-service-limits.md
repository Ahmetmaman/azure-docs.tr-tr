Redis sınırları ve boyutları için Azure Önbelleği her fiyatlandırma katmanı için farklıdır. Fiyatlandırma katmanlarını ve ilişkili boyutlarını görmek [için Redis fiyatlandırması için Azure Önbelleği'ne](https://azure.microsoft.com/pricing/details/cache/)bakın.

Redis yapılandırma sınırları için Azure Önbelleği hakkında daha fazla bilgi için varsayılan [Redis sunucu](../articles/azure-cache-for-redis/cache-configure.md#default-redis-server-configuration)yapılandırması'na bakın.

Redis örnekleri için Azure Önbelleği yapılandırması ve yönetimi Microsoft tarafından yapıldığından, redis için Azure Önbelleğinden tüm Redis komutları desteklenmez. Daha fazla bilgi için [Redis komutları için Redis için Azure Önbelleği'nde desteklenmeyen komutlara](../articles/azure-cache-for-redis/cache-configure.md#redis-commands-not-supported-in-azure-cache-for-redis)bakın.
