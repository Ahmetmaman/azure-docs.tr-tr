## <a name="use-azure-cloud-shell"></a>Azure Cloud Shell kullanma

Azure, Azure Cloud Shell, tarayıcınız üzerinden kullanabileceğiniz bir etkileşimli Kabuk ortamını barındırır. Cloud Shell'i ya da kullanmanıza imkan tanır `bash` veya `PowerShell` Azure hizmetleriyle çalışmak için. Cloud Shell önceden yüklenmiş komutları, yerel ortamınızda herhangi bir yükleme gerekmeden bu makaledeki kodu çalıştırmak için kullanabilirsiniz.

Azure Cloud Shell'i başlatmak için:

| Seçenek | Örnek/bağlantı |
|-----------------------------------------------|---|
| Kod bloğunun sağ üst köşesindeki **Deneyin**’i seçin. Seçme **deneyin** kod Cloud Shell için otomatik olarak kopyalamaz. | ![Örneği Azure Cloud Shell için deneyin](./media/cloud-shell-try-it/cli-try-it.png) |
| Git [ https://shell.azure.com ](https://shell.azure.com) veya **Cloud Shell'i Başlat** Cloud Shell, tarayıcınızda açmak için düğmeyi. | <a href="https://shell.azure.com" title="Azure Cloud Shell'i Başlat"><img name="launch-cloud-shell" src="https://docs.microsoft.com/azure/includes/media/cloud-shell-try-it/launchcloudshell.png" /></a> |
| Seçin **Cloud Shell** düğmesine sağ üst menü çubuğundaki [Azure portalında](https://portal.azure.com). | ![Azure portaldaki Cloud Shell düğmesi](./media/cloud-shell-try-it/cloud-shell-menu.png) |

Bu makalede Azure Cloud shell'de kod çalıştırmak için:

1. Cloud Shell'i başlatın.
1. Seçin **kopyalama** kodu kopyalamak için bir kod bloğu üzerinde düğmesi. 
1. Cloud Shell oturumla kodu yapıştırın **Ctrl**+**Shift**+**V** Windows ve Linux'ta veya **Cmd** + **Shift**+**V** macOS üzerinde. 
1. Tuşuna **Enter** kodu çalıştırmak için.
