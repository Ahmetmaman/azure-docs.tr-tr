Azure Blob depolama hakkında temel bilgileri öğrendiğinize göre, daha karmaşık depolama görevleri hakkında daha fazla bilgi edinmek için bu bağlantıları izleyin.

* Kullanılabilir API 'Lerle ilgili tüm ayrıntılar için, [.net Için Azure depolama Istemci kitaplığı](/dotnet/api/overview/azure/storage) 'Nda bulunan BLOB hizmeti başvuru belgelerini görüntüleyin.
* Azure depolama ile çalışmak için yazdığınız kodu nasıl basitleştireceğinizi öğrenmek için [Azure Web İşleri SDK 'sını](https://github.com/Azure/azure-webjobs-sdk/wiki) inceleyin
* Azure’da veri depolama ile ilgili ek seçenekler hakkında daha fazla bilgi edinmek için daha fazla özellik kılavuzu görüntüleyin.
  * Azure depolama tabloları ile çalışmak için bkz. [.NET kullanarak Azure Tablo Depolamayı kullanmaya başlama](../articles/cosmos-db/tutorial-develop-table-dotnet.md).
  * Azure depolama kuyrukları ile çalışmak için [.NET kullanarak Azure kuyruk depolama](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)'yı kullanmaya başlayın.
  * İlişkisel verileri depolamak için bkz. [.net (C#) kullanarak SQL veritabanı 'Na bağlanma](../articles/azure-sql/database/connect-query-dotnet-visual-studio.md).
