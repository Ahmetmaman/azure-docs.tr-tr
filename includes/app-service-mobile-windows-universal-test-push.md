1. Windows Store projesi sağ tıklayın, **başlangıç projesi olarak ayarla**, ardından Windows Store uygulamasını çalıştırmak için F5 tuşuna basın.

    Uygulama başlatıldıktan sonra cihaza anında iletme bildirimleri için kayıtlı.
2. Windows Store app durdurun ve Windows Phone Store uygulaması için önceki adımı yineleyin.

    Bu noktada, anında iletme bildirimleri almak için her iki cihaz kaydedilir.

3. Windows Store uygulaması'nı yeniden çalıştırın ve kutusuna metin **Todoıtem Ekle**ve ardından **Kaydet**.

    INSERT tamamlandıktan sonra hem Windows Store hem de Windows Phone uygulamaları anında iletme bildirimi WNS'den aldığınız unutmayın. Bile uygulama çalışmadığında bildirim Windows Phone üzerinde görüntülenir.
