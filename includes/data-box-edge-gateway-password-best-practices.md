Bu en iyi uygulamaları aklınızda bulundurun:

- Unutulursa parolayı sıfırlamak zorunda kalmamak için tüm parolaları güvenli bir yerde saklamanızı öneririz. Yönetim hizmeti varolan parolaları alamaz. Bunları yalnızca Azure portalı üzerinden sıfırlayabilir. Bir parolayı sıyrıksanız, sıfırlamadan önce tüm kullanıcıları bilgilendirdiğinizden emin olun.
- Cihazınızın Windows PowerShell arayüzüne HTTP üzerinden uzaktan erişebilirsiniz. Güvenlik en iyi uygulama olarak, http'yi yalnızca güvenilen ağlarda kullanmalısınız.
- Aygıt parolalarının güçlü ve iyi korunduğundan emin olun. [Parolaen iyi uygulamaları](https://docs.microsoft.com/azure/security/fundamentals/identity-management-best-practices#enable-password-management)izleyin.
