1. Cloud Shell'de `rg.yml` adlı bir dosya oluşturun.

    ```bash
    code rg.yml
    ```

1. Aşağıdaki kodu düzenleyiciye yapıştırın:

   ```yaml
   ---
   - hosts: localhost
     connection: local
     tasks:
       - name: Create resource group
         azure_rm_resourcegroup:
           name: ansible-rg
           location: eastus
         register: rg
       - debug:
           var: rg
   ```

1. Dosyayı kaydedin ve düzenleyiciden çıkın.

1. Kullanarak playbook çalıştırma `ansible-playbook` komutu:

   ```bash
   ansible-playbook rg.yml
   ```
