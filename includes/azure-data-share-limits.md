Azure Veri Paylaşımı, kuruluşların verileri müşterileri ve iş ortaklarıyla basit ve güvenli bir şekilde paylaşmasını sağlar.

| **Kaynak** | **Sınırı** |
| --- | --- |
| Azure aboneliği başına maksimum Veri Paylaşımı kaynağı sayısı |50 |
| Veri Paylaşımı kaynağı başına gönderilen maksimum hisse sayısı |100 |
| Veri Paylaşımı kaynağı başına maksimum alınan hisse sayısı |100 |
| Gönderilen hisse başına maksimum davet sayısı |100 |
| Gönderilen hisse başına maksimum hisse aboneliği sayısı |100 |
| Hisse başına maksimum veri kümesi sayısı |100 |
| Hisse başına en fazla anlık görüntü zamanlama sayısı |1 |
