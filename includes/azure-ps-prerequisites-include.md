## <a name="prerequisite-install-the-azure-powershell-module"></a>Önkoşul: Azure PowerShell modülünü yükleme

Bu makaledeki adımları gerçekleştirmek için [Azure PowerShell modülünü yüklemeniz ve yapılandırmanız](/powershell/azureps-cmdlets-docs) gerekir. Tüm yönergeleri tamamladığınızdan emin olun. Yükleme işlemi tamamlandıktan sonra Azure'da oturum açın ve aboneliğinizi seçin.

> [!NOTE]
> Bu adımları tamamlamak için bir Azure hesabınızın olması gerekir. Azure hesabınız yoksa [ücretsiz denemeye](../articles/active-directory/fundamentals/sign-up-organization.md) kaydolabilirsiniz.
