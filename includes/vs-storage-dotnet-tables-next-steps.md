Azure Tablo depolama alanının temellerini öğrendiğiniz için, daha karmaşık depolama görevleri hakkında bilgi edinmek için bu bağlantıları izleyin.

* Kullanılabilir API'ler hakkında tüm ayrıntılar için .NET başvurusu [için Azure Depolama İstemci Kitaplığı'ndaki](https://go.microsoft.com/fwlink/?LinkID=390731) Tablo Hizmeti başvuru belgelerini görüntüleyin.
* Azure Depolama ile çalışmak için yazdığınız kodu nasıl basitleştirdiğinizi öğrenmek için [Azure Web İşleri SDK'sının ne olduğunu](https://github.com/Azure/azure-webjobs-sdk/wiki) göz atın
* Azure’da veri depolama ile ilgili ek seçenekler hakkında daha fazla bilgi edinmek için daha fazla özellik kılavuzu görüntüleyin.
  * Azure Depolama Blob'ları ile çalışmak için [.NET'i kullanarak Azure Blob depolama alanına başlayın'a](../articles/storage/blobs/storage-dotnet-how-to-use-blobs.md)bakın.
  * Azure Depolama Kuyrukları ile çalışmak için [.NET'i kullanarak Azure Kuyruk depolama alanına başlayın.](../articles/storage/queues/storage-dotnet-how-to-use-queues.md)
  * İlişkisel verileri depolamak [için](../articles/sql-database/sql-database-develop-dotnet-simple.md)bkz.
