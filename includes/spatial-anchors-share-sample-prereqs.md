## <a name="prerequisites"></a>Önkoşullar

Bu öğreticiyi tamamlamak için şunlar sahip olduğunuzdan emin olun:

* Bir Windows makineyle <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017 +</a> ile yüklenen **ASP.NET ve web geliştirme** iş yükü.
* [.NET Core SDK'sını 2.2](https://dotnet.microsoft.com/download).
* Bir veya daha fazla cihazlar (iOS veya Android) dağıtmak ve çalıştırmak için.
  * Android kullanıyorsanız:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3 +</a> ve <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3 +</a> Windows makinenizde yüklü.
    * A <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">etkin Geliştirici</a> ve <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore özellikli</a> Android cihaz.
  * İOS kullanıyorsanız:
    * Bir macOS makineyle <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Xcode 10 +</a>, <a href="https://cocoapods.org" target="_blank">CocoaPods</a> ve <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2018.3 +</a> yüklü.
    * Etkin bir geliştirici <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit uyumlu</a> iOS cihaz.
