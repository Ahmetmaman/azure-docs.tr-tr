## <a name="prerequisites"></a>Ön koşullar

Bu öğreticiyi tamamlamak için şunlar sahip olduğunuzdan emin olun:

* [Azure Uzamsal Çapalar genel görünümünü](../articles/spatial-anchors/overview.md)okuyun.
* 5 dakikalık [Quickstarts](../articles/spatial-anchors/index.yml)biri tamamlandı.
* C# ve Unity hakkında temel bilgiler.
* Android veya iOS kullanmak istiyorsanız <a href="https://developer.apple.com/arkit/" target="_blank">ARKit</a> kullanmak istiyorsanız <a href="https://developers.google.com/ar/discover/" target="_blank">ARCore</a> hakkında temel bilgi.
* <a href="https://www.visualstudio.com/downloads/" target="_blank">Visual Studio 2017</a> veya sonraki ASP.NET **ve web geliştirme** iş yüküyle yüklü olan bir Windows bilgisayarı.
* [.NET Çekirdek 2.2 SDK](https://dotnet.microsoft.com/download).
* Bir uygulamayı dağıtmak ve çalıştırmak için bir veya daha fazla aygıt (iOS veya Android).
  * Android kullanıyorsanız, şunları yapmanız gerekir:
    * <a href="https://developer.android.com/studio/" target="_blank">Android Studio 3.3</a> veya sonrası, <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 veya 2019.2</a>ve Windows için <a href="https://git-scm.com/download/win" target="_blank">Git</a> Windows bilgisayarınızda yüklü.
    * <a href="https://developer.android.com/studio/debug/dev-options" target="_blank">Geliştirici özellikli</a> ve <a href="https://developers.google.com/ar/discover/supported-devices" target="_blank">ARCore özellikli</a> bir Android cihaz.
  * iOS kullanıyorsanız, şunları yapmanız gerekir:
    * <a href="https://geo.itunes.apple.com/us/app/xcode/id497799835?mt=12" target="_blank">Üzerinde Xcode 10</a> veya daha sonra, <a href="https://cocoapods.org" target="_blank">CocoaPods</a>ve <a href="https://unity3d.com/get-unity/download" target="_blank">Unity 2019.1 veya 2019.2</a> yüklü bir macOS bilgisayar.
    * Geliştirici özellikli <a href="https://developer.apple.com/documentation/arkit/verifying_device_support_and_user_permission" target="_blank">ARKit uyumlu</a> iOS aygıtı.
    * Git Homebrew üzerinden yüklenir. Terminalde tek bir satırda aşağıdaki `/usr/bin/ruby -e "$(curl -fsSL https://raw.githubusercontent.com/Homebrew/install/master/install)"` komutu `brew install git`girin: Sonra çalıştırın.
