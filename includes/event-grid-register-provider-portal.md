## <a name="enable-event-grid-resource-provider"></a>Event Grid kaynak sağlayıcısını etkinleştirme

Azure aboneliğinizde daha önce Event Grid kullanmadıysanız Event Grid kaynak sağlayıcısına kaydolmanız gerekebilir.

Azure portalında:

1. **Abonelikler**'i seçin.
1. Event Grid için kullandığınız aboneliği seçin.
1. **Ayarlar**'ın altında **Kaynak sağlayıcıları**'nı seçin.
1. **Microsoft.EventGrid** öğesini bulun.
1. Kayıtlı değilse **Kaydol**'u seçin. 

Kayıt işleminin tamamlanması biraz sürebilir. Durumu güncelleştirmek için **Yenile**’yi seçin. **Durum** **Kayıtlı** olduğunda, devam etmeye hazırsınız demektir.
