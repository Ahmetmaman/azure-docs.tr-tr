### <a name="open-the-mfa-page"></a><a name="mfa"></a>MFA sayfasını açma

1. Azure Portal’da oturum açın.
2. Azure **Etkin Dizini'ne gidin -> Tüm kullanıcılar.**
3. Çok faktörlü kimlik doğrulama sayfasını açmak için **Çok Faktörlü Kimlik Doğrulama'yı** seçin.

   ![Oturum aç](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa1.jpg)

### <a name="select-users"></a><a name="users"></a>Kullanıcıları seçin

1. Çok **faktörlü kimlik doğrulama** sayfasında, MFA'yı etkinleştirmek istediğiniz kullanıcı(lar)ı seçin.
2. **Etkinleştir**’i seçin.

   ![Şunu seçin:](./media/vpn-gateway-vwan-openvpn-azure-ad-mfa/mfa2.jpg)
