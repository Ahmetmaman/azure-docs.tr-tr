> [!TIP]
> Azure depolama, Azure Active Directory (Azure AD) kullanarak Blob ve kuyruk depolama kaynaklarını yetki verme erişimi destekler. Kullanıcıları veya Azure AD tarafından döndürülen bir OAuth 2.0 belirteç kullanan uygulamalar yetkilendirme paylaşılan anahtar yetkilendirme üst düzey güvenlik ve kullanım kolaylığı sağlar ve paylaşılan erişim imzaları (SAS). Azure AD ile depolama hesabı erişim anahtarı ile kod ve risk olası güvenlik açıklarını gerek yoktur.
>
> Microsoft Azure AD, Azure depolama uygulamalarınızla mümkün olduğunda kullanılmasını önerir. Daha fazla bilgi için [erişimi yetkilendirin Azure blobları ve Azure Active Directory'yi kullanarak sıralar](../articles/storage/common/storage-auth-aad.md).
