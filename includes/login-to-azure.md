## <a name="sign-in-to-azure"></a>Azure'da oturum açma

Uygulamanızı Azure'da barındırmak için gerekli kaynakları oluşturma amacıyla Azure CLI'yi kullanacaksınız. Azure CLI komutlarını Cloud Shell’de çalıştırıyorsanız, bu oturumunuzun zaten açık olduğu anlamına gelir. Azure CLI komutları yerel olarak çalıştırmak için [az login](/cli/azure/) komutuyla Azure aboneliğinizde oturum açın ve ekrandaki yönergeleri izleyin.

```azurecli
az login
```
