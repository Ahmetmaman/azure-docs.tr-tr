[Azure App Service](/azure/app-service/overview) , yapı ve kod barındırmanıza olanak sağlar. Bu kod, web uygulamaları, mobil arka uçlar ve RESTful API'leri biçiminde olabilir. App Service'i kullanarak, altyapı yönetimine gerek kalmadan kendi seçtiğiniz programlama dilinde kullanarak kod geliştirebilirsiniz. App Service, hem Windows hem de Linux destekler. GitHub ve Azure DevOps dahil olmak üzere herhangi bir Git deposundan otomatik dağıtımlar desteklenir.
