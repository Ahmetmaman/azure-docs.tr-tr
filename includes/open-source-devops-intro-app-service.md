[Azure Uygulama Hizmeti,](/azure/app-service/overview) kod oluşturmanıza ve barındırmanıza olanak tanır. Bu kod web uygulamaları, mobil arka uçlar ve YENI apiler şeklinde olabilir. Uygulama Hizmetini kullanarak, altyapıyı yönetmeden seçtiğiniz programlama dilini kullanarak kod geliştirebilirsiniz. Uygulama Hizmeti hem Windows'u hem de Linux'u destekler. GitHub ve Azure DevOps dahil olmak üzere herhangi bir Git repo'sundaki otomatik dağıtımlar desteklenir.
