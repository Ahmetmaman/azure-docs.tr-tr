## <a name="test"></a>Azure 'da işlevi doğrulama

Dağıtılan işlevi doğrulamak için bir Web tarayıcısını kullanabilirsiniz.  İşlev tuşu da dahil olmak üzere URL 'YI, Web tarayıcınızın adres çubuğuna kopyalayın. İsteği yürütmeden önce `&name=<yourname>` sorgu dizesini URL 'ye ekleyin.

![İşlevi çağırmak için bir Web tarayıcısı kullanma.](./media/functions-test-function-code/functions-azure-cli-function-test-browser.png)  

Alternatif olarak, dağıtılan işlevi doğrulamak için kıvrımlı kullanabilirsiniz. Önceki adımdan kopyaladığınız işlev anahtarı da dahil olmak üzere URL 'yi kullanarak URL 'ye `&name=<yourname>` sorgu dizesi ekleyin.

![Azure 'da işlevi çağırmak için kıvrımlı kullanma.](./media/functions-test-function-code/functions-azure-cli-function-test-curl.png)
