## <a name="clean-up-deployment"></a>Dağıtımı temizleme 

Betik örneği çalıştırıldıktan sonra kaynak grubunda kaynak grubu, Azure Redis Cache örneğine ve ilgili tüm kaynakları kaldırmak için aşağıdaki komut kullanılabilir.

```azurecli
az group delete --name contosoGroup
```
