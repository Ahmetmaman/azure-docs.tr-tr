## <a name="clean-up-deployment"></a>Dağıtımı temizleme 

Betik örneği çalıştırıldıktan sonra, kaynak grubunu, Redsıs örneği için Azure önbelleğini ve kaynak grubundaki ilgili kaynakları kaldırmak için izle komutu kullanılabilir.

```azurecli
az group delete --name contosoGroup
```
