

Bu örneği çalıştırmak için en son [Azure CLI 2.0](https://docs.microsoft.com/cli/azure/install-azure-cli) sürümünü yüklediğinizden emin olun. Başlangıç olarak, Azure ile bağlantı oluşturmak için `az login` komutunu çalıştırın.
