> [!TIP]
> İndirilen konteyner resimlerinizi listelemek için [docker images](https://docs.docker.com/engine/reference/commandline/images/) komutunu kullanabilirsiniz. Örneğin, aşağıdaki komut, indirilen her kapsayıcı resminin kimlik, depo ve etiketini tablo olarak biçimlendirilir:
>
>  ```
>  docker images --format "table {{.ID}}\t{{.Repository}}\t{{.Tag}}"
>
>  IMAGE ID         REPOSITORY                TAG
>  <image-id>       <repository-path/name>    <tag-name>
>  ```
