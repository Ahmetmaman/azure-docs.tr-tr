API 'Leri kullanarak bir Azure Stack Edge cihazında sanal makineler (VM) oluşturabilir ve yönetebilirsiniz. Bu API 'Ler, yerel Azure Stack Edge uç noktası kullanılarak çağrılan standart Azure Resource Manager API 'lerdir. Azure Resource Manager API 'Leri, bu durumda, cihazda bulunan yerel bir abonelikte VM oluşturmanıza, güncelleştirmenize ve silmesine olanak tanıyan tutarlı bir yönetim katmanı sağlar. Azure PowerShell cmdlet 'leri aracılığıyla Azure Stack Edge üzerinde çalışan Azure Resource Manager bağlanabilirsiniz.
