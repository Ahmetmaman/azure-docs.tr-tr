## <a name="install-the-azure-functions-extension"></a>Azure İşlevleri uzantısını yükleme

Işlevleri oluşturmak ve sınamak ve bunları Azure'a dağıtmak için Azure İşlevler uzantısını kullanabilirsiniz.

1. Visual Studio Code'da **Uzantıları** açın ve **azure işlevlerini**arayın veya [Visual Studio Code'da bu bağlantıyı seçin.](vscode:extension/ms-azuretools.vscode-azurefunctions)

1. Visual Studio Code uzantısını yüklemek için **Yükle'yi** seçin:

    ![Azure İşlevleri için uzantıyı yükleme](./media/functions-install-vs-code-extension/vscode-install-extension.png)

1. Yüklemeden sonra Etkinlik çubuğundaki Azure simgesini seçin. Kenar Çubuğunda bir Azure İşlevleri alanı görmeniz gerekir.

    ![Kenar Çubuğundaki Azure İşlevleri alanı](./media/functions-install-vs-code-extension/azure-functions-window-vscode.png)
