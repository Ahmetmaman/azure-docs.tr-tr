## <a name="supported-ssh-key-formats"></a>Desteklenen SSH anahtar biçimleri

Azure şu anda en az 2048 bit uzunluğunda ssh protokolü 2 (SSH-2) RSA genel-özel anahtar çiftlerini destekler. ED25519 ve ECDSA gibi diğer anahtar biçimleri desteklenmez.
