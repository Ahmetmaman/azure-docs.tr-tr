#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Klasik Azure Portalı aracılığıyla düzenli güncelleştirmeler yüklemek için
1. Üzerinde **cihazları** sayfasında, güncelleştirmeleri yüklemek istediğiniz cihazı seçin.
2. Gidin **cihazları** > **Bakım** ekranı aşağı kaydırarak **yazılım güncelleştirmelerini**.
3. Güncelleştirmeleri denetlemek için **Güncelleştirmeleri denetle** sayfanın alt kısmındaki.
4. Yazılım güncelleştirmeleri varsa bir ileti görürsünüz. Tıklayın **Güncelleştirmeleri Yükle** cihaz güncelleştirme başlatmak için.
   
    Güncelleştirme başarıyla yüklendiğinde size bildirilir.
