#### <a name="to-install-regular-updates-via-the-azure-classic-portal"></a>Klasik Azure portalı aracılığıyla düzenli güncelleştirmeleri yüklemek için
1. **Cihazlar** sayfasında, güncelleştirmeleri yüklemek istediğiniz cihazı seçin.
2. **Cihaz** > **Bakımı** ' na gidin ve **yazılım güncelleştirmeleri**' ne gidin.
3. Güncelleştirmeleri denetlemek için sayfanın alt kısmındaki **Güncelleştirmeleri denetle** 'ye tıklayın.
4. Yazılım güncelleştirmeleri varsa bir ileti görürsünüz. Cihazı güncelleştirmeye başlamak için **güncelleştirmeleri yüklensin** ' e tıklayın.
   
    Güncelleştirme başarıyla yüklendiğinde size bildirim alınacaktır.
