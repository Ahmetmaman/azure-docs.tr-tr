>[!NOTE]
>Şu anda ' de çalıştırılan bu komut gruplarını etkileyen Cloud Shell **bilinen bir sorun** var *https://shell.azure.com* : `az dt route` , `az dt model` , `az dt twin` .
>
>Çözümlemek için aşağıdakilerden birini yapabilirsiniz:
> * `az login`Komutunu çalıştırmadan önce Cloud Shell ' de çalıştırın.
> * Azure portal Cloud Shell bölmesini açın ve Cloud Shell işinizi buradan doldurun.
>  :::image type="content" source="../articles/digital-twins/media/includes/portal-cloud-shell.png" alt-text="' Cloud Shell ' simgesi vurgulanmış Azure portal görünümü ve Portal penceresinin alt kısmında görünen Cloud Shell":::
> * Cloud Shell yerine [Yerel CLI](https://docs.microsoft.com/cli/azure/install-azure-cli?view=azure-cli-latest) kullanın.
>
>Bu sorun hakkında daha fazla bilgi için bkz. [*sorun giderme: Azure dijital TWINS 'de bilinen sorunlar*](../articles/digital-twins/troubleshoot-known-issues.md#400-client-error-bad-request-in-cloud-shell).
