# <a name="microsoft-azure-documentation"></a>Microsoft Azure Belgeleri

Hoş Geldiniz açık kaynak [belgeleri](https://docs.microsoft.com/azure) , [Microsoft Azure](https://azure.microsoft.com). Lütfen Microsoft Azure belgelerine katkıda bulunan olarak nasıl yardımcı olabilecek anlamak için bu Benioku dosyasını gözden geçirin.

## <a name="getting-started"></a>Başlarken

Açık kaynağa katkıda bulunan güncelleştirmeleri daha fazlasını veriyor, bu da bize bir sorun olduğunda bildirdiğiniz. Okuma bizim [katkıda bulunan Kılavuzu](.github/CONTRIBUTING.md) daha fazla bilgi edinmek için.

### <a name="prerequisites"></a>Önkoşullar

Harika olan bir katkı sağlamak karar verdiniz! Belgelere katkıda bulunmak için gereken birkaç araç vardır.

Bir hesabınız yoksa, yönergelerini izleyin, belgelere katkıda bulunan bir GitHub hesabı gerektirir [GitHub hesap kurulumu](https://docs.microsoft.com/contribute/get-started-setup-github) bizim katkıda bulunan Kılavuzu.

#### <a name="download"></a>İndirme

Aşağıdaki araçları yükleyin:

* [Git](https://git-scm.com/download)
* [Visual Studio Code](https://code.visualstudio.com/Download)
* [Docs yazma paketi](https://marketplace.visualstudio.com/items?itemName=docsmsft.docs-authoring-pack) için Visual Studio Code uzantısı

#### <a name="install"></a>Yükleme

Bölümündeki yönergeleri izleyin [içerik yazma araçlarını yükleme](https://docs.microsoft.com/contribute/get-started-setup-tools) bizim katkıda bulunan Kılavuzu.
